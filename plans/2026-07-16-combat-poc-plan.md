# 11세력 대회 세로 슬라이스 구현 Plan

## Plan의 목적

시작 6세력과 추가 5세력을 공용 전투 엔진 위에 구현하고, 시작 문파 선택부터 9전 대회의 본선·결승까지 한 회차가 완주되게 한다.

## 고정 제약

- 전장은 10칸이며 모든 결투의 초기 배치는 A=4, B=6, 거리 2다.
- 같은 칸 중첩·교차·자리 교환이 가능하다.
- 양측은 6시간 슬롯을 비공개로 전부 선택하고 동시에 공개한다.
- AI는 플레이어 행동열을 보기 전에 자신의 행동열을 잠근다.
- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전이다.
- 유파 무공과 대성은 공용 행동의 데이터 수정만 사용한다.
- 시작 세력 6개와 추가 세력 5개를 모두 구현한다.
- 대사·세계관·관계, 덱빌딩, 장비 강화·경제는 구현하지 않는다.

## 작업 1 — 공용 전투 엔진

- [ ] 10칸 위치·거리·중첩·교차·자리 교환을 구현한다.
- [ ] 행동력·기력·내공·체력과 6슬롯 행동열을 구현한다.
- [ ] 동시 잠금·동시 공개와 슬롯 해상 순서를 구현한다.
- [ ] 이동 뒤 사거리 판정, 일반 막기·회피·밀치기·무기 교체·재장전을 구현한다.
- [ ] 독·마화·장전·방패·보조장비의 공통 상태와 결과 로그를 구현한다.

## 작업 2 — 무기·문파·대성 데이터

- [ ] 권법·검·도·곤·창·암기·활·창방패의 무기 데이터를 구현한다.
- [ ] 화산·개방·당문·소림·무당·마교의 1~5성 규칙을 구현한다.
- [ ] 양가장·백우궁·관군·아미·오독의 1~5성 규칙을 구현한다.
- [ ] 모든 문파 효과를 엔진 전용 분기 없이 공용 수정자로 연결한다.
- [ ] 암기벨트·화살통·방패의 장비 호환과 제한 횟수를 구현한다.

## 작업 3 — 9전 대회 회차

- [ ] 시작 6세력 선택 화면과 초기 무공을 구현한다.
- [ ] 내부전·추천패·예선·본선·준결승·결승의 9전 대진을 구현한다.
- [ ] 11세력 상대 풀과 반복 방지 규칙을 구현한다.
- [ ] 무공서·수련도·보조장비·회복·정탐 보상 선택을 구현한다.
- [ ] 외문 무공 습득, 1~5성 수련, 대성 효과를 구현한다.
- [ ] 패배 후 재도전 1회, 최초 무료·이후 명망 소모, 재기 보상을 구현한다.

## 작업 4 — 정보와 UI

- [ ] 대진 단계·다음 상대·정탐 정보·대성 진행을 표시한다.
- [ ] 행동 배치 시 비용·사거리·재장전·상태·유파 수정값을 표시한다.
- [ ] 공개 뒤 양측 행동열과 슬롯별 결과를 설명한다.
- [ ] 로그에 공용 행동 → 무기 → 유파/대성 → 결과 순서를 보장한다.
- [ ] AI의 주요 행동을 공개 정보 기준으로 설명한다.

## 작업 5 — 테스트와 밸런스 기록

- [ ] 공용 전장·동시 선택 항목을 [테스트 체크리스트](../docs/08_TEST_CHECKLIST.md)로 검증한다.
- [ ] 11세력 각각을 AI 또는 플레이어로 전투 종료까지 검증한다.
- [ ] 시작 6세력 각각으로 본선과 결승 도달을 검증한다.
- [ ] 각 시작 세력의 승리·패배·재도전 사례와 결과 로그를 기록한다.
- [ ] 문파별 거리·자원·행동열 선택이 실제로 달라지는지 플레이 테스트로 기록한다.

## 완료 기준

- [ ] 체크리스트의 공용 전장·무기·11세력·대진·재도전 필수 항목이 통과했다.
- [ ] 시작 6세력 모두 시작 가능하고, 추가 5세력 모두 적·습득 대상으로 동작한다.
- [ ] 9전 회차가 본선과 결승까지 중단 없이 진행된다.
- [ ] 유파 무공·대성 추가 때문에 공용 엔진에 문파 전용 행동 분기가 생기지 않았다.
- [ ] 패배 뒤 다음 회차에서 다른 문파·수련·행동열을 시험하고 싶다는 플레이 이유를 기록했다.
