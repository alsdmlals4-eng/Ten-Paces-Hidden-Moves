# 십보강호: 숨은 수의 비무
## Ten Paces: Hidden Moves

10칸 전장에서 양측이 6슬롯 행동열을 비공개로 동시에 선택하고, 거리·행동력·기력·내공을 읽어 승부하는 1대1 무협 로그라이트입니다.

## 현재 구현 범위 — 대회 세로 슬라이스

이번 구현은 단일 결투가 아니라, 시작 문파 선택부터 내부전·예선·본선·결승까지 이어지는 **9전 대회 회차**입니다.

- 시작 선택 6세력: 화산, 개방, 당문, 소림, 무당, 마교
- 추가 구현 6세력: 양가장, 백우궁, 관군, 아미, 오독, 녹림채(도적·산적 사파)
- 전 세력은 무기 규칙, 유파 규칙, 대성 단계, AI 운용 규칙을 가진다.
- 한 회차에는 9전만 있으므로 전 세력을 한 번에 만나지는 않는다. 대신 모든 세력은 적·습득 대상 또는 시작 선택으로 실제 전투에 사용된다.
- 모든 경기는 A=4번 칸, B=6번 칸(초기 거리 2)에서 시작한다.

검증할 것은 다음입니다.

> 서로 다른 문파를 골라 내부전부터 본선·결승까지 올라갈 때, 전투 규칙과 수련 선택이 매 회차 다른 전략적 재도전 욕구를 만드는가?

## 전투의 층

| 층 | 역할 |
|---|---|
| 공용 행동 | 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전 |
| 무기 규칙 | 사거리, 비용, 막기·밀치기 효율, 재장전 |
| 보조장비 | 암기벨트·화살통 등 제한 횟수의 무행동 효과 |
| 유파 무공 | 공용 행동의 조건·효율·결과를 바꾸는 문파 고유 규칙 |
| 대성 | 문파 무공을 수련 단계별로 해금하는 장기 성장 |
