# 십보강호 (Ten Paces: Hidden Moves)

> 10칸 전장에서 양측이 행동열을 비공개로 배치하고 동시에 공개하는 1대1 무협 로그라이트 전술 게임.

## 현재 검증 목표

**소림 대 당문 한 판을 끝낸 플레이어가 다른 행동열로 즉시 재대결하고 싶어 하는가?**

현재 PoC는 A 소림이 4번 칸, B 당문이 6번 칸에서 시작하는 초기 거리 2의 결투만 구현한다.

## 핵심 약속

- 반사신경보다 거리, 자원, 행동 순서를 읽는 전술 결투
- 공용 행동을 뼈대로 하고 무기·보조장비·유파 무공이 그 행동을 변형
- 카드 손패 운보다 장착 규칙과 수련 선택에 따른 빌드 전략
- 승패의 이유가 항상 보이는 공정한 AI 싱글플레이

## 현재 구현 범위

- 소림: 권법, 금강호신
- 당문: 암기, 독침, 암기벨트
- 공용 행동: 접근, 후퇴, 공격, 막기, 회피, 밀치기, 재장전
- 전장: 일자형 10칸, 같은 칸 위치와 상호 통과 허용
- 라운드: 공통 6시간 슬롯, 양측 비공개 선택 후 동시 공개
- 자원: 행동력, 기력, 내공

## 보류 범위

문파 대량 추가, 무공 20개 이상, 세계관·대화·관계, 랜덤 손패·덱빌딩, 장비 강화·경제 시스템은 PoC 통과 뒤에만 검토한다.
