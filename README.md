# 십보강호 (Ten Paces: Hidden Moves)

> 10칸 전장에서 양측이 행동열을 비공개로 배치하고 동시에 공개하는 1대1 무협 로그라이트 전술 게임.

## 현재 상태

기획 확정 단계입니다. Godot 구현, 아트, 대사·인물 관계 시스템은 아직 범위에 포함하지 않습니다.

## 핵심 약속

- 반사신경보다 거리, 자원, 행동 순서를 읽는 전술 결투
- 카드 손패 운보다 장착 무공과 대성 선택에 따른 빌드 전략
- 승패의 이유가 항상 보이는 공정한 AI 싱글플레이
- 화산·개방·당문·소림·무당·마교 중 하나로 시작해, 적 문파의 무공까지 조합하는 반복 성장

## 현재 확정 범위

- 전장: 일자형 10칸, 같은 칸 위치와 상호 통과 허용
- 라운드: 공통 6시간 슬롯, 양측 비공개 선택 후 동시 공개
- 자원: 행동력, 기력, 내공
- 성장: 회차 내 무공 1~5성 대성, 회차 종료 시 초기화
- 영구 성장: 명망을 통한 수평 해금과 제한적 기초 단련
- 회차: 평균 60분, 9전, 패배 시 회차당 1회 재도전 가능
