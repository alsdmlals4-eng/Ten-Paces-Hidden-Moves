# 십보강호: 숨은 수의 비무
## Ten Paces: Hidden Moves

10칸 전장에서 양측이 행동 두 개를 비공개로 잠그고 동시에 공개하며, 거리·합·행동력·기력·내공을 읽어 승부하는 1대1 무협 로그라이트입니다.

## 현재 구현 범위 — 대회 세로 슬라이스

전체판은 시작 문파 선택부터 내부전·예선·본선·8강·준결승·결승까지 이어지는 **10전 대회 회차**입니다.

- 전장 10칸, 라운드 행동 타이밍 10개, 대회 10전의 `10-10-10` 구조를 사용합니다.
- 시작 선택 6세력: 화산, 개방, 당문, 소림, 무당, 마교
- 추가 구현 6세력: 양가장, 백우궁, 관군, 아미, 오독, 녹림채
- 한 라운드는 10타이밍이며 2타이밍씩 묶어 총 5번 선택합니다.
- 같은 타이밍에 양측 공격이 성립하면 합을 겨루고 낮은 쪽이 합 수치 차이만큼 피해를 받습니다.
- 핵심무공은 여러 개 익힐 수 있고, 심법은 최대 2개까지 보유할 수 있습니다.
- 핵심무공은 3·6·9성에 기술, 10성에 절초를 얻습니다.
- 심법은 패시브·문파 기믹·명상 효과를 제공하고 10성에 진의를 얻습니다.
- 경기 승리 후 기본 수련 포인트 5와 성과 등급·확률·제약 보너스를 분배합니다.
- 모든 경기는 A=4번 칸, B=6번 칸에서 시작합니다.
- 결승은 10전째이며, 결승 전까지 최대 9회의 수련 단계를 거칩니다.

## 데모 범위

데모는 **1~5전**, 5전째 **지역 예선 결승**까지 구성합니다.

- 1~2전: 내부전
- 3전: 추천패 쟁탈전
- 4전: 지역 예선
- 5전: 지역 예선 결승
- 첫 4전을 모두 A등급 이상으로 승리하고 수련 포인트를 핵심무공에 집중하면, 4전 종료 후 절초를 열어 5전에서 사용할 수 있습니다.
- B등급 중심 진행에서는 예선 결승 전 절초 개방이 보장되지 않습니다.

검증할 것은 다음입니다.

> 서로 다른 문파·공용 무공·심법 조합으로 10전을 진행할 때, 2수 예측과 합, 수련, 제약 선택이 매 회차 다른 전략적 재도전 욕구를 만드는가?

## 전투와 성장의 층

| 층 | 역할 |
|---|---|
| 공용 행동 | 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전, 명상 |
| 2수 선택 | 행동 두 개 사전 잠금, 동시 공개, 1수→2수 순차 해상 |
| 합 | 쌍방 공격의 최종 합 수치 비교와 차이 피해 |
| 무기 규칙 | 사거리, 기본 합, 비용, 막기·밀치기, 재장전 |
| 공용·문파 핵심무공 | 3·6·9성 기술과 10성 절초 |
| 심법 | 패시브, 기믹 생성·강화, 명상 효과와 10성 진의 |
| 수련·제약 | 성과 포인트 분배와 무협식 선택 난이도·추가 보상 |
