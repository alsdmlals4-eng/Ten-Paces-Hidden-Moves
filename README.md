## 현재 상태

- 엔진: **Godot 4.7.1 stable**.
- 단계: 다섯 활성 기획서 체계 정비 + T0 전투 프로토타입 실행 준비.
- 포함된 프로토타입: 10칸 전장, AI 선잠금, 두 수 동시 공개, 이동, 공격, 막기, 회피, 명상, 합, 절초 기세·발동 프로토타입.
- 아직 미검증: 실제 Godot headless 실행, Windows 수동 화면·입력 플레이테스트.
- 확정 콘텐츠 방향: 10문파, 핵심무공 8종, 보조무공 8종, 문파 전용 무공서 28개.
- 숙련 구조: 3성 기술 1, 5성 기술 1 강화, 7성 기술 2, 9성 기술 2 강화, 10성 절초·진의.
- 기술 효과·비용·밸런스와 절초 기세의 정확한 수치는 진행 중입니다.

## 게임 방향

- 전장 10칸, 라운드 행동 타이밍 10개, 전체 대회 10전의 `10-10-10` 구조.
- 한 라운드는 행동 두 개씩 묶어 총 5번 잠금·동시 공개·순차 해상.
- 같은 타이밍에 양측 공격이 유효하면 `[합]`을 겨루고 낮은 쪽이 차이만큼 피해.
- 상대 내부 수치는 이명·풍문·전적·정탐으로 암시하며 개발용 태그를 직접 노출하지 않음.
- 핵심무공 10성으로 절초를 해금하고, 전투 중 절초 기세 조건을 충족해야 사용 가능.
- 보조무공 10성은 절초가 아니라 진의를 해금.

## 작업 규칙

- 모든 작업에서 영향받은 활성 기획서를 같은 작업과 같은 PR로 갱신합니다.
- 이미지 작업은 아트 기획서와 이미지 인덱스를 반드시 함께 갱신합니다.
- 사용자가 빠르게 이해할 수 있는 가독성과 새 작업자의 재현 가능성을 유지합니다.
- 생성 이미지의 한글·수치·분류를 기준 데이터로 사용하지 않습니다.
- `v2`, `final`, `latest` 복제본을 만들지 않고 현재 문서를 직접 갱신합니다.
- 자동 검사나 실제 검증이 실패한 상태를 완료로 보고하지 않습니다.
