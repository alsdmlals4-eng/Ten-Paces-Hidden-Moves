# 십보강호: 숨은 수의 비무
## Ten Paces: Hidden Moves

10칸 전장에서 양측이 행동 두 개를 비공개로 잠그고 동시에 공개하며, 거리·합·행동력·기력·내공을 읽어 승부하는 1대1 무협 로그라이트입니다.

## 현재 방향

- 전장 10칸, 라운드 행동 타이밍 10개, 전체 대회 10전의 `10-10-10` 구조입니다.
- 한 라운드는 행동 두 개씩 묶어 총 5번 잠금·동시 공개·순차 해상합니다.
- 같은 타이밍에 양측 공격이 유효하면 `[합]`을 겨루고 낮은 쪽이 차이만큼 피해를 받습니다.
- 내부 시스템은 정확한 수치와 데이터로 설계하지만, 상대의 강도·성장·수련 성향은 이명·풍문·전적·정탐 문장으로 암시합니다.
- 작업자가 바뀔 때는 [`AGENTS.md`](AGENTS.md) → [활성 컨텍스트](docs/ACTIVE_CONTEXT.md) → [문서 지도](docs/DOCUMENTATION_MAP.md) 순서로 읽습니다.

## 데모 범위

데모는 **1~5전**, 5전째 **예선 결승**까지 구성합니다.

- 1~2전: 내부전
- 3~4전: 예선
- 5전: 예선 결승·데모 최종전
- 시작 핵심무공과 심법은 각각 3성입니다.
- 핵심무공은 3·6·9성에 기술, 10성에 절초를 얻습니다.
- 심법은 패시브·기믹·명상 효과를 제공하고 10성에 진의를 얻습니다.
- 핵심무공은 여러 개 익힐 수 있고 심법은 최대 2개까지 보유할 수 있습니다.

## 수련과 행운

```text
최종 수련 포인트 = 승리 기본 5 + 등급 보너스 + 행운 앞면 수
```

| 등급 | 등급 보너스 |
|---|---:|
| C | +1 |
| B | +3 |
| A | +5 |
| S | +7 |

- 행운 다이스는 3개이며 개별 면은 비공개입니다.
- 등급별 결과표에서 앞면 0~3개를 추첨하고 `B++`처럼 `+` 개수만 표시합니다.
- A는 의도한 평균 성과입니다.
- 첫 4전이 모두 A일 때 핵심무공에 전부 투자하면 약 절반의 회차에서 5전 예선 결승 전에 절초를 개방하는 것을 목표로 합니다.
- 절초는 데모 필수 조건이 아니라 상위 성과와 행운으로 먼저 체험하는 하이라이트입니다.

## 상대 성장과 제약

- 상대는 A계열 60%, B계열 40%를 기준으로 생성합니다.
- 상대도 행운 `+`를 추첨해 실제 수련 포인트를 얻습니다.
- 균형형, 핵심무공 올인형, 심법 연계형, 다무공형 등으로 포인트를 분배합니다.
- 상대의 등급·행운·투자 결과는 회차 생성 시 고정되어 재도전에서도 바뀌지 않습니다.
- 플레이어에게 `[강적] [A++] [올인형]` 같은 개발용 태그를 직접 노출하지 않습니다.
- 이명·풍문·전적·정탐을 통해 상대를 읽고 `삼수양보` 같은 제약을 선택합니다.

## 전체판 구조

- 6~7전: 본선
- 8전: 8강
- 9전: 준결승
- 10전: 결승
- 전체판 결승 전까지 최대 9회의 수련 단계를 거칩니다.
