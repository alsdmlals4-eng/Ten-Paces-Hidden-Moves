# 십보강호 전투 규칙서

## 1. 전장과 거리

- 전장은 일자형 10칸이다.
- 같은 칸에 두 전투원이 함께 위치할 수 있다.
- 양측 이동 경로가 겹치면 서로 통과해 위치를 교환한다.
- 전장 밖으로는 이동할 수 없다.
- 실행 시점에는 상대 방향을 자동으로 바라본다.
- 거리 0은 같은 칸, 거리 1은 인접 칸이다.

| 계통 | 기본 사거리 |
|---|---:|
| 권법·장법 | 0 |
| 검·도 | 1 |
| 창·곤·극 | 2 |
| 활 | 3 이상 |
| 암기 | 2 이상 |
| 방패 타격 | 0~1 |

## 2. 라운드와 행동열

- 양측은 동일한 6시간 슬롯을 사용한다.
- 라운드 시작 시 행동력은 회복한다.
- 양측은 행동열을 비공개로 배치한 뒤 동시에 잠근다.
- 잠금 뒤 전체 행동열을 동시에 공개하고, 1번부터 6번 슬롯을 순서대로 실행한다.
- 공개 뒤 행동열을 수정할 수 없다.

행동력은 시간 슬롯과 다르다. 시간 슬롯은 모두 6칸으로 같고, 행동력은 이번 라운드에 배치할 행동의 총량이다. 무거운 행동은 행동력·기력·내공·준비 또는 회수 슬롯을 더 요구한다.

## 3. 슬롯 하나의 판정 순서

1. 준비 상태 적용
2. 양측 이동 동시 처리
3. 막기·회피·호신 상태 적용
4. 이동 후 실제 거리로 양측 공격과 밀치기 판정
5. 피해와 상태이상 동시 적용
6. 회수·장전·기력 회복 처리

양측 공격은 누가 먼저 공격했는지로 처리하지 않는다. 같은 슬롯에 사거리 안이라면 서로 피해를 준다.

## 4. 자원

| 자원 | 역할 | 회복 |
|---|---|---|
| 행동력 | 이번 라운드의 행동 배치 한도 | 라운드 시작 시 전부 회복 |
| 기력 | 무기 공격·보법·연속 행동 비용 | 대기·호흡·일부 심법으로 회복, 라운드를 넘어 유지 |
| 내공 | 경공·심법·호신강기·강화 초식 비용 | 심법 중심으로 회복, 기력보다 느림 |

경공은 행동력 1을 사용하고 내공을 소모한다. 일반 보법보다 먼 거리를 한 슬롯에 이동할 수 있다.

## 5. 공통 행동

### 이동

기본 보법은 1칸 이동한다. 경공과 무공은 2칸 이상 이동하거나 교차 후 효과를 줄 수 있다.

### 막기와 회피

- 막기는 누구나 사용할 수 있으며 같은 슬롯의 직접 피해를 기력으로 경감한다.
- 회피보는 이동 뒤에도 사거리 안에 남았을 때 추가 피해 경감을 제공한다.
- 방패는 막기 효율, 특히 투사체 피해 경감을 높인다.
- 방패 장착 시 회피보의 피해 경감은 낮아지고 기력·내공 비용이 증가한다.
- 방패술은 방패 타격·방패 자세·밀치기 강화 같은 고급 행동을 해금한다.

### 밀치기

- 모든 무기는 공통 밀치기를 사용할 수 있다.
- 밀치기 사거리는 현재 무기의 공격 사거리를 따른다.
- 적중하면 적을 1칸 밀어낸다.
- 전장 끝에서는 이동 대신 기력이 추가로 감소한다.
- 거리 0 밀치기는 배치할 때 왼쪽 또는 오른쪽 방향을 선택한다.
- 같은 슬롯의 양측 밀치기는 동시에 처리한다.
- 기본 밀치기는 피해가 없거나 매우 낮으며, 거리 변화가 주 보상이다.

### 무기 교체

- 전투 준비 때 주병장과 예비 병장을 하나씩 고른다.
- 교체는 시간 슬롯 1칸과 행동력 1을 사용한다.
- 교체 슬롯에서는 공격·막기·밀치기를 함께 사용할 수 없다.
- 다음 슬롯부터 새 무기의 사거리와 무공을 쓴다.
- 관련 무공서가 없으면 새 무기의 기본 공격과 공통 밀치기만 쓸 수 있다.

## 6. 무기별 박자

- 권법: 낮은 행동력 비용으로 거리 0 연계. 접근 실패가 약점.
- 검·도: 거리 1 균형형. 검은 연계·추격, 도는 기력 대가의 강한 압박.
- 창·곤: 거리 2 지배. 창은 회수와 기력, 곤은 제압과 밀치기에 강점.
- 활: 장거리 압박. 장전 공백과 전장 끝이 약점.
- 암기: 빠른 견제와 상태이상. 위력·횟수 제한이 약점.
- 방패: 투사체 상성, 막기와 밀치기 강화. 회피와 기동성이 약점.

## 7. 제한 횟수 보조병장

### 암기벨트

- 시간 슬롯과 행동력을 쓰지 않는 즉발 암기를 정해진 횟수만큼 제공한다.
- 이미 배치한 행동 하나에 붙여 쓰며, 한 슬롯에는 1회만 사용한다.
- 일반 공격과 같은 공격 판정 단계에서 발동한다.
- 강한 밀치기·점혈·대형 독무는 즉발 암기로 사용하지 않는다.

### 화살통

- 정해진 즉시 장전 횟수 동안 사격 후 장전 슬롯을 쓰지 않는다.
- 사격 자체의 행동력·기력 비용은 그대로다.
- 횟수를 다 쓰면 다시 장전 행동이 필요하다.
- 남은 횟수는 양측에 공개한다.

## 8. 마공과 마화

- 마공은 체력을 직접 소모해 강한 효과를 낸다.
- 마공 1회마다 마화 1중첩, 최대 3중첩이다.
- 라운드 종료 시 마화 중첩에 비례해 체력이 감소하고 마화는 전부 제거된다.
- 체력이 사용 비용보다 낮으면 해당 마공을 배치할 수 없다.
- 마화 정산으로 체력이 0이 되면 패배한다.
- 마화 외에 별도 마기 자원은 없다.

## 9. AI 공정성

- AI는 라운드 시작 상태의 공개 정보만 사용해 행동열을 생성·잠금한다.
- 플레이어가 행동열을 잠근 뒤 AI가 수정하는 것은 금지한다.
- 난이도는 정보 치팅이 아니라 예측 깊이와 실수 확률로만 조정한다.
- 상대 정보는 무기, 사거리, 대표 무공, 성향, 선호 거리부터 공개한다.
- 정탐과 영구 기록은 대표 연계·약점 슬롯·자원 부족 행동 경향을 추가 공개할 수 있다.
- 전투 후에는 적중·실패·밀치기·마화의 원인을 슬롯별로 요약한다.
