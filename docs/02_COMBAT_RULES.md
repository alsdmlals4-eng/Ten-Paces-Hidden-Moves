# 십보강호 전투 규칙서

## 1. 전장과 거리

- 전장은 일자형 10칸이다.
- 같은 칸에 두 전투원이 함께 위치할 수 있다.
- 양측 이동 경로가 겹치면 서로 통과해 위치를 교환한다.
- 전장 밖으로는 이동할 수 없다.
- 실행 시점에는 상대 방향을 자동으로 바라본다.
- 거리 0은 같은 칸, 거리 1은 인접 칸이다.

| 계통 | 기본 사거리 |
|---|---:|
| 권법·장법 | 0 |
| 검·도 | 1 |
| 창·곤·극 | 2 |
| 활 | 3 이상 |
| 암기 | 2 이상 |
| 방패 타격 | 0~1 |

## 2. 라운드와 2수 동시 선택

- 한 라운드는 10타이밍이다.
- 2타이밍을 한 묶음으로 하여 라운드마다 총 5번 행동을 선택한다.
- 각 묶음이 시작되면 AI는 현재 공개 정보만 보고 행동 두 개를 먼저 잠근다.
- 플레이어도 행동 두 개를 비공개로 선택하고 잠근다.
- 양측 행동 두 개를 동시에 공개한 뒤 첫 번째 타이밍과 두 번째 타이밍 순으로 해상한다.
- 첫 번째 타이밍 결과를 본 뒤 두 번째 행동을 변경할 수 없다.
- 다음 묶음은 이전 두 타이밍의 공개 결과가 모두 반영된 상태에서 시작한다.

행동력은 타이밍 수와 다르다. 모든 전투원은 같은 10타이밍을 사용하지만, 행동력은 한 라운드에 사용할 수 있는 행동의 총량이다. 무거운 기술은 행동력·기력·내공을 더 요구한다.

## 3. 타이밍 하나의 판정 순서

1. 양측 행동 가능 여부와 비용 확인
2. 준비 상태와 부착형 보조장비 적용
3. 양측 이동 동시 처리
4. 막기·회피·호신 상태 적용
5. 이동 후 실제 거리로 공격·밀치기 성립 여부 확인
6. 양측 공격이 모두 성립하면 합 판정
7. 한쪽 공격만 성립하면 일반 공격 판정
8. 피해와 상태이상 동시 적용
9. 회수·장전·명상·자원 회복 처리

## 4. 합 판정

- 같은 타이밍에 양측 공격이 모두 유효하고 서로 사거리 안이면 개별 공격 대신 합 판정을 한다.
- 각 공격의 최종 합 수치는 기본 공격값에 무기·성급·심법·상태 수정치를 적용해 계산한다.
- 합 수치가 높은 쪽이 승리하고, 낮은 쪽은 두 최종 합 수치의 차이만큼 체력이 감소한다.
- 합 수치가 같으면 양측 모두 체력 피해를 받지 않는다.
- 여러 차례 공방을 주고받는 장면은 결과를 표현하는 연출이며 실제 계산은 한 번만 한다.
- 한쪽 공격이 사거리 밖이거나 자원 부족으로 무효가 되면 합은 발생하지 않고 유효한 공격만 일반 판정한다.
- 막기·회피와 공격이 마주친 경우에는 합이 아니라 일반 공격 대 방어 판정을 사용한다.

### 합 특화 수정자

- 태극권과 같은 반격 무공은 합이 발생했을 때 상대 공격의 일부를 자신의 합 수치에 더하는 수정자를 가질 수 있다.
- 반격 보너스는 상대가 실제 공격을 사용했을 때만 적용한다.
- 합 보너스와 피해 차이 경감은 공용 수정자로 처리하고 문파 전용 해상 절차를 만들지 않는다.

## 5. 자원

| 자원 | 역할 | 회복 |
|---|---|---|
| 행동력 | 이번 라운드의 행동 사용 한도 | 10타이밍 종료 시 전부 회복 |
| 기력 | 무기 공격·보법·연속 행동 비용 | 명상·대기·일부 심법과 라운드 종료 시 소량 회복 |
| 내공 | 경공·심법·호신·강화 초식 비용 | 명상·심법과 라운드 종료 시 기력보다 적게 회복 |

경공은 행동력과 내공을 사용해 일반 보법보다 먼 거리를 한 타이밍에 이동한다.

## 6. 공통 행동

### 이동

기본 보법은 1칸 이동한다. 경공과 일부 무공은 2칸 이상 이동하거나 교차 후 효과를 제공한다.

### 막기와 회피

- 막기는 누구나 사용할 수 있으며 같은 타이밍의 직접 피해를 기력으로 경감한다.
- 회피보는 이동 뒤에도 사거리 안에 남았을 때 추가 피해 경감을 제공한다.
- 방패는 막기 효율, 특히 투사체 피해 경감을 높인다.
- 방패 장착 시 회피보 효율은 낮아지고 기력·내공 비용이 증가할 수 있다.

### 밀치기

- 모든 무기는 공통 밀치기를 사용할 수 있다.
- 밀치기 사거리는 현재 무기의 공격 사거리를 따른다.
- 적중하면 적을 1칸 밀어낸다.
- 전장 끝에서는 이동 대신 기력이 추가로 감소한다.
- 거리 0 밀치기는 배치할 때 왼쪽 또는 오른쪽 방향을 선택한다.
- 같은 타이밍의 양측 밀치기는 동시에 처리한다.

### 무기 교체

- 전투 준비 때 주병장과 예비 병장을 하나씩 고른다.
- 교체는 타이밍 하나와 행동력을 사용한다.
- 교체 타이밍에는 공격·막기·밀치기를 함께 사용할 수 없다.
- 다음 타이밍부터 새 무기의 사거리와 사용 가능한 무공을 적용한다.

### 명상

- 명상은 공격하지 않고 기력·내공을 회복하는 공용 행동이다.
- 심법은 명상의 회복량, 상태 유지, 다음 기술 강화 효과를 수정할 수 있다.

## 7. 무기별 박자

- 권법: 낮은 행동력 비용으로 거리 0 연계. 접근 실패가 약점.
- 검·도: 거리 1 균형형. 검은 연계·추격, 도는 기력 대가의 압박.
- 창·곤: 거리 2 지배. 창은 회수와 기력, 곤은 제압과 밀치기에 강점.
- 활: 장거리 압박. 장전 공백과 전장 끝이 약점.
- 암기: 빠른 견제와 상태이상. 위력·횟수 제한이 약점.
- 방패: 투사체 상성, 막기와 밀치기 강화. 회피와 기동성이 약점.

## 8. 제한 횟수 보조병장

### 암기벨트

- 정해진 횟수 동안 기존 행동에 즉발 암기를 부착한다.
- 한 타이밍에는 1회만 사용할 수 있다.
- 일반 공격과 같은 공격 판정 단계에서 발동한다.

### 화살통

- 정해진 즉시 장전 횟수 동안 사격 후 별도 재장전 행동을 면제한다.
- 사격 자체의 행동력·기력 비용은 그대로다.
- 남은 횟수는 양측에 공개한다.

## 9. 마공과 마화

- 마공은 체력을 직접 소모해 강한 효과를 낸다.
- 마공 1회마다 마화 1중첩, 최대 3중첩이다.
- 라운드 종료 시 마화 중첩에 비례해 체력이 감소하고 마화는 제거된다.
- 체력이 사용 비용보다 낮으면 해당 마공을 선택할 수 없다.
- 마화 정산으로 체력이 0이 되면 패배한다.

## 10. AI 공정성

- AI는 각 2수 묶음이 시작되는 시점의 공개 정보만 사용한다.
- AI는 플레이어의 비공개 행동을 보기 전에 자신의 두 행동을 잠근다.
- 플레이어가 잠근 뒤 AI가 두 행동 중 하나라도 수정하는 것은 금지한다.
- 난이도는 정보 치팅이 아니라 예측 깊이, 성향, 실수 확률로 조정한다.
- 전투 후에는 이동, 공격 실패, 합 수치 구성, 피해 차이, 상태 효과의 원인을 타이밍별로 설명한다.
