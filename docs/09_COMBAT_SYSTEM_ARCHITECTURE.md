# 전투 시스템 계층 구조

> 원칙: 공용 행동이 전투의 뼈대이고, 무기·보조장비·유파 무공·대성은 그 행동의 조건·효율·결과만 바꾼다.

## 1. 책임 분리

| 계층 | 책임 | 예시 |
|---|---|---|
| 공용 행동 | 모든 전투원이 사용할 수 있는 기본 선택 | 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전 |
| 무기 규칙 | 공용 행동의 사거리·기본 비용·기본 효율 | 권법 거리 0, 암기 거리 2~4, 방패 막기 강화, 활 재장전 필요 |
| 보조장비 | 제한 횟수 또는 조건부로 공용 행동에 붙는 효과 | 암기벨트 즉발 암기, 화살통 즉시 장전 |
| 유파 무공 | 특정 공용 행동의 조건·비용·결과를 변형 | 소림 호신공이 막기 효율 강화, 당문 독공이 공격에 독 부여 |
| 대성 | 수련 단계에 따라 다른 유파 무공과 변형 규칙을 해금 | 대성 단계 상승으로 보조 무공·고급 변형 선택지 해금 |

## 2. 공용 행동

| 행동 | 기본 역할 | 무기·무공이 바꾸는 항목 |
|---|---|---|
| 접근·후퇴 | 1칸 이동 | 이동 거리, 비용, 교차 뒤 효과 |
| 공격 | 현재 무기 사거리 안의 적에게 피해 | 사거리, 피해, 기력 비용, 상태이상, 장전 조건 |
| 막기 | 같은 슬롯 직접 피해 경감 | 경감률, 기력 손실, 반격·호신 효과 |
| 회피 | 이동과 추가 피해 경감 | 이동 거리, 경감률, 자원 비용 |
| 밀치기 | 적을 1칸 이동시키는 공간 제어 | 사거리, 기력 압박, 부가 피해 |
| 무기 교체 | 예비 병장으로 전환 | 교체 뒤 사용 가능한 공용 공격·막기 |
| 재장전 | 미장전 무기를 다시 준비 | 필요 여부, 비용, 즉시 장전 장비 |

공용 행동은 유파에 따라 삭제되지 않는다. 유파 무공은 공용 행동을 더 잘 쓰게 하거나 특별한 조건을 붙인다.

## 3. 불변 규칙

- 전장은 10칸이며 같은 칸 위치와 상호 통과를 허용한다.
- 모든 라운드는 공통 6시간 슬롯으로 진행한다.
- 공격과 밀치기는 이동 처리 뒤의 실제 거리로 판정한다.
- 이미 배치한 행동을 취소·강제 변경하는 효과는 초기 범위에서 만들지 않는다.
- 확률 명중·확률 회피를 만들지 않는다.
- 새 무공은 새 전투 단계나 예외 판정을 만들지 않고, 기존 공용 행동 데이터의 값과 조건만 변형한다.

## 4. PoC 적용: 소림 대 당문

| 항목 | 소림 | 당문 |
|---|---|---|
| 무기 규칙 | 권법: 공격·밀치기 사거리 0, 막기 기본 효율 40% | 암기: 공격·밀치기 사거리 2~4, 공격 뒤 재장전 필요, 막기 기본 효율 25% |
| 유파 규칙 | 금강호신: 막기 경감률 +15% | 독침: 암기 공격 적중 시 독 1중첩 |
| 보조장비 | 없음 | 암기벨트: 3회까지 행동 소모 없는 즉발 암기 |
| 요구되는 판단 | 언제 거리 0으로 접근할지, 막을지 | 언제 후퇴·공격·재장전·암기벨트를 쓸지 |

초기 구현의 질문은 하나다.
