# 구현 콘텐츠 카탈로그

> 이번 세로 슬라이스에서는 아래 12세력을 모두 구현한다. ‘구현’은 각 세력이 전투에서 선택·적·습득 대상 중 하나로 작동하고, 대표 무기·유파 규칙·대성 1~5성을 갖는다는 뜻이다.

| 세력 | 주 무기·거리 | 주 계통 | 공용 행동을 바꾸는 핵심 |
|---|---|---|---|
| 화산파 | 검, 1 | 검법·보법 | 이동 뒤 검 공격과 회피의 추격 효율 |
| 개방 | 곤, 2 | 곤법·호신공 | 밀치기와 막기 후 반격 기회 |
| 당문 | 암기, 2~4 | 암기·독공 | 재장전, 독, 암기벨트의 무행동 사용 |
| 소림 | 권법, 0 | 권법·호신공 | 막기 강화와 거리 0 압박 |
| 무당파 | 검, 1 | 검법·심법·보법 | 내공을 써서 회피·막기 결과 전환 |
| 마교 | 도, 1 | 도법·마공 | 체력을 대가로 강해지는 마화 |

| 녹림채(사파) | 도, 1 | 도법·외공 | 공격 적중으로 상대 기력을 빼앗는 약탈 전법 |

## 공용 구현 기준

- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전이다.
- 모든 무기는 사거리, 행동력·기력 비용, 막기 효율, 밀치기 효율, 재장전 필요 여부를 데이터로 가진다.
- 방패가 없어도 막기는 가능하다. 방패는 특히 활 피해 경감을 높이고 회피 효율을 낮춘다.
- 암기벨트·화살통은 제한 횟수 동안 행동 슬롯·행동력 없이 기존 행동에 붙는 보조장비다.
- 유파 무공은 새 전투 절차를 만들지 않고 공용 행동의 조건·비용·효율·결과만 바꾼다.
- 대성은 모든 세력에 1~5성으로 구현하고, 다른 세력 무공은 보상으로 습득해 해당 단계부터 수련한다.
- 녹림채는 도적·산적을 포괄하는 사파 세력이며, 대진 적과 습득 가능한 무공서로 등장한다.
