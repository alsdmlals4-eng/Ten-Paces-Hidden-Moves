# 십보강호 문파·무공·장비 카탈로그

## 1. 시작 세력 6종

| 세력 | 핵심 계통 | 전투 정체성 |
|---|---|---|
| 화산파 | 검법·심법·보법 | 거리 1에서 교차·추격·연계를 만드는 균형형 |
| 개방 | 곤법·외공·호신공 | 밀치기와 기력 압박으로 자세를 무너뜨리는 제압형 |
| 당문 | 암기술·독공·의술 | 원거리 견제, 독 누적, 회복과 해독을 함께 다루는 교란형 |
| 소림 | 권법·호신공·외공 | 거리 0에서 버티며 연타·반격하는 정면 압박형 |
| 무당파 | 심법·보법·호신공·검법 | 경공과 흘리기로 피해를 전환하는 대응형 |
| 마교 | 도법·마공·심법·외공 | 체력과 마화를 대가로 폭발력을 만드는 고위험형 |

시작 시 세력의 무공 후보 3개 중 대표 무공 1개와 보조 무공 1개를 고른다.

## 2. 시작 세력 대표 무공

| 세력 | 대표 무공 | 1성 | 2성 | 5성·대성 |
|---|---|---|---|---|
| 화산파 | 매화검법 | 검 공격 강화 | 낙화: 거리 1 공격 후 조건부 이동 | 서로 다른 검 초식을 연계할수록 검격·이동 효율 상승 |
| 개방 | 개방곤법 | 곤 공격·막기 강화 | 봉압: 거리 1~2 공격, 적 기력 압박 | 밀치기·막기·곤격 연계로 상대 기력 회복 약화 |
| 당문 | 당문암기 | 암기 적중 시 독 부여 | 연환비: 거리 2~4 빠른 견제 | 독 중첩에 따라 암기 행동력·사거리 효율 강화 |
| 소림 | 나한권 | 권 공격·막기 강화 | 나한타: 거리 0 강한 권격 | 방어 성공과 권격 연계 시 금강 자세 강화 |
| 무당파 | 태극검경 | 내공 효율·흘리기 강화 | 운수: 공격을 흘려 피해·기력 손실 경감 | 흘리기·보법·검격을 이어 내공을 되돌려 받음 |
| 마교 | 혈염도법 | 잃은 체력에 비례해 도 공격 강화 | 혈염참: 체력 소모 거리 1 참격 | 마공 사용 수에 따라 폭발력이 오르고 종료 마화 피해도 증가 |

## 3. 시작 세력 보조 무공 후보

| 세력 | 후보 |
|---|---|
| 화산파 | 청풍심법, 낙화보 |
| 개방 | 취개외공, 유운호신공 |
| 당문 | 독경, 약왕의술 |
| 소림 | 금강호신공, 역근공 |
| 무당파 | 태극심법, 운학보 |

## 4. 추가 세력과 습득 무공

추가 세력은 적으로 먼저 등장하며, 승리 보상·특별 수련으로 무공서를 습득할 수 있다.

| 세력 | 대표 무공 | 계통 | 대성 방향 |
|---|---|---|---|
| 양가장 | 양가창법 | 창법·외공·보법 | 거리 2 유지와 밀치기 강화 |
| 백우궁 | 백우궁술 | 궁술·심법·보법 | 장전·사격·이동 완성 시 효율 상승 |
| 관군·표국 | 철위방패술 | 방패술·창법·의술 | 투사체 방어와 밀치기 후 추격 |
| 아미파 | 금정점혈 | 점혈·검법·의술 | 다음 라운드 자원 회복과 보법 비용 압박 |
| 오독교 | 오독경 | 독공·암기술·외공 | 독 중첩을 피해 또는 기력 약화로 전환 |
| 전진교 | 전진심법 | 심법·검법·보법 | 대기·운기 뒤의 무공 비용 절감 |
| 곤륜파 | 곤륜검결 | 검법·장병기·심법 | 거리 1과 2를 오가는 변칙 운용 |

## 5. 무공 분류

| 분류 | 역할 |
|---|---|
| 권법·장법 | 거리 0 연계와 붙기 |
| 검법·도법 | 거리 1의 균형·압박 |
| 창법·곤법 | 거리 2 견제·밀치기·제압 |
| 궁술·암기술 | 장거리 압박과 상태이상 |
| 외공 | 체력·공격력·방어력·기력 강화 |
| 호신공 | 막기·반격·상태이상 저항 |
| 심법 | 내공 총량·회복·운기 효율 |
| 보법 | 이동·교차·회피·거리 선점 |
| 독공·의술 | 지속 피해·해독·회복 |
| 점혈·금나수 | 자원 압박과 위치 제어 |
| 마공 | 체력 대가의 폭발력 |

## 6. 장비

### 장비 칸

| 칸 | 역할 |
|---|---|
| 주병장 | 기본 공격 방식과 사거리 |
| 보조병장 | 주병장의 약점 보완 또는 특수 행동 |
| 호신구 | 생존·상태이상 대응 |
| 기물 | 자원 운용과 특수 규칙 |

### 대표 장비

| 계통 | 장비 | 효과 |
|---|---|---|
| 검 | 유운검집 | 보법 후 첫 검격 기력 비용 감소 |
| 도 | 철환도 | 강한 도 공격 강화, 막기 기력 비용 증가 |
| 창 | 유연 창대 | 거리 0에서도 약한 창격 사용 가능 |
| 곤 | 쇠심곤 | 밀치기 강화, 행동력 비용 증가 |
| 활 | 속사 화살통 | 정해진 횟수만큼 즉시 장전 |
| 암기 | 기연독낭 | 독 중첩 강화, 의술 회복량 감소 |
| 권법 | 철사권갑 | 연타 피해 강화, 보법 기력 비용 증가 |
| 방패 | 관통방패 | 투사체 경감 강화, 막기 기력 비용 증가 |
| 외공 | 현철호심갑 | 최대 체력·방어 증가, 경공 내공 비용 증가 |
| 심법 | 청옥패 | 내공 회복 강화, 기력 회복 감소 |
| 의술 | 금침약함 | 회복 강화, 독공 효과 감소 |
| 마공 | 혈옥 | 체력 소모 마공 강화, 마화 피해도 강화 |
