# 구현 콘텐츠 카탈로그

> 이 문서는 현재 구현 대상 콘텐츠의 범위와 수량을 정의한다. 이름·분류·숙련 구조는 `06_STARTING_FACTION_MASTERY_DATA.md`를 따른다.

## 1. 공식 무공 분류

### 핵심무공 8종

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법 / 궁술 / 암기술`

- `부법`은 공식 성장 계통에서 제외한다.
- 도끼는 개별 무기·연출 태그로만 사용할 수 있다.
- `점혈술`은 핵심무공이 아니며 의술의 세부 기술·상태 태그다.

### 보조무공 8종

`심법 / 호신공 / 독공 / 마공 / 보법 / 방패술 / 외공 / 의술`

- 신법·경공은 보법의 세부 태그다.
- 의술은 회복·해독·지혈·점혈·해혈·경맥 제어를 포함한다.

## 2. 10문파 배치

| 문파 | 핵심무공서 | 보조무공서 | 전투 정체성 |
|---|---|---|---|
| 무당파 | 태극검·태극권 | 순양무극공·제운종 | 합 반격, 내공 순환, 원형 이동 |
| 화산파 | 매화검법 | 자하신공 | 빠른 검법, 연속 운용, 매화 검광 |
| 소림사 | 나한권·달마봉법 | 금강불괴신공 | 거리 0 압박, 봉 제압, 강한 호신 |
| 당문 | 당문암기비전 | 오독심경·금침의경·귀영보 | 암기, 독, 의술, 점혈, 재배치 |
| 개방 | 항룡십팔장·타구봉법 | 취팔선보 | 밀치기, 거리 장악, 변칙 이동 |
| 마교 | 혈염도법 | 역혈마공 | 체력 대가, 기혈 역류, 폭발력 |
| 양가장 | 양가창법 | 양문철벽패술 | 거리 유지, 창진, 방패 반격 |
| 백우궁 | 백우궁술 | 유운보법 | 장거리 정밀 사격, 이동 연계 |
| 사파 | 오호단문도·추혼암기술 | 철골외공 | 변칙 난전, 암습, 육체 강화 |
| 녹림 | 녹림창법·산림궁술 | 산림잠행보 | 길목 통제, 원거리 매복, 지형 이동 |

당문은 독공·의술·보법 세 계통을 제공하지만 한 회차에는 보조무공을 최대 2개만 보유한다.

## 3. 문파 전용 무공서 콘텐츠 단위

문파 전용 무공서는 총 28개다.

무공서 하나의 카드 구성:

1. 3성 기술 카드 1장
2. 7성 기술 카드 1장
3. 10성 절초 또는 진의 카드 1장

따라서 문파 전용 카드 목표는 총 **84장**이다.

| 성급 | 콘텐츠 결과 |
|---:|---|
| 3성 | 새로운 기술 1 해금 |
| 5성 | 기술 1 강화 — 새 카드 없음 |
| 7성 | 새로운 기술 2 해금 |
| 9성 | 기술 2 강화 — 새 카드 없음 |
| 10성 | 핵심무공은 절초, 보조무공은 진의 해금 |

## 4. 공용 무공 콘텐츠 단위

각 핵심무공 종류는 문파 전용 무공 외에 공용 무공 2종을 기본 목표로 한다.

- 방향 A: 낮은 비용과 안정적인 기본기.
- 방향 B: 높은 비용, 강한 합 또는 조건부 효과.
- 공용 무공은 문파 기믹을 직접 생성하지 않는다.
- 문파 보조무공이 만든 공용 상태 수정자의 혜택은 받을 수 있다.

공용 무공서와 기술명은 아직 미확정이다.

## 6. 카드·UI 자산 단위

기술 카드 앞면은 다음 자산을 조합한다.

- 문파 문양 이미지.
- 무공 종류 아이콘 이미지.
- 기술 삽화.
- 카드 프레임.
- 사거리 UI 텍스트.
- 기술 이름 UI 텍스트.
- 하단 묘리 문구 UI 텍스트.
- 10성 절초·진의 세로 표찰.

이미지 속 텍스트와 숫자는 기준 데이터가 아니다.

상세 팝업은 후속 단계에서 이름, 대상, 효과, 특수효과, 비용, 설명·키워드를 표시한다.

## 7. 공용 구현 기준

- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전, 명상이다.
- 두 행동을 비공개로 잠그고 양측이 동시에 공개한다.
- 같은 타이밍의 유효한 쌍방 공격은 합을 판정한다.
- 모든 무공은 사거리, 비용, 대상, 효과, 특수효과와 상태를 데이터로 가진다.
- 보조무공과 문파 무공은 공용 수정자와 구조화 이벤트를 재사용한다.

## 8. 명칭 경계

- 일반 마교 교도 무공에는 `천마` 명칭을 사용하지 않는다.
- 천마 명칭은 천마 개인 또는 특수 계승 콘텐츠에만 사용한다.
- 사파 전용 계통에는 검법·장법·보법을 배치하지 않는다.
- 점혈은 의술의 기술·상태 태그다.

## 9. 현재 구현 우선순위

1. 무당파 4개 무공서의 카드·데이터 파이프라인 검증.
2. 문파 문양 10종과 무공 종류 아이콘 16종 자산 연결.
3. 28개 무공서의 ID와 성급 해금 데이터 작성.
4. 효과·비용·밸런스 설계.
5. 10문파 전체 84장 카드 확장.
6. 공용 무공 16종 이상 설계.

## 10. 보류 관리

현재 활성 구현에서 제외:

- 문파별 기믹 수행도를 성과 등급에 반영하는 규칙.
- 복잡한 랜덤 손패·덱빌딩.
- 별도 장비 강화·경제 시스템.
- 10문파 밖 신규 세력과 새로운 핵심 자원.

보류 항목은 사용자가 재개하기 전 코드·데이터·자산·테스트에 반영하지 않는다.
