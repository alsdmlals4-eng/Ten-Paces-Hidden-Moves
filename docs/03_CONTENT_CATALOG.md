# 구현 콘텐츠 카탈로그

> 이번 세로 슬라이스에서는 12세력, 문파 전용 핵심무공·심법, 각 핵심무공 종류의 공용 무공 2종을 구현한다. 무공 효과는 공용 수정자를 조합해 표현한다.

## 시작 선택 6세력

| 세력 | 주 무기·거리 | 전용 핵심무공 | 전용 심법·기믹 |
|---|---|---|---|
| 화산파 | 검, 1 | 검법 | 심법 3성에서 연격 생성, 10성 진의에서 연격 강화 |
| 개방 | 곤, 2 | 곤법 | 밀치기·막기 기세와 거리 장악 |
| 당문 | 암기, 2~4 | 암기술 | 독·재장전·암기 연계 |
| 소림 | 권법, 0 | 권법 | 호신과 거리 0 압박 |
| 무당파 | 권법·검, 0~1 | 태극권·검법 | 합 반격과 내공 전환 |
| 마교 | 도, 1 | 도법 | 체력 대가와 마화 폭발력 |

## 핵심무공 콘텐츠 단위

각 핵심무공 종류는 다음 구성을 갖는다.

1. 관련 문파 전용 무공 1종
2. 저비용·안정형 공용 무공 1종
3. 고비용·고합 또는 조건부 공용 무공 1종

초기 구현 계통은 권법·장법, 검법, 도법, 창법, 곤법, 궁술, 암기술, 방패술이다. 세부 계통 병합·분리는 데이터 작성 단계에서 확정한다.

## 심법 콘텐츠 단위

- 한 회차 심법 보유는 최대 2개다.
- 문파 전용 심법은 3성에서 핵심 기믹을 생성하거나 핵심 패시브를 제공한다.
- 6성·9성은 명상 강화, 기믹 강화, 기술 제공을 문파 특성에 따라 조합한다.
- 10성은 절초 대신 진의를 제공한다.
- 한 회차에서 심법 10성은 최대 1개다.

## 공용 구현 기준

- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전, 명상이다.
- 2타이밍 행동 묶음을 비공개로 잠그고 동시에 공개한다.
- 같은 타이밍의 쌍방 공격은 합 수치 차이만큼 피해를 준다.
- 모든 무기는 사거리, 행동력·기력 비용, 합 수치, 막기·밀치기 효율, 재장전 여부를 데이터로 가진다.
- 심법과 문파 무공은 비용·합·피해 차이 경감·사거리·이동·상태·명상·장전 수정자를 재사용한다.
- 핵심무공은 여러 개 습득 가능하며 10성은 1개, 심법은 최대 2개 보유하며 10성은 1개다.
- 핵심무공은 3·6·9성 기술, 10성 절초를 제공한다.

## 보류 관리

활성 구현에서 제외하거나 추가 검증이 필요한 항목은 [`[보류]`](./%5B보류%5D/README.md) 폴더에 기록한다.

현재 보류:

- 문파별 기믹 수행도를 성과 등급에 반영하는 개별 평가 규칙
- 세계관 설명, 대사, 관계 시스템
- 복잡한 랜덤 손패·덱빌딩
- 별도 장비 강화·경제 시스템
- 12세력 밖 신규 세력과 새로운 핵심 자원

보류 문서는 활성 기획·구현의 기본 읽기 대상이 아니며, 재개 결정 전에는 기준 문서와 코드에 반영하지 않는다.
