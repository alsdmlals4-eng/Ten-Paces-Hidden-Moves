# 구현 콘텐츠 카탈로그

> 이번 세로 슬라이스에서는 10문파, 공식 핵심무공 8종, 공식 보조무공 8종, 문파 전용 무공서와 각 핵심무공 종류의 공용 무공 2종을 구현한다. 무공 효과는 공용 수정자를 조합해 표현한다.

## 1. 공식 무공 분류

### 핵심무공 8종

- 검법
- 도법
- 권법
- 장법
- 창법
- 곤봉법
- 궁술
- 암기술

`부법`은 공식 성장 계통에서 제외한다. 도끼는 필요할 경우 도법 또는 외공 계통의 개별 무기·연출 태그로 표현한다.

`점혈술`은 핵심무공에서 제외한다. 점혈·해혈·경맥 봉쇄는 보조무공 `의술`의 세부 태그로 처리한다.

### 보조무공 8종

- 심법
- 호신공
- 독공
- 마공
- 보법
- 방패술
- 외공
- 의술

신법과 경공은 보법의 세부 태그다. 의술은 회복·해독·지혈뿐 아니라 점혈·해혈·경맥 봉쇄를 포함한다.

## 2. 10문파 배치

| 문파 | 주 전투 계통 | 전용 핵심무공 | 전용 보조무공 | 핵심 역할 |
|---|---|---|---|---|
| 화산파 | 검 | 매화검법 | 자하신공 | 연격과 빠른 연속 검법 |
| 개방 | 장·곤봉 | 항룡십팔장·타구봉법 | 취팔선보 | 밀치기·거리 장악·변칙 이동 |
| 당문 | 암기 | 당문암기비전 | 오독심경·금침의경·귀영보 | 독·암기·의술·점혈·재배치 |
| 소림사 | 권·곤봉 | 나한권·달마봉법 | 금강불괴신공 | 거리 0 압박과 강한 호신 |
| 무당파 | 검·권 | 태극검법·태극권 | 순양무극공·제운종 | 합 반격·내공 순환·회피 이동 |
| 마교 | 도 | 혈염도법 | 역혈마공 | 체력 대가와 마기 폭발력 |
| 양가장 | 창 | 양가창법 | 양문철벽패술 | 거리 유지·창 회수·방패 반격 |
| 백우궁 | 궁 | 백우궁술 | 유운보법 | 장거리 사격과 장전 공백 관리 |
| 사파 | 도·암기 | 흑풍도법·추혼암기술 | 철골외공 | 변칙 난전·암습·육체 강화 |
| 녹림 | 창·궁 | 녹림창법·산림궁술 | 산림잠행보 | 산채 길목 통제·원거리 매복 |

문파가 제공하는 보조무공 계통 수와 한 회차 동시 보유 한도는 구분한다. 당문은 독공·의술·보법을 제공하지만 실제 보유는 최대 2개 제한을 따른다.

## 3. 핵심무공 콘텐츠 단위

각 핵심무공 종류는 다음 구성을 갖는다.

1. 관련 문파 전용 무공 1종 이상
2. 저비용·안정형 공용 무공 1종
3. 고비용·고합 또는 조건부 공용 무공 1종

핵심무공은 3·6·9성에서 기술을 얻고 10성에서 절초를 해금한다.

## 4. 보조무공 콘텐츠 단위

- 보조무공은 패시브, 문파 기믹 생성·강화, 명상 강화, 자원 전환, 이동, 방어, 상태 생성·해제를 담당한다.
- 한 회차 보조무공 보유는 최대 2개다.
- 3성에서 핵심 기믹 또는 패시브를 제공한다.
- 6성·9성은 명상 강화, 기믹 강화 또는 기술을 제공한다.
- 10성은 절초 대신 진의를 제공한다.
- 한 회차에서 보조무공 10성은 최대 1개다.

## 6. 공용 구현 기준

- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전, 명상이다.
- 2타이밍 행동 묶음을 비공개로 잠그고 동시에 공개한다.
- 같은 타이밍의 쌍방 공격은 합 수치 차이만큼 피해를 준다.
- 모든 무기는 사거리, 행동력·기력 비용, 합 수치, 막기·밀치기 효율, 재장전 여부를 데이터로 가진다.
- 카드 아트의 사거리·비용 숫자는 비워 두고 데이터와 UI 레이어에서 편집한다.
- 보조무공과 문파 무공은 비용·합·피해 차이 경감·사거리·이동·상태·명상·장전 수정자를 재사용한다.

## 7. 명칭 경계

- 일반 마교 교도 무공에는 `천마` 명칭을 사용하지 않는다.
- `천마`가 붙은 무공은 천마 개인 또는 특수 계승 콘텐츠로 분리한다.
- 사파 전용 계통에서는 검법·장법·보법을 사용하지 않는다.
- 점혈은 의술의 기술·상태 태그이며 독립 핵심무공이 아니다.

## 8. 보류 관리

활성 구현에서 제외하거나 추가 검증이 필요한 항목은 [`[보류]`](./%5B보류%5D/README.md) 폴더에 기록한다.

현재 보류:

- 문파별 기믹 수행도를 성과 등급에 반영하는 개별 평가 규칙
- 세계관 설명, 대사, 관계 시스템
- 복잡한 랜덤 손패·덱빌딩
- 별도 장비 강화·경제 시스템
- 10문파 밖 신규 세력과 새로운 핵심 자원

보류 문서는 활성 기획·구현의 기본 읽기 대상이 아니며, 재개 결정 전에는 기준 문서와 코드에 반영하지 않는다.
