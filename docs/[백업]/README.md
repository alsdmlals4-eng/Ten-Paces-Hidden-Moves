# 백업 문서 보관소

이 폴더는 기준 문서를 갱신하기 전의 이전 버전을 보존하기 위한 읽기 전용 기록 공간이다.

## 작업 규칙

- 이 폴더의 문서는 현재 기획·구현·테스트의 기준으로 사용하지 않는다.
- ChatGPT, Codex 및 기타 작업 에이전트는 이 폴더의 문서를 수정 대상이나 구현 요구사항으로 선택하지 않는다.
- 현재 기준 문서는 `docs/DOCUMENTATION_MAP.md`에 연결된 원본 경로만 사용한다.
- 기준 문서를 크게 수정하기 전에는 기존 파일 전체를 `docs/[백업]/YYYY-MM-DD/` 아래에 복사한다.
- `*_v2`, `*_final`, `*_latest` 같은 활성 중복 문서를 만들지 않고 기존 기준 파일을 직접 갱신한다.
- 과거 결정의 경위를 확인할 때만 백업 문서를 참고한다.
