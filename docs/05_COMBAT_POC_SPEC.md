# 12세력 대회 세로 슬라이스 명세

> 목적: 시작 문파를 골라 내부전부터 예선·본선·결승까지 9전을 치르는 동안, 공용 행동과 문파별 무기·유파 규칙·대성이 전략적 선택을 만드는지 검증한다.

## 1. 범위

### 포함

- 10칸 전장, 1대1 AI 결투, 비공개 동시 행동열
- 시작 6세력: 화산·개방·당문·소림·무당·마교
- 추가 6세력: 양가장·백우궁·관군·아미·오독·녹림채(도적·산적 사파)
- 모든 세력의 대표 무기, 유파 규칙, 대성 1~5성
- 9전 대진, 보상 선택, 수련도, 정탐, 회차당 1회 재도전
- 암기벨트, 화살통, 방패의 공용 보조·무기 규칙

### 제외

- 세계관·대화·인물 관계
- 랜덤 손패·덱빌딩
- 별도 장비 강화·경제 시스템
- 11세력 밖의 신규 문파

## 2. 공통 전투 상태

| 항목 | 값 |
|---|---|
| 전장 | 1~10번 칸의 가로 전장. 같은 칸 중첩 가능, 교차·자리 교환 가능 |
| 초기 배치 | A=4번 칸, B=6번 칸 |
| 초기 거리 | 2 |
| 행동열 | 6시간 슬롯을 양측 비공개로 전부 배치 후 동시 공개 |
| 행동력 | 6 |
| 기력 | 12 |
| 내공 | 6 |
| 회복 | 라운드 종료 시 기력 +2, 내공 +1. 상태 이상은 이후 처리 |
| 결투 승리 | 상대 체력 0 또는 6슬롯 뒤 체력 우위 |

공용 행동과 슬롯 해상 순서는 [전투 시스템 아키텍처](09_COMBAT_SYSTEM_ARCHITECTURE.md)를 따른다.

## 3. 대진과 진행

| 경기 | 단계 | 설계 역할 |
|---:|---|---|
| 1~2전 | 내부전 | 선택 문파의 무기·대표 유파 규칙 습득 |
| 3전 | 추천패 쟁탈전 | 첫 외문 무공 또는 보조장비 획득 |
| 4~5전 | 지역 예선 | 상성 무기와 보상 선택 압박 |
| 6~7전 | 본선 | 3~4성 수련, 정탐, 복합 행동열 |
| 8전 | 준결승 | 대성 또는 혼합 빌드의 강한 상대 |
| 9전 | 결승 | 거리·자원·행동열을 모두 묻는 종합전 |

- 상대 풀은 11세력 전체다. 한 회차의 9전에서 전 세력이 나올 필요는 없으며, 같은 상대 반복은 풀을 소진하기 전에는 금지한다.
- 내부전은 선택 문파의 훈련 상대를 사용해 기본 규칙을 가르친다. 이후 대진에서는 선택 문파를 제외한 세력이 우선 등장한다.
- 시작 6세력은 플레이어 선택과 AI 적 모두 가능하다. 추가 6세력은 AI 적이며, 대표 무공은 승리 보상으로 습득할 수 있다.
- 녹림채는 도적·산적을 포괄한 사파 대진으로 등장하며, 도 공격의 기력 탈취 규칙을 사용한다.

## 4. 수련·보상·대성

각 경기 후 하나를 선택한다.

1. 무공서: 다른 세력의 1성 규칙 또는 이미 보유한 규칙의 수련도
2. 수련도: 지정 문파 무공의 다음 성급 진행
3. 보조장비: 암기벨트·화살통·방패 중 호환 장비
4. 회복: 다음 전투를 위한 체력·기력·내공 회복
5. 정탐: 다음 상대의 무기·대표 규칙·AI 성향 공개

대성은 1~5성이다. 5성은 해당 문파의 완성 규칙 또는 외문 규칙 슬롯을 연다. 한 회차에 대성은 한 계통만 권장한다.

## 5. 재도전

- 회차당 1회만 가능하며, 전체 최초 사용은 무료다.
- 이후에는 영구 재화 명망을 사용한다.
- 패배 뒤 재기 전리품 1회와 재기 수련도 1회를 받고, 장비·무공을 다시 조정한 뒤 같은 상대와 재대결한다.
- AI는 공개 정보와 동일한 보유 규칙만 쓰고, 행동열은 새로 사전 잠금한다.

## 6. 완료 기준

- 시작 6세력 모두로 새 회차를 시작할 수 있다.
- 11세력 모두가 최소 한 번 AI 또는 플레이어 측에서 전투를 끝까지 수행한다.
- 9전 회차가 내부전·예선·본선·준결승·결승 단계를 빠짐없이 통과한다.
- 패배·재도전·보상·대성 선택이 다음 전투의 공용 행동·무기·유파 결과에 반영된다.
