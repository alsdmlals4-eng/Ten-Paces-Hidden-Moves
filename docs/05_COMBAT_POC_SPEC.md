# 십보강호 전투 POC·세로 슬라이스 명세

> 책임: 현재 T0 전투 POC의 범위·구현 상태·완료 기준과 이후 T1·5전 데모 진입 조건  
> 전투 세부 판정: `docs/02_COMBAT_RULES.md`  
> 현재 실행 씬: `res://scenes/combat/combat_board_preview.tscn`

## 1. 목적

현재 단계의 목적은 12세력·10전 대회 전체를 구현하는 것이 아니다.

T0 POC는 다음 핵심 가설을 최소 범위에서 검증한다.

> **10칸 전장에서 플레이어 4번·상대 7번의 거리 3으로 시작해, 기초 행동 8종을 `3수 → 3수 → 4수`로 배치하고 거리·방향·자원·대응·중단을 읽는 과정이 한 판만으로도 이해 가능하고 반복할 만큼 재미있는가?**

성장·세력·무공·절초·보상은 이 코어가 통과한 뒤 확장한다.

## 2. 단계 구분

### T0 — 단일 전투 POC

현재 Draft PR #7의 범위다.

- 단일 1대1 전투.
- 플레이어 `[강호낭인]`과 고정 검증 상대.
- 전장 10칸, 플레이어 4번·상대 7번 시작.
- `3수 → 3수 → 4수`, 총 10수 라운드.
- 기초 행동 8종.
- 이동 목적지·공격 방향 지정.
- 기력·내력 계획 예상값.
- 대응·속공·이동·일반 공격 판정.
- 전투 로그.
- 이후 피격 중단·AI·종료·재시작.

### T1 — 최소 세로 슬라이스

T0 플레이테스트 통과 뒤 진행한다.

- 플레이 스타일 2개.
- 상대 3명 내외.
- 전투 3회 내외.
- 전투 전 상대 정보.
- 전투 후 보상 선택.
- 무공 또는 능력 강화 1회 이상.
- 최종 강적 1회.

T1은 5전 데모보다 작은 제작·학습 단위다.

### T2 — 5전 데모

- 1~2전 내부전.
- 3~4전 예선.
- 5전 예선 결승.
- 시작 무공·심법과 수련.
- 전투 전 정보·제약.
- 전투 후 성과·보상·성장.
- 조건부 절초 체험.

### 전체판

- 대회 10전.
- 8전 8강, 9전 준결승, 10전 결승.
- 시작·추가 세력을 포함한 12세력 구상.
- 무공·심법 1~10성.
- 절초·진의.
- 정탐·재도전·저장·영구 해금.

T1·T2·전체판 항목은 현재 T0 완료 조건이 아니다.

## 4. 공통 전투 상태

### 4.1 전장

| 항목 | 값 |
|---|---|
| 전장 | 1~10번 칸 |
| 플레이어 시작 | 4번 |
| 상대 시작 | 7번 |
| 시작 거리 | 3칸 |
| 칸 점유 | 한 칸에 한 전투원 |
| 전장 밖 이동 | 불가 |
| 상대 점유 칸 이동 | 불가 |
| 동일 목적지 동시 이동 | 양측 모두 제자리 |

중첩·통과·자리 교환은 현재 POC에 포함하지 않는다.

### 4.2 라운드

| 묶음 | 수 | 시작 슬롯 |
|---|---|---:|
| 1묶음 | 1~3수 | 1수 |
| 2묶음 | 4~6수 | 4수 |
| 3묶음 | 7~10수 | 7수 |

- 현재 묶음만 배치 가능하다.
- 현재 묶음의 모든 슬롯·대상·자원이 유효해야 진행한다.
- 2슬롯 행동은 마지막 점유 수에서 실행한다.
- 10수 뒤 다음 라운드 1수로 이동한다.

### 4.3 기본 자원

| 자원 | 플레이어 | 상대 | 시작 규칙 |
|---|---:|---:|---|
| 체력 | 20/20 | 20/20 | 최대치−시작 패널티 |
| 기력 | 5/5 | 5/5 | 최대치−시작 패널티 |
| 내력 | 4/4 | 4/4 | 최대치−시작 패널티 |
| 절초 기세 | fixture 지정 | fixture 지정 | 자동 완충하지 않음, 최대 5 |

HUD의 4/5·3/5 기세는 게이지 표시 검증용 값이며 정식 시작 밸런스가 아니다.

## 5. 기초 행동 계약

| ID | 이름 | 슬롯 | 기력 | 내력 | 대상·사거리 | 효과 |
|---|---|---:|---:|---:|---|---|
| `basic_move` | 이동 | 1 | 0 | 0 | 좌·우 1칸 | 선택 칸 이동 |
| `basic_footwork` | 보법 | 1 | 0 | 1 | 좌·우 1~2칸 | 1칸·2칸 선택 이동 |
| `basic_guard` | 막기 | 1 | 0 | 0 | 자신 | 방어도 4 차감 후 같은 수면 남은 피해 반감 |
| `basic_evade` | 회피 | 1 | 1 | 0 | 자신 | 같은 수 완전 회피 |
| `basic_quick_attack` | 속공 | 1 | 1 | 0 | 지정 방향 거리 1 | 이동 전 피해 6 |
| `basic_heavy_attack` | 강공 | 2 | 1 | 1 | 지정 방향 거리 1~2 | 마지막 점유 수에 피해 8 |
| `basic_meditate` | 명상 | 1 | 0 | 0 | 자신 | 기력 2·내력 1 회복 |
| `basic_stance` | 태세 | 1 | 0 | 1 | 자신·대응 결합 | 단독 다음 공격 +2 또는 대응 강화 |

태세+막기와 태세+회피는 하나의 슬롯을 사용하는 결합 행동이며 비용을 합산한다.

## 6. T0 전투 흐름

```text
전투 시작: 플레이어 4번·상대 7번
→ 체력·기력·내력 초기화
→ 1묶음 1~3수 배치
→ 이동 칸·공격 방향 지정
→ 예상 자원 확인
→ 진행
→ 대응 → 속공 → 이동 → 일반 공격 판정

→ 2묶음 4~6수
→ 3묶음 7~10수
→ 다음 라운드 1수
```

현재 체력 0에 따른 승패·종료 화면은 구현하지 않았다.

## 8. 현재 상대 fixture

정식 AI 대신 고정 행동 계획을 사용한다.

| 묶음 | 상대 행동 |
|---|---|
| 1묶음 | 이동 → 막기 → 속공 |
| 2묶음 | 이동 → 속공 → 막기 |
| 3묶음 | 이동 → 속공 → 강공 |

현재 1묶음 첫 이동은 상대를 7번에서 6번으로 이동시키는 재현 fixture다.

목적:

- 거리와 방향 판정 재현.
- 방어·회피 검증.
- 자원·로그·라운드 진행 검증.
- 동일 입력에 같은 결과 제공.

이 fixture는 정식 AI의 플레이 품질을 증명하지 않는다.
