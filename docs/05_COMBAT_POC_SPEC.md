# 12세력 대회 세로 슬라이스 명세

> 목적: 10칸 전장·10타이밍·10전 대회에서 공용 행동·합 판정·문파별 핵심무공과 심법·10성 수련·제약 선택이 전략적 차이를 만드는지 검증한다. 데모는 5전째 지역 예선 결승까지 제공한다.

## 1. 범위

### 데모 포함

- Godot 4.7.1 + GDScript, Windows 데스크톱 데모
- 10칸 전장, 1대1 AI 결투, 2수 비공개 동시 선택
- 한 라운드 10타이밍, 2타이밍 묶음 5회
- 1~5전: 내부전 2회, 추천패 쟁탈전, 지역 예선, 지역 예선 결승
- 시작 문파의 핵심무공·심법 3성
- 승리 기본 수련 포인트 5와 C/B/A/S 성과 보상
- A·S등급을 다수 획득하고 핵심무공에 집중 투자할 경우 4전 종료 후 10성 절초 개방
- 5전째 지역 예선 결승에서 절초 사용 경험
- 삼수양보 등 무협식 제약과 승리 추가 보상

### 전체판 포함

- 전장 10칸·라운드 행동 타이밍 10개·대회 10전의 `10-10-10` 구조
- 동일 타이밍 쌍방 공격의 합 판정과 차이 피해
- 시작 6세력: 화산·개방·당문·소림·무당·마교
- 추가 6세력: 양가장·백우궁·관군·아미·오독·녹림채
- 각 세력의 핵심무공·심법·10성 절초·진의 방향
- 각 핵심무공 종류의 공용 무공 2종
- 심법 보유 최대 2개
- 10전 대진, 보상 선택, 정탐, 회차당 1회 재도전
- 암기벨트, 화살통, 방패의 공용 보조·무기 규칙
- 회차 저장·불러오기와 저장 포맷 버전

### 제외

- 세계관·대화·인물 관계
- 랜덤 손패·덱빌딩
- 별도 장비 강화·경제 시스템
- 12세력 밖 신규 문파
- Steamworks 상점·업적·클라우드 연동
- 문파별 기믹 수행도를 성과 등급에 반영하는 개별 평가 규칙

## 2. 공통 전투 상태

| 항목 | 값 |
|---|---|
| 전장 | 1~10번 칸. 같은 칸 중첩, 교차·자리 교환 가능 |
| 초기 배치 | A=4번 칸, B=6번 칸 |
| 초기 거리 | 2 |
| 라운드 | 행동 타이밍 10개 |
| 선택 단위 | 행동 두 개 비공개 잠금 → 동시 공개 → 순차 해상 |
| 행동력 | 6, 10타이밍 종료 시 전부 회복 |
| 기력 | 12, 라운드 종료 시 소량 회복 |
| 내공 | 6, 라운드 종료 시 기력보다 적게 회복 |
| 합 | 양측 공격이 모두 성립하면 최종 합 수치 비교, 낮은 쪽이 차이 피해 |
| 결투 승리 | 상대 체력 0 또는 정해진 라운드 종료 시 체력 우위 |

AI는 각 2수 묶음 시작 시 공개 정보로 두 행동을 잠근다. 플레이어 행동을 확인한 뒤 수정할 수 없다.

## 3. 대진과 진행

| 경기 | 단계 | 설계 역할 | 데모 포함 |
|---:|---|---|---|
| 1~2전 | 내부전 | 문파 핵심무공·심법 기믹과 합 규칙 습득 | 포함 |
| 3전 | 추천패 쟁탈전 | 첫 공용·외문 무공 또는 보조장비 획득 | 포함 |
| 4전 | 지역 예선 | 무기 상성, 성과 등급, 제약 이해 | 포함 |
| 5전 | 지역 예선 결승 | 고성과 플레이어의 조기 절초 체험과 데모 최종전 | 포함 |
| 6~7전 | 본선 | 6·9성 기술, 제약, 복합 2수 연계 | 전체판 |
| 8전 | 8강 | 완성 직전 빌드의 상성 대응 | 전체판 |
| 9전 | 준결승 | 절초·진의 또는 복합 빌드 완성 검증 | 전체판 |
| 10전 | 결승 | 거리·합·자원·예측·성장 종합전 | 전체판 |

- 상대 풀은 12세력 전체다.
- 같은 상대 반복은 풀을 소진하기 전 금지한다.
- 시작 6세력은 플레이어 선택과 AI 적 모두 가능하다.
- 추가 6세력은 AI 적이며 대표 무공은 승리 보상으로 습득할 수 있다.
- 결승 전까지 수련 단계는 최대 9회다.

## 4. 무공·심법·수련

- 시작 문파의 핵심무공과 심법은 각각 3성이다.
- 핵심무공은 여러 개 익힐 수 있다.
- 심법은 보유 자체가 최대 2개다.
- 한 회차에서 핵심무공 10성은 최대 1개, 심법 10성은 최대 1개다.
- 핵심무공은 3·6·9성에 기술, 10성에 절초를 얻는다.
- 심법은 3·6·9성에 패시브·기믹·명상 효과를 조합하고 10성에 진의를 얻는다.
- 성급 n에 도달하는 비용은 n포인트다.
- 3성 핵심무공을 10성으로 올리는 데 49포인트가 필요하다.
- 3성 심법을 8성으로 올리는 데 30포인트가 필요하다.

## 5. 성과 등급과 수련 포인트

승리 시 기본 5포인트를 얻고 성과 등급에 따라 보너스를 받는다.

| 등급 | 확정 보너스 | 확률 보너스 | 총 획득 범위 | 기대값 |
|---|---:|---|---:|---:|
| C | +1 | 30% 확률로 +1 | 6~7 | 6.3 |
| B | +3 | 50% 확률로 +2 | 8~10 | 9.0 |
| A | +8 | 50% 확률로 +1 | 13~14 | 13.5 |
| S | +9 | +2 확정 | 16 | 16.0 |

성과 등급은 남은 체력, 자원 운용, 무효 행동, 합·거리 판단 등 공용 지표로 계산한다. 문파별 기믹 수행도는 현재 보류한다.

### 데모 절초 개방 곡선

첫 4전 종료 후 5전째 지역 예선 결승에 들어가므로, 절초를 사용하려면 첫 4전에서 핵심무공 3성→10성에 필요한 49포인트를 모아야 한다.

- A등급 4회: 최소 52포인트로 절초 확정 개방
- A등급 3회 + S등급 1회: 최소 55포인트
- A등급 2회 + S등급 2회: 최소 58포인트
- B등급 4회: 32~40포인트로 절초 미개방

따라서 예선 결승 전 절초는 일반 진행의 필수 요소가 아니라, A·S를 다수 획득한 플레이어가 체험하는 조기 성과 보상이다.

### 전체판 일반 성장 곡선

결승 전 수련 단계 9회에서 B등급 기대값은 `9×9=81포인트`다. 핵심무공 10성 49포인트와 심법 8성 30포인트의 합계 79포인트에 근접하므로 일반적인 목표 성장과 맞는다.

## 6. 전투 제약

전투 시작 전에 제약 후보를 확인하고 선택해 난이도와 승리 보상을 높인다.

| 제약 예시 | 효과 방향 |
|---|---|
| 삼수양보 | 첫 3타이밍 동안 공격·밀치기 금지 |
| 단전봉맥 | 시작 내공과 라운드 내공 회복 감소 |
| 기혈역행 | 일정량 이상 내공을 쓰면 체력 손실 |
| 일장지양 | 상대가 선호하는 거리에서 시작 |
| 병기자박 | 전투 중 예비 무기 교체 금지 |
| 호신철회 | 막기·회피 효율 일부 감소 |

- 최대 선택 수와 위험도는 밸런스 데이터로 둔다.
- 특정 문파의 주력 계통을 완전히 금지하는 제약은 만들지 않는다.
- 제약 승리 보상은 추가 수련 포인트, 보상 선택지, 새로고침, 고급 보상 확률로 구성한다.

## 7. 재도전과 저장

- 재도전은 회차당 1회다.
- 게임 전체 최초 재도전은 무료, 이후에는 명망을 사용한다.
- 재도전 시 전투 시작 전 상태로 복구하고 재기 보상을 받은 뒤 같은 상대와 재대결한다.
- AI는 각 2수 묶음을 다시 사전 잠금한다.
- 저장 데이터는 포맷 버전, 선택 문파, 현재 경기, 상대 풀, 보유 핵심무공·심법·성급·수련 포인트·장비·명망·재도전·제약 상태를 보관한다.
- 데모 저장은 5전 종료와 전체판 전환 지점을 구분한다.

## 8. 완료 기준

- 시작 6세력 모두로 새 회차를 시작할 수 있다.
- 데모는 5전째 지역 예선 결승까지 중단 없이 진행된다.
- 전체판은 10전째 결승까지 진행된다.
- 2수 비공개 잠금과 AI 공정성이 유지된다.
- 합 수치와 차이 피해가 로그·UI·테스트에서 일치한다.
- 핵심무공·심법 3·6·9·10성 해금이 실제 전투에 반영된다.
- 심법 보유 최대 2개와 계열별 10성 제한이 지켜진다.
- 첫 4전을 모두 A등급 이상으로 승리하고 핵심무공에 집중 투자하면 5전에서 절초를 사용할 수 있다.
- B등급 중심 데모 진행에서는 절초가 예선 결승 전 보장되지 않는다.
- 전체판 일반 성과로 핵심무공 10성·심법 8성 전후의 성장 곡선이 나온다.
- 제약 승리 보상이 기본 성장에 필수가 아니면서 추가 육성 선택을 제공한다.
- 저장·불러오기와 Windows 빌드가 완료된다.
