## 변경 원칙

- 새 시스템은 먼저 위 문서 중 하나에 규칙·제외 범위·검증 조건을 기록한다.
- 수치 조정은 핵심 규칙을 바꾸지 않는 한 해당 표의 수치만 수정한다.
- 구현 작업은 Goal 대신 Plan과 테스트 체크리스트를 기준으로 진행한다.
- 세계관 설명, 대사, 인물 관계는 현재 범위 밖이다.
