# 십보강호 게임 기획서

## 1. 한 줄 약속

제한된 대회 일정 안에서 문파 무공을 수련하고, 10칸 전장의 비공개 동시 선택 결투에서 상대의 거리·자원·심리를 읽어 우승하는 1대1 무협 로그라이트.

## 2. 플레이어 경험

- 권법가가 상대의 후퇴를 읽고 같은 칸까지 파고드는 쾌감
- 창술가가 거리 2를 지키기 위해 밀치기와 회수를 계산하는 긴장감
- 궁수가 장전 공백을 경공과 화살통으로 감추는 전술성
- 패배 후 재기 보상으로 ‘한 조각 부족한 빌드’를 완성해 돌파하는 만족감
- 같은 상대라도 AI의 다음 행동열은 알 수 없어 정답 암기에 머물지 않는 반복성

## 3. 구현 세로 슬라이스

### 포함

- AI 싱글플레이 1대1 결투와 9전 비공식 비무회
- 시작 6세력: 화산, 개방, 당문, 소림, 무당, 마교
- 추가 5세력: 양가장, 백우궁, 관군, 아미, 오독
- 문파별 대표 무기·유파 규칙·대성 1~5성
- 무공 습득, 보상 선택, 정탐, 재도전, 제한적 영구 해금
- 내부전 → 추천패 쟁탈전 → 지역 예선 → 본선 → 준결승 → 결승의 9전 구조

### 제외

- 대사, 인물 호감도, 관계도, 세계관 탐색
- 실시간 조작과 반사신경 중심 전투
- 무작위 손패 기반 덱빌딩
- 기절, 이미 배치한 행동 취소, 강제 행동 변경
- 멀티플레이와 PvP 밸런스
- 별도 장비 강화·경제 시스템

## 4. 회차 구조

평균 60분, 결투 9전으로 구성한다. 모든 결투의 기본 배치는 A=4번 칸, B=6번 칸, 거리 2다.

| 구간 | 경기 | 목표 |
|---|---:|---|
| 내부전 | 1~2전 | 선택 문파의 기본 전법과 무기 운용 습득 |
| 추천패 쟁탈전 | 3전 | 다른 계통의 첫 무공서 획득 |
| 지역 예선 | 4~5전 | 무기 상성과 혼합 빌드 이해 |
| 본선 | 6~7전 | 3~4성 무공과 정탐 활용 |
| 준결승 | 8전 | 주력 무공 대성 또는 혼합 빌드 완성 |
| 결승 | 9전 | 거리·행동열·자원 운영 종합 |

한 회차가 모든 세력을 소진할 필요는 없다. 대신 상대 풀은 11세력 전체로 구성하며, 시작 문파 이외의 세력은 적 또는 무공 습득 대상으로 등장한다.

경기 전에는 다음 상대의 무기, 기본 사거리, 공개 대표 무공, 선호 거리, 자원 성향을 확인한다. 경기 후에는 무공서, 수련도, 보조장비, 회복, 상대 정보 중 하나를 선택한다.

## 5. 패배와 영구 성장

### 재도전

- 게임 전체 최초 재도전은 무료다.
- 이후 재도전은 영구 재화인 명망을 사용한다.
- 회차당 재도전은 최대 1회다.
- 재도전하면 전투 시작 전 상태로 복구하고, 재기 전리품 1회와 재기 수련도 1회를 받는다.
- 장비와 운용 무공서를 다시 조정한 뒤 같은 상대와 재대결한다.
- AI의 무공서·장비·성향은 유지하지만 6슬롯 행동열은 새로 생성된다.
- 재도전에 다시 패배하면 회차가 끝난다.

### 명망

명망은 영구 공격력 누적보다 다음 회차의 선택지 확장에 쓴다.

- 시작 문파와 시작 무공 후보 해금
- 새 무공서, 보조장비, 수련 장소 해금
- 보상 선택지 새로고침과 상대 정보 해금
- 최대 체력·기력·내공의 제한적 기초 단련(전체 합계 약 5% 이내)

## 6. 무공 대성

무공의 수련도와 성급은 회차 종료 시 초기화된다. 11세력 모두 같은 1~5성 틀을 사용하고, 각 세력의 실제 효과는 공용 행동 변형으로만 작성한다.

| 성급 | 역할 |
|---:|---|
| 1성 | 문파의 기본 상시 규칙 |
| 2성 | 대표 행동의 비용·사거리·효율 개선 |
| 3성 | 대표 기술 또는 보조장비 연계 |
| 4성 | 대응·추격·역전 규칙 |
| 5성·대성 | 해당 문파 빌드 완성 또는 외문 규칙 슬롯 해금 |

한 회차의 일반적인 완성형은 ‘대성 1개 + 보조 무공 2~3성’ 또는 ‘여러 무공 3~4성 혼합’이다. 모든 무공을 대성할 수는 없다.
