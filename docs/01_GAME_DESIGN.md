# 십보강호 게임 기획서

## 1. 한 줄 약속

제한된 대회 일정 안에서 문파 무공을 수련하고, 10칸 전장의 비공개 동시 선택 결투에서 상대의 거리·자원·심리를 읽어 우승하는 1대1 무협 로그라이트.

## 2. 플레이어 경험

- 권법가가 상대의 후퇴를 읽고 같은 칸까지 파고드는 쾌감
- 창술가가 거리 2를 지키기 위해 밀치기와 회수를 계산하는 긴장감
- 궁수가 장전 공백을 경공과 화살통으로 감추는 전술성
- 패배 후 재기 보상으로 ‘한 조각 부족한 빌드’를 완성해 돌파하는 만족감
- 같은 상대라도 AI의 다음 행동열은 알 수 없어 정답 암기에 머물지 않는 반복성

## 3. 범위와 제외 범위

### 포함

- AI 싱글플레이 1대1 결투
- 비공식 비무회를 최소 배경으로 한 문파 선택과 대회 상승
- 무공 습득, 성급 대성, 장비, 정탐, 영구 해금
- 60분 내외의 로그라이트 회차

### 제외

- 대사, 인물 호감도, 관계도, 세계관 탐색
- 실시간 조작과 반사신경 중심 전투
- 무작위 손패 기반 덱빌딩
- 기절, 이미 배치한 행동 취소, 강제 행동 변경
- 멀티플레이와 PvP 밸런스

## 4. 회차 구조

평균 60분, 결투 9전으로 구성한다.

| 구간 | 경기 | 목표 |
|---|---:|---|
| 내부전 | 1~2전 | 시작 문파의 기본 전법 습득 |
| 추천패 쟁탈전 | 3전 | 다른 계통의 첫 무공서 획득 |
| 지역 예선 | 4~5전 | 무기 상성과 혼합 빌드 이해 |
| 본선 | 6~7전 | 3~4성 무공과 정탐 활용 |
| 준결승 | 8전 | 주력 무공 대성 또는 혼합 빌드 완성 |
| 결승 | 9전 | 거리·행동열·자원 운영 종합 |

경기 전에는 다음 상대의 무기, 기본 사거리, 공개 대표 무공, 선호 거리, 자원 성향을 확인한다. 경기 후에는 무공서, 수련도, 장비, 회복, 상대 정보 중 하나를 선택한다.

## 5. 패배와 영구 성장

### 재도전

- 게임 전체 최초 재도전은 무료다.
- 이후 재도전은 영구 재화인 명망을 사용한다.
- 회차당 재도전은 최대 1회다.
- 재도전하면 전투 시작 전 상태로 복구하고, 재기 전리품 1회와 재기 수련도 1회를 받는다.
- 장비와 운용 무공서를 다시 조정한 뒤 같은 상대와 재대결한다.
- AI의 무공서·장비·성향은 유지하지만 6슬롯 행동열은 새로 생성된다.
- 재도전에 다시 패배하면 회차가 끝난다.

### 명망

명망은 영구 공격력 누적보다 다음 회차의 선택지 확장에 쓴다.

- 시작 문파와 시작 무공 후보 해금
- 새 무공서, 장비, 수련 장소 해금
- 보상 선택지 새로고침과 상대 정보 해금
- 최대 체력·기력·내공의 제한적 기초 단련(전체 합계 약 5% 이내)

## 6. 무공 대성

무공의 수련도와 성급은 회차 종료 시 초기화된다.

| 성급 | 역할 |
|---:|---|
| 1성 | 기본 상시 효과 |
| 2성 | 대표 사용 기술 |
| 3성 | 대표 기술의 운용 강화 |
| 4성 | 대응·추격·역전 기술 |
| 5성·대성 | 해당 빌드의 완성 규칙 |

한 회차의 일반적인 완성형은 ‘대성 1개 + 보조 무공 2~3성’ 또는 ‘여러 무공 3~4성 혼합’이다. 모든 무공을 대성할 수는 없다.
