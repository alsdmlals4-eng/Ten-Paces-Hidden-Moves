# 12세력 대회 전투 UI·아트 명세

> 목적: 플레이어가 시작 문파 선택부터 9전 대진까지 이해하고, 전투 중에는 무공 카드만 보고도 문파·무공 종류·사거리·비용을 빠르게 판단하며, 필요할 때 상세 효과를 정확히 읽게 한다.

이 파일은 전투 UI와 무공 카드 아트의 현재 기준 문서다. 이전 버전은 `docs/[백업]/`에만 보관하며 현재 구현 기준으로 사용하지 않는다.

## 1. 회차·대진 화면

| 영역 | 표시 내용 |
|---|---|
| 시작 문파 | 화산·개방·당문·소림·무당·마교의 무기, 대표 규칙, 추천 거리 |
| 대진 진행 | 내부전 / 추천패 / 예선 / 본선 / 준결승 / 결승, 현재 경기와 9전 진행도 |
| 다음 상대 | 세력, 무기·사거리, 공개 대표 규칙, 선호 거리, 정탐 정보 |
| 보상 선택 | 무공서, 수련도, 보조장비, 회복, 정탐의 효과와 적용 대상 |
| 대성 패널 | 보유 문파 무공의 1~5성 진행, 다음 해금 효과, 외문 규칙 여부 |

추가 6세력은 대진·보상·도감에서 같은 형식으로 표시한다. 녹림채는 ‘사파 / 도적·산적’ 분류와 약탈도 기력 탈취를 함께 표시한다. 플레이어가 모르는 정보는 ‘정탐 전’으로 분명히 가린다.

## 2. 전투 화면

| 영역 | 표시 내용 |
|---|---|
| 적 상태 | 체력·기력·내공, 무기 장전, 방패, 독·마화, 대표 유파 규칙 |
| 10칸 전장 | 1~10번 칸, A=4 / B=6의 초기 배치, 같은 칸 중첩과 현재 거리 |
| 플레이어 상태 | 체력·기력·내공, 무기·보조장비, 상태, 유파 규칙 |
| 무공 카드 목록 | 사용 가능한 무공 카드, 문파·종류·사거리·비용의 즉시 판독 |
| 행동열 | 6슬롯. 배치한 카드와 공용 행동, 비용, 예상 사거리, 보조장비 부착 |
| 상세 효과 패널 | 선택하거나 마우스를 올린 카드의 대상·사거리·피해·효과·비용·조건·대성 |
| 결과 로그 | 이동 → 방어·회피 → 공격·밀치기 → 피해·상태 → 회복 순서 |

## 3. 무공 카드 아트 방향

### 3.1 전체 분위기

- 배경은 먹색·흑청색 중심의 어두운 무협 분위기로 구성한다.
- 카드 본체는 오래된 화선지·양피지처럼 밝은 미색 종이를 사용한다.
- 테두리는 얇은 금속성 금장과 전통 문양을 사용하되, 장식이 정보 영역을 침범하지 않게 한다.
- 삽화는 수묵화와 동양 무협 일러스트의 중간 톤으로 통일한다.
- 카드 외곽에는 약한 그림자를 주어 어두운 배경에서 한 장의 실물 패처럼 분리되게 한다.
- 희귀도나 대성 단계가 추가되더라도 카드 골격과 정보 위치는 바꾸지 않는다.

### 3.2 카드 정보 계층

| 위치 | 표시 정보 | 목적 |
|---|---|---|
| 좌측 상단 | 문파 배지 | 어느 문파의 무공인지 즉시 구분 |
| 상단 중앙 | 사거리 | 전투 배치 전 가장 먼저 확인하는 수치 |
| 우측 상단 | 무공 종류 배지 | 권법·각법·심법·검법·창법 등 분류 |
| 상단 제목부 | 무공 이름 | 카드의 핵심 식별 정보 |
| 중앙 | 무공 삽화 | 무공의 성격과 문파 이미지를 기억하게 함 |
| 하단 | 비용 아이콘과 숫자 | 배치 가능한지 빠르게 판단 |

카드의 기본 읽기 순서는 `문파 → 무공 이름 → 무공 종류 → 사거리 → 삽화 → 비용`이다.

### 3.3 배지 규칙

- 문파 배지는 문파명 또는 문파 상징을 함께 사용한다. 색만으로 구분하지 않는다.
- 무공 종류 배지는 아이콘과 짧은 텍스트를 함께 사용할 수 있다.
- 예시: `(무당) 태극권 (권법)`.
- 문파 배지와 종류 배지는 크기와 외곽 형태를 다르게 하여 서로 혼동되지 않게 한다.

### 3.4 사거리 표기

전투 규칙서의 거리 정의를 그대로 사용한다.

| 표기 | 의미 |
|---|---|
| `0` | 같은 칸의 대상 |
| `1` | 인접 1칸 |
| `2~4` | 최소·최대 사거리 |
| `자신` | 자신에게 사용하는 심법·호신기 |
| `전체` | 전장 전체 또는 전체 대상 |
| `-` | 거리 개념이 없는 효과 |

사거리 숫자는 카드 상단 중앙에 고정한다. 상세 효과 패널에서도 같은 값을 텍스트로 다시 표시한다.

### 3.5 비용 표기

카드 하단 비용은 항상 다음 순서로 배치한다.

1. 시간 슬롯
2. 행동력
3. 기력
4. 내공

사용하지 않는 비용도 `0`으로 표시해 카드마다 아이콘 위치가 흔들리지 않게 한다. 시간 슬롯과 행동력은 서로 다른 개념이므로 아이콘·툴팁·상세 텍스트를 분리한다.

예시: `시간 슬롯 1 / 행동력 1 / 기력 1 / 내공 0`.

## 4. 카드 상호작용

### 4.1 기본 상태

- 카드 본체에는 전투 판단에 필요한 최소 정보만 표시한다.
- 긴 효과 문장은 카드 삽화 위에 직접 배치하지 않는다.
- 사용 불가능한 카드는 숨기지 않고 어둡게 비활성화한다.
- 비활성화 이유는 마우스를 올렸을 때 즉시 표시한다.

### 4.2 마우스 오버

- 카드에 마우스를 올리면 카드 옆에 작은 상세 효과 팝업을 표시한다.
- 팝업은 카드와 행동열을 가리지 않는 방향으로 자동 배치한다.
- 팝업은 포인터가 카드 또는 팝업 위에 있는 동안 유지한다.
- 짧은 확인을 위한 팝업이므로 스크롤이 필요할 정도로 길어지면 요약과 전체 설명을 분리한다.

### 4.3 클릭 또는 선택

- 카드를 클릭하면 우측 상세 효과 패널을 고정한다.
- 다른 카드를 선택하기 전까지 상세 패널이 유지된다.
- 행동열에 배치한 카드도 클릭해 같은 상세 패널을 열 수 있다.
- 마우스 오버 팝업과 클릭 상세 패널은 동일한 데이터 원본을 사용한다.

## 5. 상세 효과 패널

### 5.1 표시 순서

1. 무공 이름
2. 대상
3. 사거리
4. 피해 또는 회복량
5. 핵심 효과
6. 비용
7. 사용 조건
8. 대성·성급 해금 효과

### 5.2 태극권 예시

> 아래 수치는 레이아웃 예시다. 실제 전투 수치는 `02_COMBAT_RULES.md`와 문파·대성 데이터 문서를 따른다.

**태극권**

- 대상: 같은 칸의 적
- 사거리: 0
- 피해: 데이터 기준값
- 효과: 막기 피해 경감 강화
- 비용: 시간 슬롯 1 / 행동력 1 / 기력 1 / 내공 0
- 조건: 없음
- 대성: 5성 시 절초 해금

예시 이미지처럼 카드 우측에 상세 패널을 배치하며, `사거리`와 `비용`을 반드시 포함한다.

## 6. 행동열 배치 규칙

- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전이다.
- 시간 슬롯과 행동력은 분리해 표기한다.
- 행동력·기력·내공 부족, 사거리 불가, 재장전 필요를 배치 시점에 표시한다.
- 이동 후 사거리 판정임을 공격·밀치기 미리보기에서 보여 준다.
- 암기벨트·화살통은 기존 행동 위에 부착된 무행동 사용 횟수로 표시한다.
- 방패는 막기 증가와 회피 감소를 동시에 표시한다.
- 양측 행동열은 모두 잠긴 뒤 같은 화면에서 공개한다.

## 7. 색상·가독성 규칙

- 문파·종류·자원은 색상만이 아니라 아이콘·텍스트·숫자로 함께 구분한다.
- 밝은 종이 카드 위 본문은 짙은 먹색을 사용한다.
- 상세 패널의 핵심 수치는 금색, 이로운 변화는 녹색 계열, 불리한 변화는 적색 계열을 보조로 사용할 수 있다.
- 비용 아이콘은 카드와 상세 패널에서 동일한 형태와 순서를 유지한다.
- 작은 화면에서도 카드 제목, 상단 사거리, 하단 비용 숫자는 축소 전 최소 판독 크기를 유지한다.

## 8. 결과 설명 기준

- 결과 로그는 항상 ‘공용 행동 → 무기 규칙 → 유파/대성 수정 → 최종 결과’ 순으로 설명한다.
- 예: ‘막기 40% + 금강호신 15% = 55% 경감’
- 예: ‘활 공격 후 재장전 필요 → 화살통 1회 사용으로 즉시 장전 유지’
- 예: ‘마화 사용: 체력 5 소모, 공격 강화, 마화 1중첩. 라운드 종료 시 중첩 피해’
- AI가 선택한 행동은 공개 후 현재 턴의 공개 정보와 행동열만으로 설명한다.

## 9. UI·아트 완료 기준

- 카드만 보고 3초 안에 문파, 무공명, 종류, 사거리, 주요 비용을 말할 수 있다.
- 마우스 오버로 대상·사거리·피해·효과·비용·조건·대성을 읽을 수 있다.
- 클릭한 상세 패널이 다른 카드를 선택하기 전까지 유지된다.
- 시간 슬롯과 행동력을 서로 다른 비용으로 이해할 수 있다.
- 거리 0을 ‘인접’이 아니라 ‘같은 칸’으로 일관되게 표시한다.
- 어떤 문파를 선택해도 공용 행동과 문파 수정 효과의 차이를 읽을 수 있다.
- 소림·당문뿐 아니라 추가 6세력의 장전·방패·독·마화·기력 탈취 차이가 숨겨지지 않는다.
- 예시 이미지의 임시 수치가 실제 전투 데이터 기준으로 오인되지 않는다.
