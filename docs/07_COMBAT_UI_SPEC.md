# 전투 UI·상호작용 명세

> 목적: 플레이어가 10칸 거리, 6슬롯 행동열, 양측 자원을 한 화면에서 읽고 AI가 치팅하지 않았음을 신뢰하게 한다.

## 1. 화면 구성

| 영역 | 위치 | 반드시 표시할 정보 |
|---|---|---|
| 적 상태 | 상단 왼쪽 | 체력, 기력, 내공, 무기, 기본 사거리, 상태이상 |
| 라운드 상태 | 상단 중앙 | 라운드 번호, 계획 단계, 양측 잠금 상태 |
| 플레이어 상태 | 상단 오른쪽 | 체력, 기력, 내공, 무기, 상태이상 |
| 10칸 전장 | 화면 중앙 | 1~10 번호, A·B 위치, 같은 칸 표시, 밀치기 방향 |
| 플레이어 행동 목록 | 하단 왼쪽 | 현재 무기 행동, 공통 행동, 자원 비용, 사거리 |
| 6슬롯 행동열 | 하단 중앙 | 1~6 슬롯, 배치 행동, 자동 회수·장전, 즉발 암기 |
| 상대 정보 | 하단 오른쪽 | 대표 무공, 선호 거리, 정탐 정보, 장전·회수 상태 |
| 결과 해설 | 전장 아래 또는 우측 확장 | 실행 뒤 적중·실패·거리 변화 이유 |

## 2. 계획 단계

### 배치 전

- 현재 위치와 두 전투원의 실제 거리를 강조한다.
- 사용 불가능한 행동은 숨기지 않고 비활성화한다.
- 비활성화 이유를 즉시 표시한다. 예: ‘기력 2 부족’, ‘창 회수 중’, ‘활 미장전’.
- 행동을 선택하면 예상 행동력·기력·내공 총합을 실시간 갱신한다.
- 상대의 행동열은 빈 6칸으로만 보인다.

### 행동 배치

- 행동은 1~6 슬롯에 배치한다.
- 자동 회수·장전 같은 강제 후딜은 별도 색으로 표시하고 수정할 수 없게 한다.
- 암기벨트 즉발 암기는 장착한 행동의 작은 부속 아이콘으로 표시한다.
- 거리 0 밀치기는 왼쪽·오른쪽 방향을 반드시 선택해야 잠글 수 있다.
- 무기 교체 슬롯 뒤에는 새 무기의 사거리 아이콘을 표시한다.

### 잠금과 공개

- 플레이어 잠금 전 AI 상태는 ‘상대가 계획을 잠갔습니다’로 표시한다.
- 플레이어가 잠그면 양측 6슬롯을 한 번에 뒤집어 공개한다.
- 공개 전에는 AI의 빈 슬롯 또는 뒷면만 보여 준다.
- 공개 뒤에는 행동열을 수정할 수 없다는 안내를 짧게 표시한다.

## 3. 실행 연출

각 슬롯은 다음 정보 순서로 보여 준다.

1. 현재 슬롯 번호와 양측 행동
2. 이동 애니메이션과 최종 위치
3. 최종 거리
4. 막기·회피·호신 상태
5. 적중·피해·밀치기·상태이상
6. 남은 체력·기력·내공

한 슬롯의 결론은 한 문장으로 요약한다.

- ‘3슬롯: B가 1칸 후퇴해 창 찌르기가 거리 3에서 빗나갔습니다.’
- ‘4슬롯: A의 방패 막기가 화살 피해 60%를 경감했습니다.’
- ‘6슬롯: A의 마화 3중첩이 체력 12를 소모했습니다.’

## 4. 색과 아이콘 규칙

| 정보 | 표현 원칙 |
|---|---|
| 행동력 | 슬롯을 채우는 금색 또는 중립 아이콘 |
| 기력 | 육체 피로를 연상하는 녹색 계열 |
| 내공 | 운기·경공을 연상하는 청색 계열 |
| 체력 | 적색 계열 |
| 마화 | 붉은 테두리와 중첩 숫자 |
| 독 | 자주색 또는 황록색과 종료 시점 |
| 자동 회수·장전 | 잠긴 슬롯 표시 |
| 방패 막기 | 투사체 경감 수치와 회피 약화 동시 표시 |

색만으로 구분하지 않고 아이콘·텍스트·수치를 같이 쓴다.

## 5. UI 완료 기준

- 전장을 처음 보는 플레이어도 A=4, B=7, 거리 3을 3초 안에 읽을 수 있다.
- 행동을 배치할 때 자원 부족과 선행 조건 오류를 클릭 없이 알 수 있다.
- 동시 공개 순간에 AI 행동열이 사전에 잠겼다는 사실이 분명하다.
- 실행 중 어느 공격이 왜 빗나갔는지 현재 위치와 최종 거리만으로 알 수 있다.
- 상태이상·마화·장전·창 회수의 남은 시간과 중첩이 항상 보인다.
