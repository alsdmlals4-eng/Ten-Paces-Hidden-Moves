# 12세력 대회 전투 UI·아트 명세

> 목적: 플레이어가 문파·무공·사거리·비용을 빠르게 읽고, 2수 동시 선택·합 결과·수련·제약을 오해 없이 판단하게 한다.

이 파일은 전투 UI와 무공 카드 아트의 현재 기준 문서다. `docs/[백업]/`과 `docs/[보류]/`의 문서는 현재 구현 기준으로 사용하지 않는다.

## 1. 회차·대진 화면

| 영역 | 표시 내용 |
|---|---|
| 시작 문파 | 핵심무공, 심법, 대표 기믹, 추천 거리 |
| 대진 진행 | 내부전 / 추천패 / 예선 / 예선 결승 / 본선 / 8강 / 준결승 / 결승, 현재 경기와 10전 진행도 |
| 데모 진행 | 1~5전 진행도와 5전째 지역 예선 결승 종료 지점 |
| 다음 상대 | 세력, 무기·사거리, 공개 대표 규칙, 선호 거리, 정탐 정보 |
| 제약 선택 | 삼수양보 등 제약명, 위험도, 전투 불이익, 승리 추가 보상 |
| 보상 선택 | 무공서, 수련 보조, 장비, 회복, 정탐 효과 |
| 수련 패널 | 핵심무공·심법 성급, 다음 필요 포인트, 3·6·9·10성 해금 미리보기 |

대진 진행은 `전장 10칸 / 라운드 10타이밍 / 전체 대회 10전`의 숫자 구조를 명확히 구분해 표시한다.

## 2. 전투 화면

| 영역 | 표시 내용 |
|---|---|
| 적 상태 | 체력·행동력·기력·내공, 무기, 장전, 방패, 독·마화, 대표 기믹 |
| 10칸 전장 | 1~10번 칸, A=4 / B=6 초기 배치, 같은 칸 중첩과 현재 거리 |
| 플레이어 상태 | 체력·행동력·기력·내공, 무기·보조장비, 상태, 연격·반격 등 공용 기믹 |
| 타이밍 진행 | 1~10 진행도와 현재 2수 묶음 표시 |
| 첫 번째 행동 | 현재 묶음의 1수 선택 |
| 두 번째 행동 | 현재 묶음의 2수 선택, 첫 결과 뒤에도 변경 불가 표시 |
| 무공 카드 목록 | 문파·종류·사거리·비용·성급·사용 불가 이유 |
| 상세 효과 패널 | 대상·사거리·합 수치·일반 피해·효과·비용·조건·성급 해금 |
| 결과 로그 | 이동 → 방어 → 공격 성립 → 합/일반 공격 → 피해·상태 → 회복 순서 |

AI의 행동 두 개는 플레이어 선택 전에 잠겼다는 사실을 잠금 아이콘과 설명으로 표시한다. 양측 선택이 끝나기 전에는 AI 카드 내용을 공개하지 않는다.

## 3. 무공 카드 아트 방향

### 3.1 전체 분위기

- 배경은 먹색·흑청색 중심의 어두운 무협 분위기다.
- 카드 본체는 오래된 화선지처럼 밝은 미색 종이를 사용한다.
- 테두리는 얇은 금속성 금장과 전통 문양을 사용하되 정보 영역을 침범하지 않는다.
- 삽화는 수묵화와 동양 무협 일러스트의 중간 톤으로 통일한다.
- 카드 외곽에는 약한 그림자를 주어 어두운 배경과 분리한다.
- 성급이 올라가도 카드 골격과 정보 위치는 바꾸지 않는다.

### 3.2 카드 정보 계층

| 위치 | 표시 정보 |
|---|---|
| 좌측 상단 | 문파 배지 또는 `공용` 배지 |
| 상단 중앙 | 사거리 |
| 우측 상단 | 권법·검법·심법·창법 등 종류 배지 |
| 상단 제목부 | 무공 이름 |
| 중앙 | 무공 삽화 |
| 하단 | 타이밍·행동력·기력·내공 비용 |

카드의 기본 읽기 순서는 `문파/공용 → 무공 이름 → 종류 → 사거리 → 비용 → 효과`다.

### 3.3 사거리 표기

| 표기 | 의미 |
|---|---|
| `0` | 같은 칸 |
| `1` | 인접 칸 |
| `2~4` | 최소·최대 사거리 |
| `자신` | 심법·명상·호신 효과 |
| `전체` | 전장 전체 또는 전체 대상 |
| `-` | 거리 개념 없음 |

### 3.4 비용 표기

카드 하단 비용은 항상 다음 순서로 배치한다.

1. 타이밍
2. 행동력
3. 기력
4. 내공

사용하지 않는 비용도 `0`으로 표시한다. 일반 행동은 타이밍 1을 사용하며 즉발 부착형 장비만 타이밍 0을 사용할 수 있다.

## 4. 카드 상태

| 상태 | 표현 |
|---|---|
| 기본 | 정상 카드 |
| 마우스 오버 | 작은 상세 팝업 |
| 선택 | 테두리 강조, 우측 상세 패널 고정 |
| 1수 배치 | 첫 번째 행동 영역과 같은 번호 표시 |
| 2수 배치 | 두 번째 행동 영역과 같은 번호 표시 |
| 잠금 완료 | 자물쇠 표시, 수정 불가 |
| 사용 불가 | 어둡게 표시하고 부족 자원·사거리·장전 이유 제공 |
| 합 예상 | 양측 공격 가능 시 `합 가능` 표시, 수치는 상대 행동 공개 전 확정하지 않음 |
| 실행 중 | 현재 타이밍 카드 강조 |
| 실행 완료 | 비용·상태·결과 요약 표시 |

## 5. 상세 효과 패널

표시 순서:

1. 무공 이름과 현재 성급
2. 문파 또는 공용 분류
3. 대상
4. 사거리
5. 합 수치와 일반 피해
6. 핵심 효과
7. 비용
8. 사용 조건
9. 3·6·9성 기술과 10성 절초·진의
10. 관련 심법·기믹 수정

### 태극권 예시

- 대상: 사거리 안의 적
- 합: 공격 대 공격일 때 반격 수정 적용 가능
- 효과: 상대 공격 수치 일부를 자신의 합 수치에 가산
- 조건: 상대가 같은 타이밍에 유효한 공격 사용
- 비용: 데이터 기준

예시 수치는 레이아웃 참고이며 실제 값은 전투 규칙과 무공 데이터가 기준이다.

## 6. 2수 선택 상호작용

- 플레이어는 첫 번째와 두 번째 행동을 모두 선택해야 잠글 수 있다.
- 두 번째 행동의 예상 사거리와 자원은 첫 번째 행동이 정상 실행된다는 가정으로 미리보기 한다.
- 첫 번째 행동 실패 가능성이 있으면 두 번째 미리보기에 조건부 경고를 표시한다.
- 양측 잠금 후 행동 두 개를 한꺼번에 공개한다.
- 첫 번째 타이밍을 재생한 뒤 두 번째 타이밍을 재생한다.
- 첫 결과를 본 뒤 두 번째 행동을 바꿀 수 없음을 명확히 표시한다.

## 7. 합 연출과 결과 설명

- 합이 발생하면 짧은 공방 연출 후 최종 결과를 표시한다.
- 공방 횟수는 연출 길이이며 계산 횟수가 아니다.
- 로그는 `기본 합 수치 → 무기 수정 → 성급 기술 → 심법·기믹 → 최종 합 수치 → 차이 피해` 순서로 설명한다.
- 동률이면 `합 무승부 / 체력 피해 없음`을 표시한다.
- 태극권 반격처럼 상대 공격을 이용한 보너스는 별도 행으로 표시한다.

## 8. 수련 화면

전투 승리 후 다음 순서로 표시한다.

1. 기본 수련 포인트 `+5`
2. 성과 등급 확정 보너스
3. 확률 보너스 성공·실패 결과
4. 제약 승리 추가 보상
5. 최종 수련 포인트
6. 무공별 포인트 분배

성과 등급별 총 획득 범위:

| 등급 | 총 수련 포인트 |
|---|---:|
| C | 6~7 |
| B | 8~10 |
| A | 13~14 |
| S | 16 |

각 무공에는 현재 성급, 현재 누적 포인트, 다음 성급 비용, 다음 해금 효과를 표시한다. 분배 확정 전에는 되돌리기를 제공한다.

### 데모 절초 목표 표시

- 5전째 지역 예선 결승 전까지 남은 수련 횟수를 표시한다.
- 핵심무공 3성→10성에 필요한 총 49포인트와 현재 확보량을 표시한다.
- 첫 4전에서 A등급 이상을 유지하면 절초 개방 가능 상태를 `예선 결승 사용 가능`으로 표시한다.
- B등급 중심 진행에서는 `예선 결승 전 개방 미보장` 안내를 표시한다.
- 4전 종료 후 10성이 되면 절초 카드가 5전 전투 준비 화면에 신규 해금으로 강조된다.

### 10성 제한 표시

- 핵심무공 10성 슬롯: 1개
- 심법 10성 슬롯: 1개
- 심법 보유 슬롯: 2개

한 계열에서 이미 10성이 존재하면 다른 무공의 9→10 투자는 잠금 이유와 함께 비활성화한다.

## 9. 제약 선택 화면

- 제약명은 `삼수양보`, `단전봉맥`, `기혈역행`, `일장지양`처럼 무협식 이름을 사용한다.
- 이름 아래에 현대적인 규칙 설명을 함께 표시한다.
- 위험도, 전투 불이익, 승리 보상을 같은 카드에 표시한다.
- 문파 핵심 기믹을 완전히 사용할 수 없게 만드는 제약에는 경고가 아니라 생성 금지 규칙을 적용한다.

## 10. 완료 기준

- 카드만 보고 3초 안에 문파/공용, 무공명, 종류, 사거리, 주요 비용을 말할 수 있다.
- 첫 번째와 두 번째 행동, 잠금 상태, 현재 타이밍을 혼동하지 않는다.
- 합과 일반 공격의 차이를 로그와 연출로 설명할 수 있다.
- 성과 보상과 확률 보상, 제약 보상이 분리되어 보인다.
- 3·6·9성 기술과 10성 절초·진의의 다음 해금을 수련 화면에서 확인할 수 있다.
- 심법 보유 2개와 계열별 10성 제한을 UI에서 오해하지 않는다.
- 데모 진행도 5전과 전체 대회 진행도 10전을 혼동하지 않는다.
- 4전 종료 후 절초를 개방한 경우 5전째 지역 예선 결승에서 신규 절초를 바로 선택할 수 있다.
