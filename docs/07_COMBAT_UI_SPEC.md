# 전투 UI·상호작용 명세

> 목적: 플레이어가 거리 2에서 시작한 소림 대 당문 결투를 보고, 공용 행동과 유파 규칙의 차이를 즉시 이해한다.

## 1. 화면 구성

| 영역 | 위치 | 표시 내용 |
|---|---|---|
| 적 상태 | 상단 왼쪽 | 당문 체력·기력·내공, 암기 장전 상태, 독침, 암기벨트 잔여 횟수 |
| 라운드 상태 | 상단 중앙 | 라운드 번호, 계획 단계, AI 잠금 상태 |
| 플레이어 상태 | 상단 오른쪽 | 소림 체력·기력·내공, 금강호신 |
| 10칸 전장 | 화면 중앙 | 1~10 번호, A=4, B=6, 현재 거리=2 |
| 공용 행동 목록 | 하단 왼쪽 | 접근, 후퇴, 공격, 막기, 회피, 밀치기, 재장전, 대기 |
| 6슬롯 행동열 | 하단 중앙 | 1~6 슬롯, 행동 비용, 암기벨트 부착 아이콘 |
| 무기·유파 설명 | 하단 오른쪽 | 현재 무기 규칙, 유파 규칙, 사용 불가 이유 |
| 결과 해설 | 전장 아래 | 이동 뒤 실제 거리와 적중·실패 원인 |

## 2. 정보 계층

- 공용 행동 카드는 모든 전투원에게 같은 이름·아이콘으로 보인다.
- 현재 무기가 쓸 수 없는 공용 행동은 숨기지 않고 비활성화한다.
- 유파 규칙은 공용 행동 카드의 수정 아이콘과 짧은 문장으로 표시한다.
  - 소림 막기: ‘금강호신 +15%’
  - 당문 공격: ‘독침: 적중 시 독 +1’
- 암기벨트는 행동 카드가 아니라 배치한 슬롯 위에 붙는 작은 아이콘이다.

## 3. 계획·공개 UX

- A=4, B=6, 거리 2를 전장 중앙에서 항상 표시한다.
- 행동을 배치하면 행동력·기력·내공 총합과 남은 수치를 즉시 갱신한다.
- 당문이 미장전이면 공격과 밀치기 카드에 ‘재장전 필요’를 표시한다.
- 소림이 거리 2에 있으면 공격 카드에 ‘현재 거리 밖, 이동 뒤 판정’을 표시한다.
- AI가 먼저 잠금한 상태를 ‘상대 계획 잠금 완료’로 명확히 표시한다.
- 플레이어 잠금 뒤 양측 6슬롯을 동시에 뒤집는다.

## 4. UI 수동 완료 기준

- 처음 보는 플레이어가 3초 안에 A=4, B=6, 거리 2를 말할 수 있다.
- 소림과 당문의 공용 행동 목록은 같지만 공격·막기 결과가 다름을 설명할 수 있다.
- 암기벨트가 별도 행동이 아니라 기존 슬롯에 붙는 효과임을 이해한다.
- AI 잠금·동시 공개 흐름을 보고 AI가 행동열을 보고 대응하지 않았다고 판단한다.
