# 보류 항목 인덱스

> 이 폴더는 아이디어를 삭제하지 않고 보존하되, 현재 기획·구현·테스트의 기준에서 제외하기 위한 공간이다.

## 사용 규칙

- 확정되지 않았거나 현재 세로 슬라이스 작업량을 크게 늘리는 항목을 기록한다.
- 보류 문서는 활성 기준 문서의 기본 읽기 대상이 아니다.
- Codex와 다른 구현 에이전트는 사용자가 재개를 확정하기 전까지 이 폴더의 내용을 구현하지 않는다.
- 보류 항목을 재개할 때는 관련 기준 문서에 통합하고, 이 폴더에는 재개 여부와 연결 경로만 남긴다.
- 보류 문서를 `v2`, `final`, `latest` 같은 활성 기준 문서의 대체본으로 사용하지 않는다.
