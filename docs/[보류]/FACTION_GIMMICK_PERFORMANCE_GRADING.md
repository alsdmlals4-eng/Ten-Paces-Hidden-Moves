# 문파별 기믹 수행도 기반 성과 평가 — 보류

## 아이디어

전투 성과 등급 계산에 문파별 핵심 기믹 수행도를 일부 반영한다.

예시:

- 화산: 연격 유지와 연속 검법 성공
- 무당: 태극권 합 반격 성공
- 당문: 독과 암기 연계
- 소림: 거리 0 압박 유지
- 개방: 밀치기로 유효 거리 확보

## 보류 이유

- 12세력마다 별도 평가 항목과 예외 조건이 필요해 구현·테스트 작업량이 크게 증가한다.
- 플레이어가 점수를 위해 같은 기믹을 반복하는 고정 행동을 선택할 위험이 있다.
- 문파별 평가 기준이 서로 공정한지 검증하려면 충분한 플레이 데이터가 필요하다.
- 현재 세로 슬라이스는 남은 체력, 자원 운용, 무효 행동, 합·거리 판단 같은 공용 성과 지표만으로 검증할 수 있다.

## 재개 조건

다음 조건을 모두 충족한 뒤 별도 Plan으로 검토한다.

1. 12세력의 핵심무공·심법과 공용 수정자가 안정됨
2. 공용 성과 등급의 분포와 수련 포인트 곡선이 검증됨
3. 문파별 플레이 로그를 충분히 수집함
4. 기믹 평가가 최적 행동을 왜곡하지 않는다는 테스트 방법이 마련됨

## 현재 구현 금지

사용자가 재개를 명시하기 전까지 성과 등급 계산, UI, 저장 데이터, AI 평가에 이 문서의 문파별 점수를 추가하지 않는다.
