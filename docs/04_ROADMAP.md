# 십보강호 구현 로드맵과 검증 기준

> 이 문서는 구현 순서와 완료 기준이다. 각 단계는 별도 **Plan**과 필요 시 GitHub Issue로 분리한다.

## 단계 1 — 11세력 대회 세로 슬라이스

목적: 시작 문파 선택부터 내부전·예선·본선·결승까지의 9전 회차가, 공용 행동을 공유하면서도 문파별 전략 차이를 만드는지 검증한다.

### 범위

- 10칸 전장, A=4·B=6, 초기 거리 2
- 시작 선택 6세력과 추가 5세력의 전투 데이터
- 모든 세력의 대표 무기, 유파 규칙, 대성 1~5성
- 접근·후퇴·공격·막기·회피·밀치기·무기 교체·재장전
- 6시간 슬롯, 행동력·기력·내공
- AI 행동열 사전 잠금과 동시 공개, 슬롯별 결과 설명
- 9전 대진, 보상·수련도·대성·정탐·재도전

### 승인 기준

- [ ] [11세력 대회 세로 슬라이스 구현 Plan](../plans/2026-07-16-combat-poc-plan.md)의 완료 기준을 통과한다.
- [ ] [전체 세력·대회 테스트 체크리스트](08_TEST_CHECKLIST.md)의 필수 항목을 통과한다.
- [ ] 시작 6세력 각각으로 본선 진입이 가능하다.
- [ ] 추가 5세력은 적 또는 습득 무공으로 실제 전투에서 작동한다.
- [ ] 공용 행동 하나를 고칠 때 특정 문파 전용 예외가 엔진에 추가되지 않는다.

## 단계 2 — 밸런스와 콘텐츠 밀도

세로 슬라이스 검증 후 승률·재도전율·문파별 선호도를 보고 수치와 보상 풀을 조정한다. 필요할 때만 각 세력의 추가 무공·장비를 늘린다.

## 단계 3 — 후속 확장

세로 슬라이스가 안정된 뒤, 위 11세력 밖의 문파·무기·대회 변주를 추가하는 Plan을 작성한다.
