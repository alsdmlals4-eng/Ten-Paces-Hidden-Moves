# 십보강호 구현 로드맵과 검증 기준

> 이 문서는 구현 순서와 완료 기준이다. 각 단계는 별도 Plan과 필요 시 GitHub Issue로 분리한다.

## 단계 0 — 인수 가능한 컨텍스트 유지

목적: 작업자가 바뀌어도 저장소만 읽고 현재 방향과 다음 작업을 복원한다.

- `AGENTS.md → ACTIVE_CONTEXT.md → DOCUMENTATION_MAP.md` 읽기 순서를 유지한다.
- 작업 종료 시 확정 사항, 미확정 사항, 테스트 상태, 다음 작업을 최신화한다.
- 폐기 규칙과 중복 문서를 제거하고 보류 항목을 활성 구현과 분리한다.
- 실제 구현과 테스트를 확인하지 않은 상태는 완료로 기록하지 않는다.

## 단계 1A — 예선 결승 데모

목적: 5전 데모에서 10칸 전장·10타이밍·2수 선택·합·수련·행운·상대 읽기·제약·절초의 핵심 재미를 검증한다.

### 데모 범위

- 1~2전: 내부전.
- 3~4전: 예선.
- 5전: 예선 결승·데모 최종전.
- 시작 핵심무공과 심법은 각각 3성.
- 3전부터 공용·외문 무공이나 보조장비 보상 후보 제공.
- 승리 기본 5 + 등급 보너스(C1/B3/A5/S7) + 비공개 행운 앞면 0~3.
- 첫 4전 A 4회 약 50%, A3+S1 약 60%, A3+B1 약 30%의 절초 개방 목표.
- 상대 등급·행운·수련 스타일은 회차 생성 시 고정.
- 상대 정보는 개발용 태그가 아니라 무림식 이명·풍문·전적·정탐으로 표시.
- 상대 정보를 보고 삼수양보 등 제약을 선택.

### 데모 승인 기준

- [ ] 시작 문파를 선택하고 5전 예선 결승까지 진행할 수 있다.
- [ ] 3~5전이 하나의 예선 구간으로 표시된다.
- [ ] 성과 등급과 행운 결과가 `A++` 형식과 실제 포인트 정산에서 일치한다.
- [ ] 첫 4전 A 4회 핵심무공 올인 시 절초 개방률이 목표 범위에 가깝다.
- [ ] 절초 없이도 예선 결승을 시작하고 완료할 수 있다.
- [ ] 절초를 개방한 회차에서는 5전에서 카드·비용·합·효과·연출을 확인할 수 있다.
- [ ] 상대 이명·풍문이 내부 강도·행운·수련 스타일과 모순되지 않는다.
- [ ] 재도전에서 같은 상대의 성장과 암시 정보가 바뀌지 않는다.

## 단계 1B — 12세력 10전 대회 세로 슬라이스

목적: 시작 문파 선택부터 내부전·예선·본선·8강·준결승·결승까지의 10전 회차가 공용 전투 엔진 위에서 문파별 핵심무공·심법·합·수련·제약의 차이를 만드는지 검증한다.

### 전체판 범위

- 전장 10칸, A=4·B=6, 초기 거리 2.
- 한 라운드 행동 타이밍 10개, 2수 비공개 동시 선택 5회.
- 대회 10전, 8전 8강·9전 준결승·10전 결승.
- 동일 타이밍 쌍방 공격의 합 판정과 차이 피해.
- 시작 선택 6세력과 추가 6세력의 전투 데이터.
- 문파 전용 핵심무공·심법, 각 무공 종류의 공용 무공 2종.
- 핵심무공·심법 1~10성, 3·6·9성 해금, 절초·진의.
- 핵심무공 10성 최대 1개, 심법 10성 최대 1개, 심법 보유 최대 2개.
- 접근·후퇴·공격·막기·회피·밀치기·교체·재장전·명상.
- AI 2수 사전 잠금과 동시 공개, 타이밍별 결과 설명.
- 상대 A계열 60%·B계열 40% 기반 성장, 행운 `+`, 수련 스타일과 단계별 강도.
- 무림식 정보 표현과 정탐 강화.
- 보상·수련·제약·재도전·저장.

### 전체판 승인 기준

- [ ] [구현 Plan](../plans/2026-07-16-combat-poc-plan.md)의 완료 기준을 통과한다.
- [ ] [테스트 체크리스트](08_TEST_CHECKLIST.md)의 필수 항목을 통과한다.
- [ ] 시작 6세력 각각으로 본선 진입이 가능하다.
- [ ] 추가 6세력은 적 또는 습득 무공으로 실제 전투에서 작동한다.
- [ ] 공용 행동·합 규칙을 고칠 때 특정 문파 전용 엔진 분기가 추가되지 않는다.
- [ ] 일반적인 전체판 진행에서 핵심무공 10성·심법 8성 전후가 가능하다.
- [ ] `10칸·10타이밍·10전` 구조가 UI·저장·테스트에서 동일하다.
- [ ] 상대 생성 결과와 투자 스타일이 저장·재도전에서 유지된다.

## 단계 2 — 밸런스와 콘텐츠 밀도

데모와 전체 세로 슬라이스 검증 후 다음 지표를 보고 수치와 보상 풀을 조정한다.

- 문파별 승률과 합 발생률.
- 절초 개방 경기와 데모 절초 체험률.
- C/B/A/S 비율과 행운 `+` 분포.
- 상대 강도 체감과 이명·풍문 해석 정확도.
- 수련 스타일별 승률과 제약 선택률.
- 재도전율과 보상 선택 편중.

필요할 때만 공용 무공, 이명·풍문 문구, 장비와 상대 프로필을 늘린다.
