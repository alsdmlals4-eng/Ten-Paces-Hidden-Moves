# 십보강호 구현 로드맵과 검증 기준

> 이 문서는 구현 순서와 완료 기준이다. 각 단계는 별도 Plan과 필요 시 GitHub Issue로 분리한다.

## 단계 1A — 예선 결승 데모

목적: 5전짜리 데모에서 10칸 전장·10타이밍·2수 선택·합·수련·제약·절초의 핵심 재미를 검증한다.

### 데모 범위

- 1~2전: 내부전
- 3전: 추천패 쟁탈전
- 4전: 지역 예선
- 5전: 지역 예선 결승
- 시작 핵심무공과 심법은 각각 3성
- 첫 4전에서 A·S등급을 다수 획득하고 핵심무공에 집중 투자하면 4전 종료 후 10성 절초 개방
- 5전째 지역 예선 결승에서 절초를 실제로 사용하고 효과를 검증
- 데모 종료 시 전체판의 본선·8강·준결승·결승 진행을 예고

### 데모 승인 기준

- [ ] 시작 문파를 선택하고 5전째 지역 예선 결승까지 진행할 수 있다.
- [ ] 첫 4전을 모두 A등급 이상으로 승리하고 핵심무공에 전부 투자하면 절초가 개방된다.
- [ ] B등급 중심 플레이에서는 예선 결승 전 절초 개방이 보장되지 않는다.
- [ ] 지역 예선 결승에서 절초의 카드·비용·합·효과·연출을 확인할 수 있다.
- [ ] 절초 없이도 예선 결승 진입은 가능하며, 절초는 상위 성과의 조기 보상으로 작동한다.

## 단계 1B — 12세력 10전 대회 세로 슬라이스

목적: 시작 문파 선택부터 내부전·예선·본선·8강·준결승·결승까지의 10전 회차가, 공용 전투 엔진 위에서 문파별 핵심무공·심법·합·수련·제약의 차이를 만드는지 검증한다.

### 전체판 범위

- 전장 10칸, A=4·B=6, 초기 거리 2
- 한 라운드 행동 타이밍 10개, 2수 비공개 동시 선택 5회
- 대회 10전, 8전 8강·9전 준결승·10전 결승
- 동일 타이밍 쌍방 공격의 합 판정과 차이 피해
- 시작 선택 6세력과 추가 6세력의 전투 데이터
- 문파 전용 핵심무공·심법, 각 무공 종류의 공용 무공 2종
- 핵심무공·심법 1~10성, 3·6·9성 해금, 절초·진의
- 핵심무공 10성 최대 1개, 심법 10성 최대 1개, 심법 보유 최대 2개
- 접근·후퇴·공격·막기·회피·밀치기·교체·재장전·명상
- AI 2수 사전 잠금과 동시 공개, 타이밍별 결과 설명
- 승리 기본 수련 포인트 5, 성과 등급 보너스와 확률 보너스
- 삼수양보 등 전투 전 제약 선택과 추가 보상
- 보상·수련·정탐·재도전·저장

### 전체판 승인 기준

- [ ] [12세력 대회 세로 슬라이스 구현 Plan](../plans/2026-07-16-combat-poc-plan.md)의 완료 기준을 통과한다.
- [ ] [전체 세력·대회 테스트 체크리스트](08_TEST_CHECKLIST.md)의 필수 항목을 통과한다.
- [ ] 시작 6세력 각각으로 본선 진입이 가능하다.
- [ ] 추가 6세력은 적 또는 습득 무공으로 실제 전투에서 작동한다.
- [ ] 공용 행동·합 규칙을 고칠 때 특정 문파 전용 엔진 분기가 추가되지 않는다.
- [ ] 일반 B등급 기대값으로 결승 전 핵심무공 10성·심법 8성 전후가 가능하다.
- [ ] 핵심무공 집중 시 절초가 본선 전후에 개방되어 남은 경기에서 사용된다.
- [ ] `10칸·10타이밍·10전` 구조가 UI·저장·테스트에서 동일하게 유지된다.

## 단계 2 — 밸런스와 콘텐츠 밀도

데모와 전체 세로 슬라이스 검증 후 승률, 합 발생률, 절초 개방 경기, 재도전율, 문파별 선호도, 제약 선택률을 보고 수치와 보상 풀을 조정한다. 필요할 때만 공용 무공과 장비를 늘린다.

## 단계 3 — 후속 확장

세로 슬라이스가 안정된 뒤 12세력 밖의 문파, 추가 무기, 대회 변주, 보류 기능을 별도 Plan으로 재검토한다.
