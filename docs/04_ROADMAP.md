# 십보강호 구현 로드맵과 검증 기준

> 이 문서는 구현 순서와 완료 기준이다. 각 단계는 별도 Codex Goal 또는 GitHub Issue로 분리한다.

## 단계 1 — 전투 PoC

목표: 10칸 전장에서 공정한 비공개 동시 선택 결투가 성립하는지 검증한다.

### 범위

- 10칸 전장, 같은 칸 위치, 상호 통과
- 6시간 슬롯과 행동력
- 기력·내공
- 보법, 경공, 권법, 검법, 창법, 궁술
- 이동 후 공격 판정
- 공통 막기·회피보·밀치기
- AI 행동열 사전 잠금과 동시 공개
- 슬롯별 결과 설명

### 완료 기준

- [ ] 거리 0 권법, 거리 1 검법, 거리 2 창법, 거리 3 활 공격이 정확히 판정된다.
- [ ] 양측이 같은 슬롯에 이동하면 상호 통과 또는 같은 칸 위치가 정확히 처리된다.
- [ ] 공격은 이동 뒤 실제 거리로 판정된다.
- [ ] 양측 동시 공격이 모두 사거리 안이면 양쪽 피해가 적용된다.
- [ ] 밀치기는 현재 무기 사거리 안에서만 적중하고, 끝 칸에서는 기력을 추가 감소시킨다.
- [ ] AI는 플레이어의 잠금 뒤 행동열을 수정하지 않는다.
- [ ] 전투 결과 화면에서 각 실패·적중 이유를 확인할 수 있다.

## 단계 2 — 무공과 장비 PoC

목표: 대성, 보조병장, 무기 교체가 전략 선택을 만드는지 검증한다.

### 범위

- 화산, 소림, 당문, 마교의 대표 무공 4종
- 무공 1성·2성·5성의 최소 구현
- 외공, 심법, 보법, 호신공, 독공, 의술
- 주병장·보조병장·호신구·기물
- 예비 병장과 슬롯 1칸 무기 교체
- 암기벨트, 화살통
- 방패의 투사체 상성과 회피 약화
- 마공과 마화

### 완료 기준

- [ ] 대성 1개와 혼합 빌드가 서로 다른 전투 패턴을 만든다.
- [ ] 방패 미장착 상태에서도 막기가 가능하다.
- [ ] 방패 장착 시 투사체 방어는 강해지고 회피보 효율은 낮아진다.
- [ ] 암기벨트는 정해진 횟수만큼 행동 슬롯 없이 즉발 암기를 사용한다.
- [ ] 화살통은 정해진 횟수만큼 장전 슬롯을 면제한다.
- [ ] 마공은 체력 비용과 마화 종료 피해를 명확히 표시한다.
- [ ] 무기 교체 뒤에는 새 무기 사거리와 무공만 사용할 수 있다.

## 단계 3 — 로그라이트 회차

목표: 9전·60분 회차에서 수련과 보상이 빌드 선택으로 이어지는지 검증한다.

### 범위

- 시작 세력 6종
- 9전 대회 구조
- 수련도, 1~5성 대성, 운용 무공서 6칸
- 경기 후 3개 보상 선택
- 적 세력과 습득 무공
- 명망, 최초 무료 재도전, 회차당 재도전 1회
- 재기 전리품과 재기 수련도

### 완료 기준

- [ ] 한 회차에 대성 1개 또는 혼합 3~4성 빌드가 가능하다.
- [ ] 모든 무공을 대성하는 선택은 불가능하다.
- [ ] 패배 후 재도전은 장비·무공서 조정을 허용하고 AI 행동열을 재생성한다.
- [ ] 영구 성장이 직접 공격력 누적보다 선택지 해금 중심으로 작동한다.
- [ ] 창·궁·방패·점혈·독공 계통이 적 세력과 보상에서 등장한다.

## 단계 4 — 밸런스와 콘텐츠 확장

목표: 시작 6세력과 추가 세력의 전투 정체성을 유지한다.

### 우선 콘텐츠

1. 양가장 창법
2. 백우궁 궁술
3. 관군·표국 방패술
4. 아미 점혈
5. 오독 독공
6. 전진 심법
7. 곤륜 변칙 검결

### 밸런스 체크

- 권법이 거리 0에 붙은 뒤에도 창·궁이 무조건 패배하지 않는가?
- 방패가 활을 억제하지만 활을 무력화하지는 않는가?
- 경공이 행동력 1이라는 장점에 비해 내공 대가가 충분한가?
- 마공이 체력·마화 대가보다 안정적인 최적 해법이 되지 않는가?
- 저성급 상시 효과를 많이 모으는 빌드가 대성 빌드를 압도하지 않는가?
- AI가 치팅 없이도 읽을 만하고 도전적인가?
