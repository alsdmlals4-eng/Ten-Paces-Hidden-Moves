# 십보강호 구현 로드맵과 검증 기준

> 이 문서는 구현 순서와 완료 기준이다. 각 단계는 별도 **Plan**과 필요 시 GitHub Issue로 분리한다.

## 단계 1 — 소림 대 당문 전투 PoC

목적: 공용 행동 중심의 10칸 비공개 동시 선택 결투가 재대결 욕구를 만드는지 검증한다.

### 범위

- 시작 배치 A=4, B=6, 초기 거리 2
- 소림 권법·금강호신
- 당문 암기·독침·암기벨트
- 접근, 후퇴, 공격, 막기, 회피, 밀치기, 재장전
- 6시간 슬롯, 행동력·기력·내공
- AI 행동열 사전 잠금과 동시 공개
- 슬롯별 결과 설명

### 승인 기준

- [ ] [소림 대 당문 전투 PoC 실행 Plan](../plans/2026-07-16-combat-poc-plan.md) 승인 기준을 통과한다.
- [ ] [소림 대 당문 전투 PoC 테스트 체크리스트](08_TEST_CHECKLIST.md)의 필수 항목을 통과한다.
- [ ] 새 문파·무공·경제 시스템 없이 재대결 의향을 확인한다.

## 단계 2 — 시작 6세력 확장

PoC 통과 뒤 화산, 개방, 무당, 마교를 공용 행동 변형 규칙으로 추가한다. 각 세력은 대표 규칙 하나와 보조 규칙 하나만 추가한다.

## 단계 3 — 장거리·방패 확장

양가장 창법, 백우궁 궁술, 관군 방패술을 추가해 거리 2·3 이상과 방패 상성을 검증한다.

## 단계 4 — 회차와 대성

9전·60분 회차, 수련도, 대성, 보상 선택, 명망, 재도전을 추가한다.

## 보류

- 문파 대량 추가
- 무공 20개 이상 설계
- 세계관·대화·관계
- 복잡한 랜덤 손패·덱빌딩
- 장비 강화·경제 시스템
