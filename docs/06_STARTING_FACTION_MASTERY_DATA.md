# 시작 6세력 대성 데이터

> 목적: 시작 세력마다 다른 거리·자원·행동열 감각을 제공한다. 모든 효과는 회차 내 수련도로 해금하며, 회차 종료 시 초기화된다.

## 1. 공통 수련도 규칙

| 성급 | 다음 성급까지 필요 수련도 | 누적 수련도 | 역할 |
|---:|---:|---:|---|
| 1성 | 4 | 4 | 기본 상시 효과 |
| 2성 | 7 | 11 | 대표 기술 |
| 3성 | 11 | 22 | 대표 기술 강화 |
| 4성 | 16 | 38 | 대응·추격 기술 |
| 5성·대성 | - | 38 | 빌드 완성 규칙 |

- 시작 시 대표 무공 1개는 1성으로 제공한다.
- 시작 보조 무공 1개를 1성으로 선택한다.
- 5성에 필요한 누적 수련도 38은 한 회차에 주력 무공 하나를 대성하거나, 여러 무공을 3~4성으로 나누도록 설계한 초기값이다.
- 실제 수련도 보상량은 9전 회차에서 주력 대성 1개가 가능한지로 검증한다.

## 2. 화산파

### 대표: 매화검법

| 성급 | 효과 |
|---:|---|
| 1성 | 검 공격 피해 +2 |
| 2성 | **낙화**: 거리 1, 행동력 2, 기력 2, 피해 15. 직전 슬롯에 보법·경공을 사용했다면 적중 뒤 원하는 방향으로 1칸 이동 |
| 3성 | 낙화가 보법 직후 사용되면 기력 1 회복 |
| 4성 | **화영**: 상대와 교차한 뒤 거리 1일 때 사용 가능. 행동력 2, 기력 2, 피해 12, 적중 시 다음 검 공격 피해 +4 |
| 5성 | 한 라운드에 서로 다른 매화검 기술을 2개 이상 사용하면, 그 라운드 첫 보법의 기력 비용 0 |

보조 후보: 청풍심법(내공 회복), 낙화보(교차·추격)

## 3. 개방

### 대표: 개방곤법

| 성급 | 효과 |
|---:|---|
| 1성 | 곤 공격과 공통 밀치기 사용 시 상대 기력 손실 +1 |
| 2성 | **봉압**: 거리 1~2, 행동력 2, 기력 2, 피해 12. 적중 시 상대 기력 -1 |
| 3성 | 막기 성공 뒤 다음 곤 공격의 기력 비용 -1 |
| 4성 | **횡타**: 거리 1~2, 행동력 2, 기력 3, 피해 8, 적중 시 1칸 밀치기 |
| 5성 | 같은 라운드에 막기·밀치기·곤 공격을 모두 성공하면 상대의 다음 라운드 기력 회복 -1 |

보조 후보: 취개외공(체력·기력), 유운호신공(막기·밀치기)

## 4. 당문

### 대표: 당문암기

| 성급 | 효과 |
|---:|---|
| 1성 | 암기 적중 시 독 1중첩. 독은 라운드 종료 시 중첩당 피해 2 |
| 2성 | **연환비**: 거리 2~4, 행동력 2, 기력 1, 피해 8, 독 1중첩 |
| 3성 | 독 상태 적에게 적중한 암기는 기력 피해 +1 |
| 4성 | **우침**: 거리 2~4, 행동력 2, 기력 2, 피해 6, 독 2중첩 |
| 5성 | 적 독이 3중첩 이상일 때 암기 공격 후 독 1중첩을 소모해 행동력 1을 회복한다. 라운드당 1회 |

보조 후보: 독경(독 강화·해독), 약왕의술(회복·해독)

## 5. 소림

### 대표: 나한권

| 성급 | 효과 |
|---:|---|
| 1성 | 권 공격 피해 +2, 막기 중 기력 손실 -1 |
| 2성 | **나한타**: 거리 0, 행동력 1, 기력 2, 피해 14 |
| 3성 | 막기 성공 뒤 다음 권 공격 기력 비용 -1 |
| 4성 | **반진**: 거리 0, 행동력 2, 기력 2. 같은 슬롯에 적 공격이 사거리 안이면 피해 10을 되돌려 준다 |
| 5성 | 한 라운드에 막기 성공 후 권 공격을 적중하면 금강 자세 1중첩. 다음 라운드 첫 피해를 30% 경감한다 |

보조 후보: 금강호신공(피해 경감·반격), 역근공(체력·공격·기력)

## 6. 무당파

### 대표: 태극검경

| 성급 | 효과 |
|---:|---|
| 1성 | 내공 행동의 내공 비용 -1. 최소 비용은 1 |
| 2성 | **운수**: 행동력 1, 기력 1, 내공 1. 같은 슬롯의 직접 피해 50% 경감 |
| 3성 | 운수로 피해를 경감하면 다음 보법·경공의 기력 비용 -1 |
| 4성 | **태극운검**: 거리 1, 행동력 2, 기력 1, 내공 1, 피해 12. 직전 슬롯에 운수·회피보를 사용했을 때만 가능 |
| 5성 | 한 라운드에 운수·보법 또는 경공·검 공격을 모두 사용하면 라운드 종료 시 내공 +1 |

보조 후보: 태극심법(내공 순환), 운학보(경공·회피)

## 7. 마교

### 대표: 혈염도법

| 성급 | 효과 |
|---:|---|
| 1성 | 체력이 50 이하일 때 도 공격 피해 +3 |
| 2성 | **혈염참**: 거리 1, 행동력 2, 기력 2, 체력 6, 피해 20, 마화 +1 |
| 3성 | 마화가 1 이상일 때 혈염참 피해 +4 |
| 4성 | **천마보**: 행동력 1, 체력 4, 2칸 이동, 마화 +1 |
| 5성 | 같은 라운드에 마공을 2회 사용하면 두 번째 마공 피해 +6. 마화 종료 피해도 +2 |

마화: 마공 1회당 1중첩, 최대 3중첩. 라운드 종료 시 중첩당 피해 4를 받고 전부 제거한다.

보조 후보: 역맥심법(기력·내공 운용), 마혈외공(잃은 체력 기반 강화)

## 8. 데이터 검증 원칙

- 2성 대표 기술은 해당 문파의 핵심 거리에서 가장 읽기 쉬운 행동이어야 한다.
- 4성 기술은 기존 행동열의 빈틈을 보완하되, 다른 문파의 핵심 약점을 완전히 지우지 않는다.
- 대성은 단순 피해 증가보다 행동력·기력·내공·거리 운용 규칙을 바꾼다.
- 독, 마화, 금강 자세 같은 중첩 상태는 최대 중첩과 종료 시점을 항상 UI에 표시한다.
