# 10문파 무공서·숙련·기술 데이터

> 이 문서는 10문파의 무공서 분류, 1~10성 성장 구조, 3·7성 기술과 10성 절초·진의 명칭의 상세 책임 원본이다. 효과·비용·수치는 별도 밸런스 작업에서 확정한다.

## 1. 공식 무공 분류

### 핵심무공 8종

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법 / 궁술 / 암기술`

- 직접 공격, 합, 피해, 밀치기, 장전과 사거리 운용을 담당한다.
- 한 회차에 여러 핵심무공을 익힐 수 있다.
- 한 회차에서 핵심무공 10성은 최대 1개다.
- `부법`은 공식 성장 계통에서 제외한다. 도끼는 개별 무기·연출 태그로 사용할 수 있다.
- `점혈술`은 핵심무공이 아니다. 점혈은 의술의 세부 기술·상태 태그다.

### 보조무공 8종

`심법 / 호신공 / 독공 / 마공 / 보법 / 방패술 / 외공 / 의술`

- 자원, 방어, 이동, 상태, 회복, 문파 기믹과 핵심무공 연계를 담당한다.
- 신법과 경공은 보법의 세부 태그다.
- 의술은 회복·해독·지혈·점혈·해혈·경맥 제어를 포함한다.
- 한 회차에 보유할 수 있는 보조무공은 최대 2개다.
- 한 회차에서 보조무공 10성은 최대 1개다.

## 2. 1~10성 성장 구조

| 성급 | 해금·강화 내용 |
|---:|---|
| 1성 | 무공서의 기본 성질과 기초 패시브 |
| 2성 | 기본 수치·조건·자원 효율 강화 |
| 3성 | **새로운 기술 1 해금** |
| 4성 | 무공서 공통 숙련·연계 강화 |
| 5성 | **3성 기술 1 강화 — 새 카드 없음** |
| 6성 | 자원 순환·조건·문파 기믹 강화 |
| 7성 | **새로운 기술 2 해금** |
| 8성 | 상위 숙련·연계·대응 강화 |
| 9성 | **7성 기술 2 강화 — 새 카드 없음** |
| 10성·대성 | 핵심무공은 **절초**, 보조무공은 **진의** 해금 |

### 카드 수 원칙

무공서 하나가 제공하는 사용 가능 카드는 총 3장이다.

1. 3성 기술 카드
2. 7성 기술 카드
3. 10성 절초 또는 진의 카드

5성과 9성은 기존 카드의 상세 효과·수치·조건을 강화하며 별도 카드를 추가하지 않는다.

## 5. 기술 설계 원칙

- 3성 기술은 해당 무공의 기본 운용법을 가르친다.
- 5성 강화는 3성 기술의 비용·수치·조건·연계 중 일부를 개선한다.
- 7성 기술은 상위 운용, 조건부 연계 또는 역할 확장을 제공한다.
- 9성 강화는 7성 기술의 성능과 안정성을 개선한다.
- 10성 절초·진의는 별도 카드이며 일반 기술의 단순 수치 강화가 아니다.
- 절초는 필살기 역할을 하며 10성 해금과 전투 중 절초 기세 조건을 모두 요구한다.
- 진의는 보조무공의 원리를 완성하고 자원·상태·연계 방식을 바꾸는 최종 기술이다.

## 6. 카드 표시 계약

- 카드 좌상단은 문파 문양 이미지다.
- 카드 우상단은 검법·권법·심법 같은 분류의 텍스트가 아니라 아이콘 이미지다.
- 사거리와 비용 숫자는 아트에 굽지 않고 UI·데이터 레이어에서 표시한다.
- 카드 하단은 비용·효과 영역이 아니라 무공의 묘리 한 소절 전용이다.
- 10성 핵심무공은 `절초`, 보조무공은 `진의` 세로 표찰을 사용한다.
- 상세 팝업은 이름, 대상, 효과, 특수효과, 비용, 설명·키워드를 표시한다.

## 7. 구현 경계

- 무공 효과는 비용, 합, 피해, 방어도, 사거리, 이동, 상태, 명상, 장전과 사용 횟수 수정자로 작성한다.
- 핵심무공 10성 제한과 보조무공 10성 제한은 서로 별개다.
- 이미 핵심무공 10성이 있으면 다른 핵심무공은 최대 9성이다.
- 이미 보조무공 10성이 있으면 다른 보조무공은 최대 9성이다.
- 이미지의 오탈자나 임시 분류가 이 문서의 데이터를 덮어쓰지 않는다.
- 현재 효과·비용·밸런스는 미확정이며 이름과 분류만 확정 상태다.
