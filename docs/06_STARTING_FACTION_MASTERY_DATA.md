# 10문파 핵심무공·보조무공·10성 데이터

> 모든 문파 효과는 공용 행동, 합 수치, 비용, 자원, 상태와 명상 수정자로 표현한다. 문파 전용 해상 순서나 전용 행동 슬롯을 만들지 않는다.

## 1. 무공 보유 구조

### 핵심무공

- 공식 핵심무공은 `검법·도법·권법·장법·창법·곤봉법·궁술·암기술` 8종이다.
- 핵심무공은 직접 공격, 합 판정, 피해, 밀치기, 장전과 사거리 운용을 담당한다.
- 한 회차에 여러 핵심무공을 익힐 수 있다.
- 각 핵심무공 종류는 문파 전용 1종 이상과 공용 무공 2종을 기본 콘텐츠 단위로 한다.
- 한 회차에서 핵심무공 10성은 최대 1개다.
- `부법`은 공식 성장 계통에서 제외한다. 도끼는 개별 무기·연출 태그로만 사용할 수 있다.
- `점혈술`은 핵심무공에서 제외한다. 점혈은 의술의 세부 기술·상태 태그다.

### 보조무공

- 공식 보조무공은 `심법·호신공·독공·마공·보법·방패술·외공·의술` 8종이다.
- 패시브, 문파 기믹 생성·강화, 명상 강화, 자원 전환, 이동, 방어, 상태 생성·해제를 담당한다.
- 신법과 경공은 보법의 세부 태그다.
- 의술은 회복·해독·지혈뿐 아니라 점혈·해혈·경맥 봉쇄를 포함한다.
- 한 회차에 보유할 수 있는 보조무공은 최대 2개다.
- 한 회차에서 보조무공 10성은 최대 1개다.
- 보조무공은 관련 핵심무공을 단방향으로 강화한다. 핵심무공 숙련도가 보조무공을 자동 강화하지는 않는다.

## 2. 1~10성 공통 구조

| 성급 | 핵심무공 | 보조무공 |
|---:|---|---|
| 1성 | 기본 패시브 | 기초 자원·방어·이동 패시브 |
| 2성 | 기본 수치·조건 강화 | 기초 패시브 강화 |
| 3성 | 기술 1 | 패시브, 기믹 생성 또는 기술 1 |
| 4성 | 기술 1 효율 강화 | 기믹·명상 효율 강화 |
| 5성 | 비용·합·사거리 강화 | 자원 순환·조건 강화 |
| 6성 | 기술 2 | 기술 2 또는 명상 강화 |
| 7성 | 연계·대응 효과 강화 | 기믹 유지·중첩 강화 |
| 8성 | 상위 효율 강화 | 주력 무공 연계 강화 |
| 9성 | 기술 3 | 기술 3 또는 상위 패시브 |
| 10성 | 절초 1개 해금 | 진의 1개 해금 |

- 시작 문파의 핵심무공과 선택한 보조무공은 각각 3성에서 시작한다.
- 성급 n에 도달하려면 n 수련 포인트가 필요하다.
- 핵심무공 3성에서 10성까지는 누적 49포인트가 필요하다.
- 보조무공 3성에서 8성까지는 누적 30포인트가 필요하다.

## 3. 10문파 전용 계통과 진의 방향

아래 이름은 현재 권장 기준이다. 카드 수치와 세부 효과는 구현 데이터 작성 단계에서 확정한다.

| 문파 | 핵심무공 | 보조무공 | 3성 핵심 기믹 | 10성 진의 방향 |
|---|---|---|---|---|
| 화산 | 매화검법 | 자하신공 | 검법 사용 후 다음 검법 비용을 줄이는 `연격` | 연격의 비용 감소·유지·연속 사용 보너스 강화 |
| 개방 | 항룡십팔장·타구봉법 | 취팔선보 | 장법·봉법의 밀치기와 변칙 이동으로 기세 획득 | 기세를 합·밀치기·거리 유지로 전환 |
| 당문 | 당문암기비전 | 오독심경·금침의경·귀영보 | 독·암기·의술 점혈·보법 재배치 연계 | 독과 점혈 상태를 암기 비용·장전·회복으로 전환 |
| 소림 | 나한권·달마봉법 | 금강불괴신공 | 거리 0 압박과 공격·곤봉 사용 중 피해 경감 | 합 패배 차이 경감, 호신 유지, 제압 강화 |
| 무당 | 태극검법·태극권 | 순양무극공·제운종 | 방어·회피 뒤 합 반격과 내공 전환 | 상대 공격 수치를 합 보너스로 전환하고 보법 회피 강화 |
| 마교 | 혈염도법 | 역혈마공 | 체력을 대가로 공격과 마기 강화 | 체력 소모를 비용·합 강화와 승리 회복으로 전환 |
| 양가장 | 양가창법 | 양문철벽패술 | 거리 유지·창 회수·막기 후 창 반격 | 방패 경감 일부 유지, 다음 창법 합·밀치기 강화 |
| 백우궁 | 백우궁술 | 유운보법 | 이동과 사격으로 장전 공백 관리 | 이동→사격·사격→이동 시 재장전 부담 감소 |
| 사파 | 흑풍도법·추혼암기술 | 철골외공 | 변칙 도법과 암습을 외공으로 버티며 난전 가속 | 피해·경직을 다음 도법·암기 합과 비용 감소로 전환 |
| 녹림 | 녹림창법·산림궁술 | 산림잠행보 | 창으로 길목을 막고 궁으로 매복 사격 | 지형 이동과 거리 유지로 창·궁 전환 부담 감소 |

문파가 제공하는 보조무공 수와 실제 동시 보유 한도는 구분한다. 당문은 독공·의술·보법 세 계통을 제공하지만 한 회차에는 최대 2개만 보유할 수 있다.

## 4. 보조무공 유형별 역할

| 보조무공 | 시스템 역할 |
|---|---|
| 심법 | 내공·명상·자원 전환과 주력 무공 강화 |
| 호신공 | 피해 경감·막기·합 패배 차이 완화 |
| 독공 | 독 생성·누적·확산·소모 |
| 마공 | 체력·위험을 대가로 공격과 자원 강화 |
| 보법 | 이동·회피·거리 조절·장전 공백 보완 |
| 방패술 | 막기·전진·밀치기·반격 준비 |
| 외공 | 체력·경직 저항·육체 강화·난전 지속력 |
| 의술 | 회복·해독·지혈·점혈·해혈·경맥 상태 제어 |

## 5. 화산 연격 기준

### 자하신공 3성 — 연격 생성

- 검법을 사용하면 다음 검법에 적용되는 연격 상태를 1회 얻는다.
- 연격은 다음 검법의 행동력 또는 기력 비용을 감소시킨다.
- 공용 검법에도 적용하지만 다른 무기 계통에는 적용하지 않는다.
- 비용은 최소 1 아래로 내려가지 않는다.
- 기본 지속은 다음 검법 사용 또는 현재 2수 묶음 종료까지다.

### 중간 성급 강화

- 4~5성: 비용 감소량 또는 적용 가능한 검법 범위 강화
- 6성: 명상 효과를 강화하고 명상으로 연격을 유지·재개하는 기술 제공
- 7~8성: 같은 2수 묶음의 연속 검법 보너스 강화
- 9성: 상위 연격 기술 제공

### 보조무공 10성 — 진의

- 연격 저장·지속 또는 연속 사용 보너스를 강화한다.
- 특정 연격 횟수에서 다음 검법의 합 수치를 높일 수 있다.
- 무제한 무료 연격을 막기 위해 최대 중첩, 발동 횟수, 최소 비용을 데이터로 제한한다.

## 6. 공용 무공 기준

각 핵심무공 종류는 문파 전용 무공 외에 공용 무공 2종을 제공한다.

| 공용 방향 A | 공용 방향 B |
|---|---|
| 낮은 비용·안정적인 기본기 | 높은 비용·강한 합 또는 조건부 효과 |
| 연계 시작에 적합 | 연계 마무리에 적합 |
| 문파 기믹을 생성하지 않음 | 문파 기믹을 생성하지 않음 |

공용 무공은 문파 정체성을 대신하지 않는다. 문파 보조무공이 생성한 연격·반격·독기·기세 같은 공용 상태 수정자의 혜택을 받을 수는 있다.

## 7. 명칭 및 구현 경계

- 일반 마교 교도 무공에는 `천마` 명칭을 사용하지 않는다. 천마 명칭은 천마 개인 또는 특수 계승 콘텐츠로 분리한다.
- 사파 전용 계통에서는 검법·장법·보법을 사용하지 않는다.
- 점혈은 의술의 기술·상태 태그이며 독립 핵심무공이 아니다.
- 카드 이미지의 사거리와 비용 숫자는 비워 두고 데이터·UI 레이어에서 편집한다.
- 모든 성급 효과는 비용, 합 수치, 일반 피해, 피해 차이 경감, 사거리, 이동량, 상태, 명상 회복, 장전, 사용 횟수 수정자로 작성한다.
- 핵심무공 10성 제한과 보조무공 10성 제한은 서로 별개다.
- 이미 핵심무공 10성이 있으면 다른 핵심무공은 최대 9성이다.
- 이미 보조무공 10성이 있으면 다른 보조무공은 최대 9성이다.
- 깨달음은 상승 가능한 무공 중 현재 성급이 가장 높은 무공을 1단계 올리며, 위 10성 제한을 넘지 않는다.
- 문파별 기믹 수행도를 성과 등급에 반영하는 규칙은 현재 보류한다.
