# 12세력 핵심무공·심법·10성 데이터

> 모든 문파 효과는 공용 행동, 합 수치, 비용, 자원, 상태와 명상 수정자로 표현한다. 문파 전용 해상 순서나 전용 행동 슬롯을 만들지 않는다.

## 1. 무공 보유 구조

### 핵심무공

- 검법·도법·권법·장법·창법·곤법·궁술·암기술 등 직접 전투 계통이다.
- 한 회차에 여러 핵심무공을 익힐 수 있다.
- 각 무공 종류는 문파 전용 1종과 공용 무공 2종을 기본 콘텐츠 단위로 한다.
- 한 회차에서 핵심무공 10성은 최대 1개다.

### 심법

- 패시브, 문파 기믹 생성·강화, 명상 강화, 자원 전환을 담당한다.
- 한 회차에 보유할 수 있는 심법은 최대 2개다.
- 한 회차에서 심법 10성은 최대 1개다.
- 심법은 관련 핵심무공을 단방향으로 강화한다. 핵심무공 숙련도가 심법을 자동 강화하지는 않는다.

## 2. 1~10성 공통 구조

| 성급 | 핵심무공 | 심법 |
|---:|---|---|
| 1성 | 기본 패시브 | 기초 자원·방어 패시브 |
| 2성 | 기본 수치·조건 강화 | 기초 패시브 강화 |
| 3성 | 기술 1 | 패시브, 기믹 생성 또는 기술 1 |
| 4성 | 기술 1 효율 강화 | 기믹·명상 효율 강화 |
| 5성 | 비용·합·사거리 강화 | 자원 순환·조건 강화 |
| 6성 | 기술 2 | 기술 2 또는 명상 강화 |
| 7성 | 연계·대응 효과 강화 | 기믹 유지·중첩 강화 |
| 8성 | 상위 효율 강화 | 메인 무공 연계 강화 |
| 9성 | 기술 3 | 기술 3 또는 상위 패시브 |
| 10성 | 절초 1개 해금 | 진의 1개 해금 |

- 시작 문파의 핵심무공과 심법은 각각 3성에서 시작한다.
- 성급 n에 도달하려면 n 수련 포인트가 필요하다.
- 핵심무공 3성에서 10성까지는 누적 49포인트가 필요하다.
- 심법 3성에서 8성까지는 누적 30포인트가 필요하다.

## 3. 문파별 전용 계통과 진의 방향

아래 이름과 세부 수치는 구현 데이터 작성 시 확정한다. 이 표는 문파의 역할과 허용된 수정 방향을 고정한다.

| 세력 | 핵심무공 | 심법 | 3성 핵심 기믹 | 10성 진의 방향 |
|---|---|---|---|---|
| 화산 | 검법 | 심법 | 검법 사용 후 다음 검법 비용을 줄이는 `연격` | 연격의 비용 감소·유지·연속 사용 보너스 강화 |
| 개방 | 곤법 | 심법·호신 계통 | 밀치기·막기 성공으로 기세 획득 | 기세를 다음 곤법의 합·밀치기·거리 유지로 전환 |
| 당문 | 암기술 | 독공 | 독과 재장전·암기 연계 | 독이 있는 적에게 다음 암기 비용·장전 부담 감소 |
| 소림 | 권법 | 호신심법 | 거리 0 압박과 공격 중 피해 경감 | 연속 권법 중 합 패배 차이 경감과 호신 유지 |
| 무당 | 태극권·검법 | 내공심법 | 방어·회피 뒤 태극권 합 반격 | 상대 공격 수치 일부를 합 보너스로 전환하는 반격 강화 |
| 마교 | 도법 | 마공심법 | 체력 대가로 공격 강화 | 체력 소모를 다음 도법 비용·합 강화와 승리 회복으로 전환 |
| 양가장 | 창법 | 심법 | 거리 2 유지와 창 회수 | 연속 창법의 회수 부담·추적·합 강화 |
| 백우궁 | 궁술 | 심법·보법 계통 | 이동과 사격으로 장전 공백 관리 | 이동→사격·사격→이동 시 재장전 부담 감소 |
| 관군 | 창법·방패술 | 군진심법 | 막기 후 창 반격과 방패 전진 | 막기 경감 일부 유지, 다음 창법 합·밀치기 강화 |
| 아미 | 점혈·검법 | 의술심법 | 적 기력·내공 압박과 회복 | 자원 감소를 자신의 회복·다음 점혈 비용 감소로 전환 |
| 오독 | 독장 | 독맥심법 | 거리 0 독 누적 | 독 피해를 내공 회복·합 보너스·차이 피해 경감으로 전환 |
| 녹림채 | 도법 | 외공심법 | 기력 탈취와 난전 가속 | 탈취 기력을 다음 도법 비용 감소·합 보너스로 전환 |

## 4. 화산 연격 기준

### 심법 3성 — 연격 생성

- 검법을 사용하면 다음 검법에 적용되는 연격 상태를 1회 얻는다.
- 연격은 다음 검법의 행동력 또는 기력 비용을 감소시킨다.
- 공용 검법에도 적용하지만 다른 무기 계통에는 적용하지 않는다.
- 비용은 최소 1 아래로 내려가지 않는다.
- 기본 지속은 다음 검법 사용 또는 현재 2수 묶음 종료까지다.

### 중간 성급 강화

- 4~5성: 비용 감소량 또는 적용 가능한 검법 범위 강화
- 6성: 명상 효과를 강화하고 명상으로 연격을 유지·재개하는 기술 제공
- 7~8성: 같은 2수 묶음의 연속 검법 보너스 강화
- 9성: 상위 연격 기술 제공

### 심법 10성 — 진의

- 연격 저장·지속 또는 연속 사용 보너스를 강화한다.
- 특정 연격 횟수에서 다음 검법의 합 수치를 높일 수 있다.
- 무제한 무료 연격을 막기 위해 최대 중첩, 발동 횟수, 최소 비용을 데이터로 제한한다.

## 5. 공용 무공 기준

각 핵심무공 종류는 문파 전용 무공 외에 공용 무공 2종을 제공한다.

| 공용 방향 A | 공용 방향 B |
|---|---|
| 낮은 비용·안정적인 기본기 | 높은 비용·강한 합 또는 조건부 효과 |
| 연계 시작에 적합 | 연계 마무리에 적합 |
| 문파 기믹을 생성하지 않음 | 문파 기믹을 생성하지 않음 |

공용 무공은 문파 정체성을 대신하지 않는다. 문파 심법이 생성한 연격·반격·독기·기세 같은 공용 상태 수정자의 혜택을 받을 수는 있다.

## 6. 구현 경계

- 모든 성급 효과는 비용, 합 수치, 일반 피해, 피해 차이 경감, 사거리, 이동량, 상태, 명상 회복, 장전, 사용 횟수 수정자로 작성한다.
- 핵심무공 10성 제한과 심법 10성 제한은 서로 별개다.
- 이미 핵심무공 10성이 있으면 다른 핵심무공은 최대 9성이다.
- 이미 심법 10성이 있으면 다른 심법은 최대 9성이다.
- 깨달음은 상승 가능한 무공 중 현재 성급이 가장 높은 무공을 1단계 올리며, 위 10성 제한을 넘지 않는다.
- 문파별 기믹 수행도를 성과 등급에 반영하는 규칙은 현재 보류한다.
