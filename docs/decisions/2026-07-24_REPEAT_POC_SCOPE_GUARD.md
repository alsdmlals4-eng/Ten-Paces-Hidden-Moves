# REPEAT_POC 범위 가드

## 이번 Goal에 포함

- 활성 정본의 기준 SHA·AI source 정렬.
- board schema 17 전환.
- 라이벌 1명·복수 합리 후보·seed 재현.
- 플레이어 가설 6개 선택지.
- 결정적 원인 summary와 최소 복기 UI.
- 키보드·모션 감소 정보 보존.
- 고정 SHA 신규 플레이어 5명 STEP 14.

## 이번 Goal에서 제외

- 새 기초 행동 또는 절초.
- 합·피해·방어·필중·중단·강건 수치 변경.
- 12세력·10성·10전 캠페인.
- 무공·심법·성장·보상 경제.
- 저장·불러오기·영구 진행.
- 다중 라이벌 제작.
- 난이도 모드.
- 머신러닝·플레이어 모델링.
- 승률·예측률·정답 행동 추천.
- 대규모 UI 아트 교체.

## 변경 중단 신호

- 변경 파일이 위 포함 범위를 넘어간다.
- `CombatResolutionEngine` 판정 공식이 바뀐다.
- 복기 panel이 상세 로그와 별도 판정을 계산한다.
- AI가 `combat_state` 외 planning UI 객체를 참조한다.
- 후보 다양화를 위해 근거 없는 완전 랜덤을 추가한다.
- 신규 플레이어 검증 전에 콘텐츠 수량을 늘린다.

중단 신호가 나타나면 구현을 계속하지 않고 finding·영향·축소안을 보고한다.
