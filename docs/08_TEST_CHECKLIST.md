# 12세력 대회 세로 슬라이스 테스트 체크리스트

> 초점: 어떤 시작 문파를 골라도 공용 2수 선택·합·수련·행운·상대 읽기·제약 위에서 다른 전략을 만들고 데모 5전과 전체판 10전을 완주할 수 있는가.

## 1. 공용 전장·2수

- [ ] 모든 결투가 A=4, B=6, 초기 거리 2에서 시작한다.
- [ ] 10칸 전장에 같은 칸 중첩, 교차, 자리 교환이 가능하다.
- [ ] 한 라운드는 행동 타이밍 10개이고 2타이밍 묶음을 5회 선택한다.
- [ ] AI는 각 묶음 시작 시 두 행동을 플레이어 선택 전에 잠근다.
- [ ] 양측 두 행동을 한꺼번에 공개하고 1수→2수 순으로 해상한다.
- [ ] 첫 결과를 본 뒤 둘째 행동을 수정할 수 없다.
- [ ] 접근·후퇴·공격·막기·회피·밀치기·교체·재장전·명상이 공통으로 작동한다.

## 2. 합

- [ ] 같은 타이밍에 양측 공격이 모두 유효하면 합이 발생한다.
- [ ] 낮은 최종 합 수치 쪽이 차이만큼 피해를 받는다.
- [ ] 합 동률이면 체력 피해가 없다.
- [ ] 한쪽 공격이 무효면 합이 아니라 유효한 공격만 일반 판정한다.
- [ ] 공격 대 막기·회피는 일반 공격 대 방어로 처리한다.
- [ ] 태극권 반격 수정자는 상대 공격이 있을 때만 적용된다.
- [ ] 연출 횟수와 실제 계산 횟수가 분리되어 있다.
- [ ] 로그가 기본 합→무기→성급→심법·기믹→반격→최종 합→차이 피해 순으로 설명한다.

## 3. 무기·장비·상태

- [ ] 권법 0, 검·도 1, 창·곤 2, 암기 2~4, 활 3~5 사거리가 데이터와 UI에 일치한다.
- [ ] 방패는 활 피해 막기를 높이고 회피 효율을 낮춘다.
- [ ] 밀치기는 해당 무기 사거리를 따른다.
- [ ] 무기 교체는 타이밍과 행동력을 사용한다.
- [ ] 암기벨트와 화살통이 제한 횟수 공용 장비로 작동한다.
- [ ] 독·마화·장전·명상 회복이 정확한 타이밍에 처리된다.

## 4. 핵심무공·심법

- [ ] 핵심무공은 여러 개 보유할 수 있다.
- [ ] 심법 세 번째 습득이 거부되고 최대 2개가 유지된다.
- [ ] 각 무공 종류에 문파 전용 1종과 공용 2종이 존재한다.
- [ ] 공용 무공은 문파 기믹을 생성하지 않지만 공용 수정자의 혜택을 받을 수 있다.
- [ ] 시작 핵심무공과 심법은 각각 3성이다.
- [ ] 핵심무공 3·6·9성 기술과 10성 절초가 해금된다.
- [ ] 심법 3·6·9성 효과와 10성 진의가 해금된다.
- [ ] 핵심무공 10성 최대 1개, 심법 10성 최대 1개가 유지된다.
- [ ] 화산 심법 3성 연격과 10성 진의 강화가 공용 수정자로 작동한다.

## 5. 성과·행운·수련

### 기본 정산

- [ ] 승리 기본 수련 포인트가 5다.
- [ ] 등급 보너스가 C +1, B +3, A +5, S +7이다.
- [ ] A가 의도한 평균 성과가 되도록 공용 평가 임계값을 조정할 수 있다.
- [ ] 문파별 기믹 수행도는 성과 점수에 포함되지 않는다.
- [ ] 성급 `n` 상승 비용이 `n`포인트다.
- [ ] 핵심무공 3성→10성이 49포인트다.
- [ ] 심법 3성→8성이 30포인트다.

### 행운 결과표

- [ ] 행운은 3개 다이스의 개별 독립 판정이 아니라 등급별 앞면 수 결과표에서 추첨한다.
- [ ] C 결과 분포가 0/0/5/95%다.
- [ ] B 결과 분포가 5/5/5/85%다.
- [ ] A 결과 분포가 20/5/20/55%다.
- [ ] S 결과 분포가 60/10/15/15%다.
- [ ] 개별 다이스 면은 UI에 공개되지 않는다.
- [ ] 앞면 수가 `등급 + 0~3개의 +`로 표시된다.
- [ ] B++가 기본 5 + 등급 3 + 행운 2 = 총 10점으로 정산된다.
- [ ] 저장·불러오기 뒤 이미 확정된 플레이어 행운 결과가 바뀌지 않는다.

### 데모 절초

- [ ] 1~4전 포인트를 핵심무공에 전부 투자할 수 있다.
- [ ] A 4회 절초 개방률이 약 50% 목표에 가깝다.
- [ ] A 3회+S 1회가 약 60% 목표에 가깝다.
- [ ] A 3회+B 1회가 약 30% 목표에 가깝다.
- [ ] 절초를 개방한 회차는 5전에서 사용할 수 있다.
- [ ] 절초가 없어도 5전 예선 결승을 완료할 수 있다.

## 6. 상대 생성과 성장

- [ ] 상대 등급 추첨의 기준 분포가 A계열 60%, B계열 40%다.
- [ ] 상대도 등급별 행운 결과표에서 `+`를 추첨한다.
- [ ] 상대 경기당 수련이 기본 5 + 등급 보너스 + 행운으로 계산된다.
- [ ] 등장 경기 전까지의 누적 포인트가 실제 무공·심법에 투자된다.
- [ ] 균형형이 핵심무공과 심법을 고르게 투자한다.
- [ ] 핵심무공 올인형이 6·9·10성 경계를 우선한다.
- [ ] 심법 연계형이 기믹·명상·자원 순환 경계를 우선한다.
- [ ] 다무공형이 주력·심법·보조 무공에 분산한다.
- [ ] 상대의 등급, 행운, 스타일과 실제 투자가 회차 생성 시 고정된다.
- [ ] 재도전에서 상대 성장 결과가 다시 추첨되지 않는다.
- [ ] 저장·불러오기 뒤 같은 상대 프로필이 유지된다.

## 7. 무림식 상대 정보

- [ ] `[강적] [A++] [핵심무공 올인형]` 같은 개발용 태그가 플레이어 화면에 노출되지 않는다.
- [ ] 이명·수식언이 내부 종합 강도와 대표 전법을 암시한다.
- [ ] 풍문·전적이 행운과 성장세를 암시한다.
- [ ] 무공 평가 문장이 수련 스타일을 암시한다.
- [ ] 같은 내부 의미에 같은 어휘 계열이 반복 사용된다.
- [ ] 표시 문구와 실제 상대 데이터가 모순되지 않는다.
- [ ] 정탐 성공 시 숫자표 대신 더 구체적인 무림식 문장이 제공된다.
- [ ] 상대 정보를 읽은 뒤 제약을 선택할 수 있다.

## 8. 12세력 기믹

- [ ] 화산: 연격과 검법 연속 사용.
- [ ] 개방: 곤 거리 2, 밀치기·막기 기세.
- [ ] 당문: 암기 재장전, 독, 암기벨트.
- [ ] 소림: 권법 거리 0, 호신과 압박.
- [ ] 무당: 태극권 합 반격과 내공 운용.
- [ ] 마교: 체력 대가와 마화.
- [ ] 양가장: 창 거리 2, 밀치기·회수.
- [ ] 백우궁: 활 거리 3~5, 이동·사격·재장전.
- [ ] 관군: 방패 활 상성, 막기 뒤 창 반격.
- [ ] 아미: 점혈 자원 압박과 의술 회복.
- [ ] 오독: 거리 0 독 누적과 독맥 순환.
- [ ] 녹림채: 기력 탈취와 난전 강화.

## 9. 제약

- [ ] 전투 전 제약 후보와 불이익·위험도·승리 보상을 확인할 수 있다.
- [ ] 삼수양보는 첫 3타이밍 동안 공격·밀치기를 막는다.
- [ ] 단전봉맥·기혈역행·일장지양 등이 설명대로 작동한다.
- [ ] 문파 핵심무공 전체를 사용 불가로 만드는 제약은 생성되지 않는다.
- [ ] 제약 패배 시 추가 보상을 받지 않는다.
- [ ] 제약 없이도 일반 성장 목표가 가능하다.

## 10. 대진·저장·재도전

- [ ] 데모가 1~2전 내부전, 3~4전 예선, 5전 예선 결승으로 진행된다.
- [ ] 전체판 8전은 8강, 9전은 준결승, 10전은 결승이다.
- [ ] 같은 상대가 풀 소진 전 반복되지 않는다.
- [ ] 재도전 1회와 재기 보상이 작동한다.
- [ ] 저장 데이터에 상대 프로필·이명 키·등급·행운·수련 스타일·실제 투자 결과가 포함된다.
- [ ] 데모 종료 상태와 전체판 진행 상태를 구분한다.
- [ ] 저장·불러오기 뒤 AI 잠금 공정성과 회차 상태가 유지된다.

## 11. 보류 경계

- [ ] 문파별 기믹 수행도 성과 평가는 구현되지 않는다.
- [ ] `docs/[보류]/` 항목은 사용자 재개 전 구현되지 않는다.

## 12. 빌드 승인

- [ ] Godot headless 테스트에서 거리·2수·합·상태·수련·행운·상대 생성·저장이 통과한다.
- [ ] Windows 데모 빌드에서 새 회차부터 5전 예선 결승 결과까지 실행된다.
- [ ] Windows 전체판 빌드에서 10전 결승 결과까지 실행된다.
- [ ] 플레이어가 상대 이명·풍문을 근거로 제약 선택 이유를 설명할 수 있다.
- [ ] 플레이어가 다른 문파·무공·심법·수련 스타일에 대응해 다시 시작할 이유를 말할 수 있다.
