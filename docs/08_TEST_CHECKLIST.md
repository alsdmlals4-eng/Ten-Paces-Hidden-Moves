# 소림 대 당문 전투 PoC 테스트 체크리스트

> 기록 방식: 각 항목에 통과·실패, 재현 절차, 실제 결과, 수정 커밋을 남긴다. 이 체크리스트는 Codex Plan의 승인 기준이다.

## A. 초기 배치와 거리

- [ ] 전장은 정확히 10칸이다.
- [ ] 시작 배치가 `빈칸 3 / A 소림 / 빈칸 1 / B 당문 / 빈칸 4`으로 표시된다.
- [ ] A는 4번 칸, B는 6번 칸, 초기 거리는 2다.
- [ ] 같은 칸 위치와 반대 이동 상호 통과가 오류 없이 처리된다.
- [ ] 전장 밖 이동은 배치 단계에서 비활성화된다.

## B. 공용 행동

- [ ] 소림과 당문 모두 접근·후퇴·막기·회피·밀치기·대기를 본다.
- [ ] 접근과 후퇴는 각각 상대 방향·반대 방향으로 1칸 이동한다.
- [ ] 회피는 이동 뒤에도 사거리 안이면 피해를 50% 추가 경감한다.
- [ ] 밀치기는 현재 무기 사거리를 사용하고, 전장 끝에서는 대상 기력 2를 줄인다.
- [ ] 미장전 당문은 공격·밀치기를 배치할 수 없고 재장전만 가능하다.
- [ ] 공용 행동의 비용을 초과하면 행동열을 잠글 수 없다.

## C. 무기와 유파 규칙

- [ ] 소림 권법 공격과 밀치기는 거리 0에서만 적중한다.
- [ ] 소림 막기는 기본 막기 40%가 아니라 금강호신 적용 55%로 경감한다.
- [ ] 당문 암기는 거리 2~4에서만 적중한다.
- [ ] 당문 암기 공격 뒤에는 미장전 상태가 된다.
- [ ] 당문 암기 적중은 독 1중첩을 부여하고, 라운드 종료 시 중첩당 피해 2를 준다.
- [ ] 유파 규칙이 새로운 행동 버튼을 만들지 않고 기존 공용 행동의 결과만 바꾼다.

## D. 암기벨트

- [ ] 당문은 전투 시작 시 암기벨트 3회를 보유한다.
- [ ] 암기벨트는 기존 행동 슬롯에 붙이며 행동력·시간 슬롯을 소모하지 않는다.
- [ ] 한 슬롯에 즉발 암기는 1회만 붙일 수 있다.
- [ ] 즉발 암기는 거리 2~4에서 피해 3과 독 1중첩을 준다.
- [ ] 남은 횟수는 공개되고 사용 슬롯은 공개 전까지 숨겨진다.
- [ ] 3회 모두 사용하면 추가 부착이 불가능하다.

## E. 동시 선택과 AI

- [ ] AI는 플레이어 잠금 전에 행동열을 생성·잠근다.
- [ ] 플레이어 잠금 뒤 AI 행동열 데이터가 변경되지 않는다.
- [ ] 이동 후 실제 거리로 양측 공격·밀치기를 판정한다.
- [ ] 같은 슬롯의 양측 피해와 상태이상은 동시에 적용된다.
- [ ] 결과 해설이 이동, 최종 거리, 적중·실패, 독 정산을 모두 설명한다.

## F. 재미 검증 수동 테스트

- [ ] 플레이어가 소림으로 패배한 뒤 ‘다음에는 이 슬롯에서 접근 또는 막기를 바꾸겠다’고 말한다.
- [ ] 플레이어가 당문으로 패배한 뒤 ‘다음에는 재장전·후퇴·암기벨트 순서를 바꾸겠다’고 말한다.
- [ ] 같은 상대와 재대결해도 AI 행동열이 달라 정답 암기가 되지 않는다.
- [ ] 테스트 참가자가 ‘소림 대 당문 한 판을 다시 해 보고 싶다’고 평가한다.
