# 전투 PoC 테스트 체크리스트

> 사용법: 구현자는 항목마다 실제 결과를 기록하고, 실패 항목은 원인·재현 절차·수정 커밋을 남긴다. 자동 테스트가 어려운 UI 항목은 수동 테스트 영상 또는 스크린샷으로 확인한다.

## A. 전장과 거리

- [ ] 시작 배치가 `빈칸 3 / A / 빈칸 2 / B / 빈칸 3`으로 표시된다.
- [ ] A가 4번 칸, B가 7번 칸에 배치되고 초기 거리가 3으로 계산된다.
- [ ] A와 B가 같은 칸에 동시에 도착해도 오류 없이 거리 0이 된다.
- [ ] 양측이 반대 방향으로 이동해 경로가 겹치면 위치를 교환한다.
- [ ] 전장 끝에서 바깥으로 이동하는 행동은 잠금 전에 비활성화된다.
- [ ] 거리 0·1·2·3·5에서 각 무기 사거리 표시와 실제 판정이 일치한다.

## B. 행동열과 자원

- [ ] 양측이 정확히 6시간 슬롯을 가진다.
- [ ] 행동력 6을 넘는 배치는 잠글 수 없다.
- [ ] 기력·내공이 부족한 행동은 배치할 수 없고 부족 수치가 표시된다.
- [ ] 대기 슬롯은 행동력 0으로 배치되고 기력을 1 회복한다.
- [ ] 라운드 종료 시 기력 2, 내공 1이 최대치를 넘지 않고 회복된다.
- [ ] 창 찌르기 뒤 다음 슬롯에 창 회수가 자동 표시되고 다른 행동으로 바꿀 수 없다.
- [ ] 궁수는 미장전 상태에서 사격을 배치할 수 없고 장전 후에만 사격 가능하다.

## C. 이동 후 판정

- [ ] 공격 사거리는 슬롯 시작 위치가 아니라 이동 뒤 실제 거리로 계산된다.
- [ ] A=4, B=7에서 B가 왼쪽 보법을 쓰고 A가 창 찌르기를 쓰면 거리 2에서 적중한다.
- [ ] A=4, B=7에서 A가 경공으로 6번 칸에 이동해도 창 찌르기는 거리 1에서 빗나간다.
- [ ] 같은 슬롯의 양측 공격이 사거리 안이면 양쪽 피해가 함께 적용된다.
- [ ] 막기는 같은 슬롯의 직접 피해를 40% 경감한다.
- [ ] 회피보는 이동 뒤에도 사거리 안이면 피해를 추가 50% 경감한다.

## D. 무기 정체성

- [ ] 권법은 거리 0에서만 피해를 주고, 거리 1 이상에서는 빗나간다.
- [ ] 연환권은 같은 라운드에 첫 권격 이후에만 배치 가능하다.
- [ ] 창 찌르기는 거리 2에서만 피해 16을 주고 회수 슬롯을 만든다.
- [ ] 활 사격은 거리 3~5에서만 피해 14를 주고 미장전 상태가 된다.
- [ ] 궁수 장전은 슬롯 1칸과 행동력 1, 기력 1을 소모한다.
- [ ] 권법·창법·궁술이 같은 초기 자원에서 서로 다른 6슬롯 행동열을 만든다.

## E. 밀치기

- [ ] 공통 밀치기는 현재 무기 사거리 안에서만 적중한다.
- [ ] 적중한 밀치기는 대상의 반대 방향 1칸 이동을 만든다.
- [ ] 전장 끝의 밀치기는 이동 대신 대상 기력 2 감소를 만든다.
- [ ] 거리 0 밀치기는 왼쪽·오른쪽 방향 선택 없이는 잠글 수 없다.
- [ ] 양측의 같은 슬롯 밀치기는 우선권 없이 동시에 처리된다.

## F. AI 공정성과 결과 해설

- [ ] AI는 플레이어 행동열 잠금 전에 자신의 행동열을 생성·잠근다.
- [ ] 플레이어 잠금 뒤 AI의 행동열 데이터가 바뀌지 않는다.
- [ ] AI 난이도를 바꿔도 AI가 플레이어의 비공개 행동열을 읽지 않는다.
- [ ] 동시 공개 전 AI 슬롯은 빈칸 또는 뒷면으로만 보인다.
- [ ] 실행 뒤 모든 적중·실패에 최종 거리와 원인이 기록된다.
- [ ] 재현 시드가 같으면 AI 행동열과 실행 결과가 동일하다.

## G. 사용자 경험 수동 점검

- [ ] 처음 보는 사람이 10칸 전장의 현재 거리와 누가 유리한지 설명할 수 있다.
- [ ] 처음 보는 사람이 활의 장전 공백과 창의 회수 공백을 설명할 수 있다.
- [ ] 처음 보는 사람이 AI가 치팅하지 않았다고 판단할 수 있다.
- [ ] 한 라운드 종료 뒤 ‘다음 라운드에는 무엇을 바꿀지’ 선택 이유를 말할 수 있다.
