# 12세력 대회 세로 슬라이스 테스트 체크리스트

> 테스트의 초점은 어떤 시작 문파를 골라도 공용 2수 선택·합·수련·제약 규칙 위에서 다른 전략을 만들고, 데모 5전과 전체판 10전 회차를 완주할 수 있는지다.

## 1. 공용 전장·2수 동시 선택

- [ ] 모든 결투가 A=4, B=6, 초기 거리 2에서 시작한다.
- [ ] 10칸 전장에 같은 칸 중첩, 교차, 자리 교환이 가능하다.
- [ ] 한 라운드는 행동 타이밍 10개이고 2타이밍 묶음을 5회 선택한다.
- [ ] AI는 각 묶음 시작 시 두 행동을 플레이어 선택 전에 잠근다.
- [ ] 양측 두 행동을 한꺼번에 공개하고 1수→2수 순으로 해상한다.
- [ ] 첫 번째 결과를 본 뒤 두 번째 행동을 수정할 수 없다.
- [ ] 접근·후퇴·공격·막기·회피·밀치기·교체·재장전·명상이 공통으로 작동한다.
- [ ] 이동 후 최종 위치를 기준으로 공격·밀치기 사거리를 판정한다.

## 2. 합 판정

- [ ] 같은 타이밍에 양측 공격이 모두 유효하면 합이 발생한다.
- [ ] 높은 최종 합 수치와 낮은 최종 합 수치의 차이만큼 낮은 쪽이 피해를 받는다.
- [ ] 합 동률이면 양측 체력 피해가 없다.
- [ ] 한쪽 공격이 무효면 합이 아니라 유효한 공격만 일반 판정한다.
- [ ] 공격 대 막기·회피는 합이 아닌 일반 공격 대 방어로 처리한다.
- [ ] 태극권 반격 수정자가 상대 공격이 있을 때만 적용된다.
- [ ] 합 연출 횟수와 실제 계산 횟수가 분리되어 있다.
- [ ] 로그가 기본 합→무기→성급→심법·기믹→최종 합→차이 피해 순으로 설명한다.

## 3. 무기·보조장비·상태

- [ ] 권법 0, 검·도 1, 창·곤 2, 암기 2~4, 활 3~5 사거리가 데이터와 UI에 일치한다.
- [ ] 일반 막기는 가능하고 방패는 활 피해 막기를 높이며 회피 효율을 낮춘다.
- [ ] 밀치기는 모든 무기가 쓸 수 있고 해당 무기 사거리를 따른다.
- [ ] 무기 교체는 타이밍과 행동력을 사용한다.
- [ ] 암기벨트는 제한 횟수 내 기존 행동에 부착된다.
- [ ] 화살통은 제한 횟수 내 재장전 행동을 면제한다.
- [ ] 독·마화·장전·명상 회복이 정확한 타이밍에 처리된다.

## 4. 핵심무공·심법·공용 무공

- [ ] 핵심무공은 여러 개 보유할 수 있다.
- [ ] 심법은 세 번째 습득을 시도하면 거부되고 보유 최대 2개가 유지된다.
- [ ] 각 구현 무공 종류에 문파 전용 1종과 공용 무공 2종이 존재한다.
- [ ] 공용 무공은 문파 기믹을 새로 생성하지 않지만 공용 수정자의 혜택을 받을 수 있다.
- [ ] 시작 문파 핵심무공과 심법은 각각 3성이다.
- [ ] 핵심무공 3·6·9성에 기술이 해금되고 10성에 절초가 해금된다.
- [ ] 심법 3·6·9성에 패시브·기믹·명상 효과가 해금되고 10성에 진의가 해금된다.
- [ ] 핵심무공 10성은 최대 1개, 심법 10성은 최대 1개다.
- [ ] 이미 10성이 있는 계열의 다른 9성 무공은 10성 투자가 잠긴다.

## 5. 12세력 기믹

각 항목은 AI 또는 플레이어로 전투 종료까지 한 번 이상 검증한다.

- [ ] 화산: 심법 3성 연격, 검법 연속 사용 비용 감소, 진의 강화
- [ ] 개방: 곤 거리 2, 밀치기·막기 기세
- [ ] 당문: 암기 재장전, 독, 암기벨트
- [ ] 소림: 권법 거리 0, 호신과 압박
- [ ] 무당: 태극권 합 반격과 내공 운용
- [ ] 마교: 체력 대가와 마화
- [ ] 양가장: 창 거리 2, 밀치기·회수
- [ ] 백우궁: 활 거리 3~5, 이동·사격·재장전
- [ ] 관군: 방패 활 상성, 막기 뒤 창 반격
- [ ] 아미: 점혈 자원 압박과 의술 회복
- [ ] 오독: 거리 0 독 누적과 독맥 순환
- [ ] 녹림채: 도 공격의 기력 탈취와 난전 강화

## 6. 성과·수련·절초 개방

- [ ] 승리 기본 수련 포인트가 5다.
- [ ] C등급이 6~7점, B등급이 8~10점을 지급한다.
- [ ] A등급이 확정 보너스 +8과 50% 확률 +1로 총 13~14점을 지급한다.
- [ ] S등급이 확정 보너스 +9와 확정 확률 보너스 +2로 총 16점을 지급한다.
- [ ] 확률 보너스 성공·실패가 별도 결과로 기록된다.
- [ ] 성급 n의 상승 비용이 n포인트다.
- [ ] 3성 핵심무공을 10성으로 올리는 데 49포인트가 필요하다.
- [ ] 3성 심법을 8성으로 올리는 데 30포인트가 필요하다.

### 데모 조기 절초

- [ ] 1~4전에서 받은 포인트를 핵심무공에 전부 투자할 수 있다.
- [ ] A등급 4회는 최소 52포인트를 지급해 4전 종료 후 절초를 확정 개방한다.
- [ ] A등급 3회와 S등급 1회는 최소 55포인트를 지급한다.
- [ ] B등급 4회는 최대 40포인트이므로 예선 결승 전 절초를 개방하지 못한다.
- [ ] 4전 종료 후 개방한 절초를 5전째 지역 예선 결승에서 사용할 수 있다.
- [ ] 절초가 없어도 5전째 지역 예선 결승을 시작하고 완료할 수 있다.

### 전체판 성장

- [ ] 결승 전 수련 단계가 최대 9회다.
- [ ] B등급 기대값 9점×9회가 81점으로 계산된다.
- [ ] B등급 중심 일반 플레이에서 결승 전 핵심무공 10성·심법 8성 전후가 가능하다.
- [ ] 10전 종료 후 남는 수련 포인트의 처리 방식이 데이터와 UI에서 일치한다.

## 7. 제약

- [ ] 전투 전 제약 후보와 위험도·불이익·승리 보상을 확인할 수 있다.
- [ ] 삼수양보는 첫 3타이밍 동안 공격·밀치기를 막는다.
- [ ] 단전봉맥·기혈역행·일장지양 등 제약이 설명대로 동작한다.
- [ ] 특정 문파의 핵심무공 전체를 사용 불가로 만드는 제약은 생성되지 않는다.
- [ ] 제약에 패배하면 추가 보상을 받지 않는다.
- [ ] 제약 없이도 일반 목표 성장 곡선을 달성할 수 있다.

## 8. 대진·데모·재도전·저장

- [ ] 시작 6세력 각각으로 새 회차를 시작할 수 있다.
- [ ] 추가 6세력이 적으로 등장하고 대표 무공을 보상으로 획득할 수 있다.
- [ ] 데모 5전이 내부전→추천패→지역 예선→지역 예선 결승으로 진행된다.
- [ ] 전체판 10전이 내부전→추천패→예선→본선→8강→준결승→결승으로 진행된다.
- [ ] 8전은 8강, 9전은 준결승, 10전은 결승이다.
- [ ] 같은 상대가 풀 소진 전 반복되지 않는다.
- [ ] 패배 후 재도전 1회와 재기 보상이 작동한다.
- [ ] 저장 데이터에 무공·심법·성급·수련 포인트·제약·성과·재도전 상태가 포함된다.
- [ ] 데모 종료 상태와 전체판 진행 상태를 구분해 저장한다.
- [ ] 저장·불러오기 뒤 AI 2수 잠금 공정성과 회차 상태가 유지된다.

## 9. 보류 경계

- [ ] 문파별 기믹 수행도는 성과 점수 계산에 포함되지 않는다.
- [ ] `docs/[보류]/`의 항목은 사용자 재개 결정 전 구현되지 않는다.
