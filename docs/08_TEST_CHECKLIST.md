# 12세력 대회 세로 슬라이스 테스트 체크리스트

> 테스트의 초점은 ‘소림 대 당문 한 판’이 아니라, 어떤 시작 문파를 골라도 내부전부터 본선·결승까지 공용 규칙 위에서 다른 전략을 만들고 완주 욕구를 주는지다.

## 1. 공용 전장·동시 선택

- [ ] 모든 결투가 A=4번 칸, B=6번 칸, 초기 거리 2에서 시작한다.
- [ ] 10칸 전장에 같은 칸 중첩, 교차, 자리 교환이 가능하다.
- [ ] 양측은 6슬롯 전체를 비공개로 잠근 뒤 동시에 공개한다.
- [ ] AI 행동열은 플레이어 잠금 전 사전 생성되며, 이후 변경되지 않는다.
- [ ] 접근·후퇴·공격·막기·회피·밀치기·무기 교체·재장전이 전 세력에서 공통으로 작동한다.
- [ ] 이동 후 최종 위치를 기준으로 공격·밀치기 사거리를 판정한다.
- [ ] 결과 로그가 공용 행동 → 무기 규칙 → 유파/대성 → 최종 결과 순서로 설명된다.

## 2. 무기·보조장비·상태

- [ ] 권법 0, 검·도 1, 창·곤 2, 암기 2~4, 활 3~5 사거리가 데이터와 UI에 일치한다.
- [ ] 일반 막기는 가능하고, 방패는 활 피해 막기를 높이며 회피 효율을 낮춘다.
- [ ] 밀치기는 모든 무기가 쓸 수 있고 해당 무기 사거리를 따른다.
- [ ] 무기 교체는 행동 슬롯 1개와 행동력 1을 사용한다.
- [ ] 암기벨트는 제한 횟수 내 슬롯·행동력 없이 기존 행동에 붙는다.
- [ ] 화살통은 제한 횟수 내 재장전 슬롯·행동력을 면제한다.
- [ ] 독은 적중·라운드 종료 피해가, 마화는 체력 대가·라운드 종료 자해가 정확히 처리된다.

## 3. 11세력 전투 데이터

각 항목은 세력별로 한 번 이상 AI 또는 플레이어로 전투를 끝까지 수행한다.

- [ ] 화산: 검 이동·회피 연계
- [ ] 개방: 곤 거리 2, 밀치기·막기
- [ ] 당문: 암기 재장전, 독침, 암기벨트
- [ ] 소림: 권법 거리 0, 금강호신
- [ ] 무당: 내공 보법·수비 전환
- [ ] 마교: 마화 체력 대가와 중첩 피해
- [ ] 양가장: 창 거리 2, 밀치기·회수
- [ ] 백우궁: 활 거리 3~5, 재장전·화살통
- [ ] 관군: 방패 활 상성·회피 감소
- [ ] 아미: 점혈 기력 압박·의술 회복
- [ ] 오독: 독장 근접 독 누적
- [ ] 녹림채(사파): 도 거리 1, 약탈도 기력 탈취

## 4. 수련·대성·대진

- [ ] 시작 6세력 각각으로 새 회차를 시작할 수 있다.
- [ ] 추가 6세력 모두가 적으로 대진에 편성되고, 대표 무공서는 보상으로 획득 가능하다.
- [ ] 녹림채가 사파 분류로 표시되고, 도적·산적 대진으로 등장한다.
- [ ] 모든 세력의 1~5성 해금 설명과 실제 전투 효과가 일치한다.
- [ ] 외문 무공을 1성부터 습득·수련할 수 있다.
- [ ] 9전이 내부전(1~2) → 추천패(3) → 예선(4~5) → 본선(6~7) → 준결승(8) → 결승(9)으로 진행된다.
- [ ] 같은 상대가 대진 풀 소진 전 반복되지 않는다.
- [ ] 보상 선택이 다음 전투의 행동열·무기·유파 규칙에 실제 영향을 준다.
- [ ] 패배 후 재도전 1회, 재기 전리품, 재기 수련도가 정상 작동한다.

## 5. 플레이 검증과 승인 기준

- [ ] 시작 6세력 각각으로 본선(6전)에 최소 1회 진입한다.
- [ ] 시작 6세력 각각으로 9전 결승까지 최소 1회 도달한다.
- [ ] 11세력 전부에 대해 승리·패배 결과와 로그를 기록한다.
- [ ] AI가 비공개 행동을 알고 대응했다는 인상을 주는 사례가 없고, 모든 주요 선택을 설명할 수 있다.
- [ ] 플레이어가 패배 뒤 다음 회차에서 바꿀 거리·행동열·수련도·보상 선택을 하나 이상 말할 수 있다.
- [ ] 플레이어가 다른 시작 문파로 다시 대회를 시작하고 싶다고 평가한다.
