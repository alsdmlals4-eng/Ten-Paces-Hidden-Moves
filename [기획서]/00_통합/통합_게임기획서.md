# 십보강호 통합 게임 기획서

## 한 줄 약속

10칸 전장에서 양측이 두 행동을 비공개로 잠그고 동시에 공개하며, 거리·자원·합·상대 성향을 읽어 승부하는 1대1 무협 로그라이트.

## 핵심 경험

- 상대의 다음 두 수를 읽고 접근·후퇴·공격·방어를 묶어 선택한다.
- 같은 타이밍의 공격은 합으로 충돌하고 차이만큼 피해가 난다.
- 절초는 성장으로 해금하고 전투 판단으로 기세를 채워 발동한다.
- 절초가 없어도 기본 전투 판단만으로 정상 승리할 수 있다.
- 상대 내부 수치는 이명·풍문·전적으로 암시하며 개발용 태그를 직접 노출하지 않는다.

## 현재 제작 단계

| 영역 | 상태 |
|---|---|
| 핵심 GDD·전투 구조 | 승인 기획 |
| 절초 기세 규칙 | T0 플레이테스트 기준선 |
| Godot 프로젝트 | 4.7.1용 최소 프로토타입 구성 |
| 자동 전투 테스트 | 스크립트 작성, 실제 4.7.1 실행 확인 필요 |
| 수동 플레이테스트 | 진행표·중단 기준 준비 |
| 5전 대회·성장 | T1 후속 범위 |
| 12세력 전체 | T0·T1 검증 뒤 확장 |

- 프로젝트: `project.godot`
- 메인 씬: `scenes/combat_playtest.tscn`
- 전투 도메인: `src/combat/combat_simulator.gd`
- 전투 UI: `src/ui/combat_playtest.gd`
- 자동 테스트: `tests/run_all.gd`
- 실행 가이드: `PLAYTEST.md`
