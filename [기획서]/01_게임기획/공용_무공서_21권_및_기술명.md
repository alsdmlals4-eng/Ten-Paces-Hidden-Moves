# 공용 무공서 21권 및 기술명

## 1. 적용 범위

공용 무공은 특정 문파에 속하지 않는 강호의 기본기·전승 무공이다. 카드 좌상단에는 문파 문양 대신 `무(無)` 상징 이미지를 사용한다.

공용 무공은 문파 고유 기믹을 새로 생성하지 않는다. 다만 문파 무공이나 보조무공이 이미 만든 상태·자원·연계 효과의 적용 대상이 될 수 있다.

### 공용 핵심무공 6종

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법`

- 궁술과 암기술은 문파 정체성이 강하므로 공용 무공에서 제외한다.
- 궁술은 백우궁·녹림, 암기술은 당문·사파의 전용 계통으로 유지한다.

### 공용 보조무공 4종

`심법 / 호신공 / 보법 / 외공`

- 독공·의술은 당문, 마공은 마교, 방패술은 양가장의 정체성을 유지하기 위해 공용에서 제외한다.

## 2. 성장 구조

| 성급 | 결과 |
|---:|---|
| 3성 | 새로운 기술 1 해금 |
| 5성 | 기술 1 강화 — 신규 카드 없음 |
| 7성 | 새로운 기술 2 해금 |
| 9성 | 기술 2 강화 — 신규 카드 없음 |
| 10성·대성 | 핵심무공은 절초, 보조무공은 진의 해금 |

공용 무공서 한 권당 신규 카드 이미지는 3성·7성·10성 총 3장이다.

- 공용 무공서: 21권
- 신규 공용 카드: 63장
- 5성·9성 강화는 기존 카드의 상세 데이터만 변경한다.

## 5. 명칭 출처와 창작 경계

### 실존·전승 또는 널리 알려진 계통명

`청평검 / 장권 / 통배권 / 팔극권 / 철사장 / 육합창 / 제미곤 / 토납 / 철포삼 / 구궁보`

- 청평검은 점·붕·자·운·말·괘·도·료·벽·제·발·전·절·란·초·소 같은 전통 검술 동작을 포함하는 계통으로 알려져 있다.
- 통배권은 등과 어깨에서 힘을 연결해 길고 빠른 타격을 내는 특징을 반영한다.
- 팔극권의 `철산고`, `맹호경파산`은 장르와 전승에서 널리 알려진 대표 어휘를 우선 사용한다.
- 육합창의 `난·나·찰`은 창술의 막기·붙잡기·찌르기 기본 원리를 합친 이름이다.
- 제미곤은 사용자의 눈썹 높이에 가까운 봉을 뜻하는 전통 병기명에서 가져왔다.

### 무협 범용 또는 프로젝트 창작명

나머지 기술·절초·진의 이름은 해당 무공의 전통 이미지와 십보강호의 성향 구분을 바탕으로 고안했다. 특정 작품의 고유 초식 목록을 그대로 복제하지 않는다.

## 6. 중복 명칭 정리

공용 무공서 `철포삼`이 확정되었으므로 소림의 `금강불괴신공` 3성 기술에 사용하던 `철포삼` 명칭은 중복을 피하기 위해 **금종호체**로 변경한다.

- 금강불괴신공(호신공): **금종호체 / 금종조 / 진의·금강불괴**

## 7. 제작 경계

현재 확정 범위:

- 무공서 21권의 이름
- 각 무공서의 성향
- 3성·7성·10성 카드 이름
- 핵심무공의 절초, 보조무공의 진의 구분

현재 미확정 범위:

- 피해·회복·방어·이동 수치
- 합·막기·밀치기 같은 특수효과
- 비용과 사거리
- 5성·9성 강화 내용
- 묘리 한 소절
- 무공서·기술 카드 이미지

효과와 이미지는 별도 승인 단계에서 작성한다.
