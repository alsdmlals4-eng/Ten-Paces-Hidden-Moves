# 십보강호 게임 기획서

## 0. 한눈에 보기

**십보강호**는 출신 배경을 가진 강호낭인이 천하제일인을 목표로 산을 오르며, 분기형 경로에서 비무·수련·기연·문파 교류를 선택하고 자신만의 무학을 완성하는 **1대1 무협 로그라이트**다.

핵심 경험은 네 가지다.

1. 10칸 전장에서 상대의 다음 두 수를 읽고 자신의 두 수를 먼저 잠그는 예측전.
2. 합·거리·자원·행동 순서에 따라 같은 기술의 가치가 달라지는 전술성.
3. 수련포인트와 한정된 일정 안에서 소수 무공을 대성하거나 여러 무공을 넓게 익히는 성장 선택.
4. 갈림길을 고르고 문파·가문·기인·사건과 조우하며 회차마다 다른 무공서를 얻는 등반 경험.

주인공의 신분은 항상 **강호낭인**이다. 게임 시작 시 출신 배경을 선택하고, 개방된 초기 무공서 중 4권을 골라 모두 3성으로 시작한다. 따라서 첫 전투부터 각 무공의 3성 기술 하나씩, 총 기술 4개를 보유한다.

## 1. 제품 구조와 등반 목표

### 1.1 천하제일인을 향한 산행

플레이어는 산 아래에서 출발해 갈림길을 선택하며 정상으로 오른다. 각 경로에는 비무, 강적, 수련, 기연, 문파 교류, 탐색, 객잔, 장터, 돌파 시험이 배치된다.

```text
출신 배경 선택
→ 시작 무공서 4권 선택·3성 시작
→ 등반 경로 확인
→ 다음 노드 선택
→ 전투·수련·사건·교류 해결
→ 무공서 습득 또는 수련 투자
→ 다음 갈림길
→ 정상의 최종 비무
→ 천하제일인 도전 결과 정산
```

- 경로 선택은 되돌릴 수 없다.
- 지도에서 일부 노드의 종류는 보이지만 `?` 사건의 실제 내용은 숨긴다.
- 전투 외 노드도 일정을 소비하므로 한 회차에 모든 성장 기회를 선택할 수 없다.
- 선택한 길에 따라 만나는 인물, 세력, 무공서와 강적이 달라진다.

상세 규칙은 [강호 등반 경로 구조](강호_등반_경로_구조.md)를 따른다.

### 1.2 10-10-10 구조

| 항목 | 기준 |
|---|---:|
| 전장 | 직선 10칸 |
| 라운드 | 행동 타이밍 10개 |
| 한 회차 | 정상까지 핵심 비무 10전 목표 |
| 데모 | 하단 구간부터 중간 관문까지 축소 |

한 라운드는 두 행동을 한 묶음으로 선택하는 과정을 다섯 번 반복한다.

```text
AI 두 수 선잠금
→ 플레이어 두 수 선택·잠금
→ 양측 두 수 동시 공개
→ 1수 해상
→ 2수 해상
```

첫 수의 결과를 본 뒤 둘째 수를 바꿀 수 없다.

핵심 비무 10전을 모든 지도에서 정확히 고정할지는 경로 밀도와 플레이타임 테스트 후 최종 확정한다.

### 1.3 단일 이벤트 원칙

전투가 게임의 중심이므로 이벤트는 길게 이어지는 서사 분기가 아니라 한 화면에서 끝나는 단일 사건으로 구성한다.

```text
짧은 상황 설명
→ 선택지 2~4개
→ 조건 확인 및 필요 시 능력 판정
→ 성공·실패 결과
→ 보상·손실·성향·관계·일정 반영
→ 지도 또는 전투로 복귀
```

- 능력치·성향·관계·무공·소지품 조건으로 선택지가 잠길 수 있다.
- 잠긴 선택지는 특별한 비밀 조건을 제외하면 필요한 조건을 보여준다.
- 실패는 무효 결과로 끝내지 않고 부상·관계 변화·추가 전투 등 짧은 후속 결과로 연결한다.
- 이벤트는 전투 템포를 방해하지 않도록 한 번의 선택과 한 번의 결과를 기본으로 한다.

## 2. 전투 핵심 규칙

### 2.1 두 수 선결정

전투의 중심은 한 행동의 강함이 아니라 **두 행동의 연결과 상대 예측**이다.

- 양측은 연속 행동 두 개를 미리 잠근다.
- 첫 행동의 결과가 둘째 행동의 위치, 자원, 사거리와 사용 조건에 영향을 준다.
- 첫 행동 해결 뒤 둘째 행동을 교체할 수 없다.
- 상대의 거리, 남은 자원, 보유 기술, 성향과 직전 행동을 읽을 근거를 제공한다.

### 2.2 거리와 위치

- 양측은 10칸 직선 전장에서 싸운다.
- 각 행동 해결 뒤의 위치를 기준으로 사거리와 대상 유효성을 다시 판정한다.
- 접근·후퇴·보법·밀치기·끌어당기기·위치 교환이 거리 싸움을 만든다.
- 무기와 무공은 고유 사거리와 위치 조건을 데이터로 가진다.

### 2.3 합

같은 타이밍에 양측 공격이 모두 유효하면 `[합]`이 발생한다.

- 최종 합 수치를 비교한다.
- 낮은 쪽이 차이만큼 피해를 받는다.
- 동률이면 양측 모두 피해를 받지 않는다.
- 합과 일반 공격은 로그·애니메이션·VFX·SFX에서 명확히 구분한다.

기술의 가치는 피해량 하나가 아니라 합, 거리, 조건, 자원, 후속 연계로 평가한다.

### 2.4 기본 행동과 자원

공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전, 명상이다.

주요 전투 자원:

- 행동력: 한 라운드의 행동 수행 자원.
- 기력: 육체적 기술과 무기 운용 자원.
- 내력: 심법·내공·특수 무공에 소비하는 자원.
- 절초 기세: 10성 절초의 전투 중 발동 조건. 정확한 획득·상한·소모 수치는 미확정이다.

UI·애니메이션·VFX·사운드는 전투 결과를 계산하지 않고 전투 도메인이 만든 구조화 이벤트를 표현한다.

### 2.5 기본 능력치와 성향

플레이어의 기본 성장 능력치는 다음 6종이다.

| 능력치 | 주요 역할 |
|---|---|
| 근골 | 최대 체력·기력과 부상 저항 |
| 내공 | 최대 내력, 내공 운용과 기맥 판정 |
| 외공 | 물리 공격력, 완력과 파괴 판정 |
| 호신 | 방어력, 막기와 충격 저항 |
| 경공 | 이동·회피와 일정 수치 도달 시 행동력 증가 |
| 오성 | 비급 해독, 사건 통찰, 수련과 돌파 보조 |

파생 전투값은 체력·기력·내력·공격력·방어력·행동력으로 구분한다.

성향은 하나의 연속 축으로 관리한다.

```text
협 ←──────── 중용 ────────→ 실리
```

- 협은 약속·보호·명분을 우선하는 행동으로 기운다.
- 중용은 상황에 따른 절충과 균형을 뜻한다.
- 실리는 보상·생존·거래와 확실한 이익을 우선하는 행동으로 기운다.
- 어느 성향도 절대적인 정답이나 악으로 취급하지 않으며 사건 선택지·보상·NPC 반응과 무공 획득 경로를 다르게 만든다.

## 3. 무공 성장과 시작 빌드

### 3.1 공식 무공 분류

현재 공식 분류는 다음을 기준으로 하며, 장풍·빙한·음파 같은 차이는 새로운 대분류보다 속성·운용 태그를 우선 사용한다.

**핵심무공 8종**

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법 / 궁술 / 암기술`

**보조무공 8종**

`심법 / 호신공 / 독공 / 마공 / 보법 / 방패술 / 외공 / 의술`

- 부법은 공식 성장 계통에서 제외한다.
- 도끼는 개별 무기·연출 태그로 사용한다.
- 점혈은 의술의 세부 기술·상태 태그다.
- 신법·경공은 보법의 세부 태그다.
- 음공의 공식 대분류 추가 여부는 신규 무공 설계 후 결정한다.

### 3.2 성급 해금

| 성급 | 결과 |
|---:|---|
| 3성 | 새로운 기술 1 해금 |
| 5성 | 기술 1 강화 — 신규 카드 없음 |
| 7성 | 새로운 기술 2 해금 |
| 9성 | 기술 2 강화 — 신규 카드 없음 |
| 10성·대성 | 핵심무공은 절초, 보조무공은 진의 해금 |

기술 강화는 기존 기술을 대체하지 않는다. 카드 이름과 삽화는 유지하고 상세 효과·수치·조건을 강화한다.

무공서 한 권이 제공하는 사용 가능 카드는 3성·7성·10성 총 3장이다.

### 3.3 무공서 정보 구조

무공서는 단순한 카드 묶음이 아니라 무공의 기원, 전투 방식과 철학을 전달하는 콘텐츠 단위다.

```text
무공 설명
+ 무도(武道)
+ 3성 기술 카드
+ 7성 기술 카드
+ 10성 절초 또는 진의 카드
```

- **무공 설명**: 무공의 출처, 기본 운용 방식과 전투 성향.
- **무도**: 해당 무공이 추구하는 길과 원리를 표현하는 문장.
- **기술 묘리**: 각 초식의 핵심 원리를 카드 아래 최대 두 줄로 표현.
- 설명·무도·기술 묘리는 문파 전승, 공용, 군문, 산림, 작자 불명 무공에 모두 적용한다.

### 3.4 시작 배경과 시작 무공

주인공은 항상 강호낭인이며 문파를 선택하지 않는다. 게임 시작 시 출신 배경을 하나 고른다.

배경은 다음에만 영향을 준다.

- 초기 무공서 후보.
- 시작 수련포인트·금전·장비.
- 일부 사건 선택지와 인물의 첫 반응.
- 초기 성향 또는 정보.

배경은 이후 무공 습득을 제한하지 않는다.

| 항목 | 확정 기준 |
|---|---:|
| 시작 무공서 | 4권 |
| 시작 숙련 | 모두 3성 |
| 시작 기술 | 4개 |
| 시작 후보 | 배경과 기본 개방 목록이 제공하는 무공서 |
| 이후 습득 | 출처·문파·계통과 무관하게 자유 |

```text
출신 배경 선택
→ 초기 후보에서 무공서 4권 선택
→ 선택 무공 전부 3성으로 시작
→ 각 무공의 3성 기술 해금
→ 첫 전투부터 기술 4개 사용 가능
```

### 3.5 자유 수련과 성장 선택

플레이어는 어느 문파·가문·유파의 무공이든 자유롭게 습득하고 함께 익힐 수 있다. 계통 충돌이나 문파 제한을 기본 제약으로 두지 않는다.

자유 수련을 제한하는 것은 다음 두 자원이다.

- **수련포인트**: 새 무공 입문과 성급 상승에 소비.
- **일정**: 비무·수련·교류·탐색·휴식 중 무엇을 선택할지 제한하는 시간.

이에 따라 두 성장 방향이 경쟁한다.

- **집중 수련**: 소수 무공을 7성·10성까지 성장해 강한 상위 기술과 절초·진의를 확보.
- **박학 수련**: 여러 무공을 3성·7성까지 익혀 더 많은 기술과 두 수 조합을 확보.

핵심무공과 보조무공의 대성 개수에 별도의 강제 상한을 두지 않는다. 실제 대성 수는 수련 비용, 일정과 돌파 조건으로 제한한다.

### 3.6 무공 데이터 책임

무공 데이터는 `무공서 → 숙련 → 기술 → 표시` 계층으로 분리한다.

- 무공서는 출처·분류·설명·무도·기술 참조를 소유한다.
- 기술은 대상·사거리·비용·효과·조건·키워드를 소유한다.
- 5성·9성은 새 카드를 만들지 않고 기존 기술에 강화 패치를 적용한다.
- 이름·기술 묘리·아이콘·삽화는 표시 데이터이며 전투 계산에 관여하지 않는다.
- 상세 필드는 [무공 데이터 구조](무공_데이터_구조.md)를 따른다.

## 4. 문파·유파와 무공서 출처

문파는 플레이어 클래스가 아니라 등반 중 만나고 교류하는 세계관 세력이다.

- 문파·가문·유파·산채·기인은 무공서의 출처다.
- 우호도, 사건 선택, 성향, 비무 결과에 따라 전수와 돌파 기회가 달라진다.
- 이미 익힌 무공은 관계가 악화되어도 잃지 않는다.
- 정식 사사 외에도 기연, 거래, 탈취, 유실 비급, 독자적 깨달음으로 무공을 얻을 수 있다.

### 4.1 처음 조우와 도감

문파를 처음 조우하면 전투나 사건에 진입하기 전에 짧은 문파 소개 화면을 한 번 표시하고, 이후 도감에서 다시 확인할 수 있게 한다.

```text
문파 문양 + 문파명
문파 설명
추구하는 무학
대표 캐릭터 1~2명
핵심 무공서 2~4권
```

- **문파 설명**: 역사·거점·강호 내 위치와 현재 성격.
- **추구하는 무학**: 문파 전체가 무공을 통해 이루려는 경지와 전투 원칙. 문파 소개에서 사용하던 구 `무공의 묘리` 명칭을 이 용어로 교체한다.
- 핵심 무공서는 표지·이름·분류를 보여주며, 개별 기술은 무공서 상세 화면에서 확인한다.
- 문파 설명과 추구하는 무학은 이미지에 굽지 않고 편집 가능한 UI 텍스트로 관리한다.

용어는 다음처럼 구분한다.

- 문파·유파 전체: `추구하는 무학`.
- 무공서 한 권: `무도`.
- 기술 카드 한 장: `기술 묘리`.

상세 시각 구조는 [문파 소개·도감 시트 명세](../04_아트디자인/문파_소개_도감_시트_명세.md)를 따른다.

### 4.2 현존 문파 전승

- **무당파**: 태극검 / 태극권 / 태극신공 / 제운종.
- **화산파**: 매화검법 / 자하신공.
- **소림사**: 나한권 / 달마봉법 / 금강불괴신공.
- **당문**: 당문암기비전 / 오독심경 / 금침의경 / 귀영보.
- **개방**: 항룡십팔장 / 타구봉법 / 취팔선보.
- **마교**: 혈염도법 / 역혈마공. 일반 교도의 무공에는 `천마` 명칭을 사용하지 않는다.

### 4.3 가문·군문·지역 전승

- **양씨 군문 전승**: 양가창법 / 양문철벽패술.
- **설산 궁술 유파 전승**: 백우궁술 / 유운보법.
- **산채·녹림 전승**: 녹림창법 / 산림궁술 / 산림잠행보.
- **강호 비정통 전승**: 오호단문도 / 추혼암기술 / 철골외공.

이전의 양가장·백우궁·녹림·사파는 플레이어 시작 문파 슬롯으로 사용하지 않으며, 역사·문화·전투 색채는 무공서의 설명과 무도에 남긴다.

### 4.4 개방 무공서

기존 공용 무공서는 총 21권이다.

- 핵심무공서 13권: 검법·도법·권법·장법·창법·곤봉법.
- 보조무공서 8권: 심법·호신공·보법·외공.
- 상세 목록과 기술명은 [공용 무공서 21권](공용_무공서_21권_및_기술명.md)을 따른다.

`공용`은 무색무취한 무공이라는 뜻이 아니라, 시작 후보나 일반 강호 경로에서 비교적 쉽게 접할 수 있는 **개방 전승**을 뜻한다.

### 4.5 신규 무공 확장

새로운 문파 전체를 만들지 않아도 무공서 한 권과 출처 사건만으로 새로운 플레이 스타일을 추가할 수 있다.

확장 후보:

- 장풍·장력 계통.
- 빙공·한기 계통.
- 음공·악기·사자후 계통.
- 기문·진법·점혈 특화 계통.
- 열양·혈기·섭혼 같은 희귀 또는 금기 전승.

## 5. 카드와 UIUX

### 5.1 무공 카드 앞면

- 좌상단: 무공서 출처 문양. 개방·작자 불명 전승은 `무(無)` 상징 이미지.
- 최상단 중앙: 사거리 슬롯. 숫자는 아트에 굽지 않고 UI 레이어에서 표시.
- 상단 중앙: 기술 이름. 편집 가능한 텍스트 레이어.
- 우상단: 검법·권법·심법 같은 분류를 텍스트가 아닌 아이콘 이미지로 표시.
- 중앙: 기술 삽화.
- 하단: 비용·효과 없이 **기술 묘리** 한 소절.
- 기술 묘리는 최대 두 줄, 큰 글자와 높은 대비를 사용한다.
- 10성 핵심무공은 우측 세로 `절초`, 보조무공은 `진의` 표찰을 사용한다.

### 5.2 기술 선택 UI

플레이어가 익힌 무공과 기술 수에는 인위적인 기본 상한을 두지 않는다.

- 해금한 기술은 전투에서 모두 사용 후보가 된다.
- 무공서별 묶음, 공격·이동·방어·자원 역할 필터와 즐겨찾기를 제공한다.
- 무작위 손패를 뽑는 덱빌딩 구조는 사용하지 않는다.
- 두 행동 선택 화면에서 첫 수와 둘째 수의 비용·사거리·예상 위치 변화를 함께 확인할 수 있어야 한다.

### 5.3 상세 팝업

상세 팝업은 다음 순서로 정보를 표시한다.

1. 이름.
2. 대상.
3. 효과 — 피해, 회복량, 방어도, 이동량.
4. 특수효과 — 합, 막기, 밀치기, 이동 방식, 반격, 상태 등.
5. 비용.
6. 필요한 경우 설명과 키워드 해설.

### 5.4 합성 파이프라인

최종 카드는 생성 이미지 한 장을 그대로 사용하지 않고 편집 가능한 합성 구조로 제작한다.

- 이미지: 무공서 표지, 기술 삽화, 출처 문양, 분류 아이콘, 카드 프레임, 절초·진의 표찰.
- 데이터·텍스트: 이름, 설명, 무도, 기술 묘리, 성급, 사거리, 비용, 효과, 키워드.
- 생성 이미지 속 문자와 수치는 `[아트 방향 참고용]`이며 기준 데이터가 아니다.

## 6. 아트와 세계관 표현

- 주인공은 특정 문파 복식으로 고정하지 않고 출신 배경과 획득 장비가 반영되는 강호낭인으로 디자인한다.
- 문파·가문·유파 시트는 플레이어 클래스 소개가 아니라 세계관 세력과 무공서 출처를 전달하는 화면이다.
- 문파 소개 시트는 `대표 캐릭터 + 문파 설명 + 추구하는 무학 + 핵심 무공서` 구조를 사용한다.
- 각 무공서 표지에는 전승 집단의 색, 지역, 상징과 무도를 담되 텍스트는 합성 레이어로 분리한다.
- 현재까지 생성된 문파·공용 무공 이미지는 모두 `[아트 방향 참고용]`으로 관리한다.
- 경로 지도는 산 아래에서 정상으로 이어지는 세로형 분기 구조와 무협식 노드 문양을 사용한다.

## 7. 밸런스 원칙

- 무공 간 차이를 단순 피해량으로만 만들지 않는다.
- 거리, 합, 자원, 준비 시간, 조건, 위험과 대응 가능성으로 역할을 만든다.
- 시작 기술 4개가 공격·방어·이동·자원 중 최소 두 역할 이상을 수행할 수 있도록 정보를 제공한다.
- 3성 기술은 해당 무공의 기본 운용법을 가르친다.
- 7성 기술은 조건부 연계와 상위 운용을 제공한다.
- 절초·진의는 강력하지만 높은 수련 비용, 10성 해금과 전투 중 조건을 요구한다.
- 여러 무공을 넓게 익힌 빌드와 한 무공을 대성한 빌드가 서로 다른 장점을 가져야 한다.
- 문파 관계는 습득과 성장 경로를 바꾸되 이미 익힌 무공의 사용을 금지하지 않는다.
- 플레이어가 패배 이유를 거리·자원·합·행동 순서 중 하나 이상으로 설명할 수 있어야 한다.

## 8. 현재 상태

### 확정

- 주인공은 출신 배경을 선택하는 강호낭인.
- 목표는 산 정상의 천하제일인 도전.
- 분기형 경로에서 비무·수련·기연·교류 노드를 선택하는 진행 구조.
- 이벤트는 선택과 결과가 한 화면에서 끝나는 단일 사건 구조.
- 기본 능력치는 근골·내공·외공·호신·경공·오성.
- 성향은 협·중용·실리의 단일 축.
- 10칸 전장과 두 수 선잠금·동시 공개·순차 해상.
- 3·5·7·9·10성 성장 구조.
- 시작 무공서 4권·전부 3성·시작 기술 4개.
- 모든 무공을 출처·문파·계통 제한 없이 습득·혼합 가능.
- 수련포인트와 일정이 성장의 자연 제약.
- 집중 대성과 박학 수련의 성장 선택.
- 무공서의 `무공 설명 + 무도 + 3성·7성·10성 카드` 구조.
- 문파 소개의 `캐릭터 + 문파 설명 + 추구하는 무학 + 핵심 무공서` 구조.
- `추구하는 무학 / 무도 / 기술 묘리` 용어 분리.
- 카드 앞면 정보 구조, 절초·진의 표찰과 편집형 합성 파이프라인.

### 진행 중

- 출신 배경 목록과 초기 후보군.
- 등반 지도 구간·노드·보상 구조.
- 수련포인트와 일정의 정확한 수치.
- 문파·기인·성향에 따른 무공서 획득 조건.
- 능력치 파생값과 이벤트 판정식.
- 무공 데이터 필드와 계산 규칙.
- 각 기술의 효과·비용·사거리·밸런스.
- 절초 기세의 획득·상한·소모 수치.
- 문파 설명·추구하는 무학·무공 설명·무도·기술 묘리 작성.
- 최종 실장용 지도·캐릭터·무공서·기술 카드 합성 자산.

### 보류

- 복잡한 랜덤 손패·덱빌딩.
- 계통 충돌에 의한 강제 수련 패널티.
- 문파 선택형 플레이어 클래스.
- 별도 장비 강화·경제 시스템의 상세화.
- 패배 후 메타 성장과 영구 해금 범위.
