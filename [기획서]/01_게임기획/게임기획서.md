# 십보강호 게임 기획서

## 0. 한눈에 보기

**십보강호**는 10칸 전장에서 상대의 거리·자원·성향을 읽고 두 수를 미리 잠근 뒤 동시에 공개하는 **1대1 무협 로그라이트 비무 대회 게임**이다.

핵심 경험은 세 가지다.

1. 상대가 낼 수를 읽고 두 수를 미리 설계하는 예측전.
2. 합·거리·자원·무공 조합 때문에 같은 기술도 상황에 따라 가치가 달라지는 전술성.
3. 문파 무공과 공용 무공을 조합하고 수련해 새로운 기술과 절초·진의를 얻는 성장감.

게임 시작 시 플레이어는 **본인 문파 무공과 공용 무공 중 4개를 선택**하며, 선택한 무공은 모두 **3성**으로 시작한다. 따라서 첫 전투부터 각 무공의 3성 기술 하나씩, 총 **기술 4개**를 보유한다.

## 1. 제품 구조

### 1.1 10-10-10 구조

| 항목 | 기준 |
|---|---:|
| 전장 | 10칸 |
| 라운드 | 행동 타이밍 10개 |
| 전체 대회 | 10전 |
| 데모 | 1~5전, 예선 결승까지 |

한 라운드는 두 행동을 한 묶음으로 선택하는 과정을 다섯 번 반복한다.

```text
AI 두 수 선잠금
→ 플레이어 두 수 선택·잠금
→ 양측 두 수 동시 공개
→ 1수 해상
→ 2수 해상
```

첫 수의 결과를 본 뒤 둘째 수를 바꿀 수 없다.

### 1.2 기본 플레이 흐름

```text
문파 선택
→ 시작 무공 4개 선택
→ 상대 정보 확인
→ 선택 제약·보상 결정
→ 비무
→ 성과·행운 정산
→ 수련 투자
→ 다음 상대
```

상대 강도와 성장 성향은 개발용 태그가 아니라 이명·풍문·전적·정탐 문장으로 전달한다.

## 2. 전투 규칙

### 2.1 거리와 위치

- 양측은 10칸 직선 전장에서 싸운다.
- 행동 해상 뒤의 위치를 기준으로 공격 사거리와 대상 유효성을 판정한다.
- 접근·후퇴·보법·밀치기·끌어당기기·위치 교환이 거리 싸움을 만든다.
- 무기와 무공은 고유 사거리를 데이터로 가진다.

### 2.2 합

같은 타이밍에 양측 공격이 모두 유효하면 `[합]`이 발생한다.

- 최종 합 수치를 비교한다.
- 낮은 쪽이 차이만큼 피해를 받는다.
- 동률이면 양측 모두 피해를 받지 않는다.
- 합과 일반 공격은 로그·애니메이션·SFX에서 명확히 구분한다.

### 2.3 기본 행동과 자원

공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전, 명상이다.

주요 자원:

- 행동력: 한 라운드의 행동 수행 자원.
- 기력: 육체적 기술과 무기 운용 자원.
- 내공: 내공·심법·특수 무공 자원.
- 절초 기세: 10성 절초의 전투 중 발동 조건. 정확한 획득·상한·소모 수치는 미확정이다.

UI·애니메이션·VFX·사운드는 전투 결과를 계산하지 않고 전투 도메인이 만든 구조화 이벤트를 표현한다.

## 3. 무공 성장과 시작 빌드

### 3.1 공식 무공 분류

**핵심무공 8종**

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법 / 궁술 / 암기술`

**보조무공 8종**

`심법 / 호신공 / 독공 / 마공 / 보법 / 방패술 / 외공 / 의술`

- 부법은 공식 성장 계통에서 제외한다.
- 도끼는 개별 무기·연출 태그로만 사용한다.
- 점혈은 의술의 세부 기술·상태 태그다.
- 신법·경공은 보법의 세부 태그다.

### 3.2 성급 해금

| 성급 | 결과 |
|---:|---|
| 3성 | 새로운 기술 1 해금 |
| 5성 | 기술 1 강화 — 신규 카드 없음 |
| 7성 | 새로운 기술 2 해금 |
| 9성 | 기술 2 강화 — 신규 카드 없음 |
| 10성·대성 | 핵심무공은 절초, 보조무공은 진의 해금 |

기술 강화는 기존 기술을 대체하지 않는다. 카드 이름과 삽화는 유지하고 상세 효과·수치·조건이 강화된다.

무공서 하나의 사용 가능 카드는 3성·7성·10성 총 3장이다.

### 3.3 무공서 정보 구조

무공서는 단순한 카드 묶음이 아니라 무공의 성격과 철학을 전달하는 콘텐츠 단위다.

```text
무공 설명
+ 무도(武道)
+ 3성 기술 카드
+ 7성 기술 카드
+ 10성 절초 또는 진의 카드
```

- **무공 설명**: 무공의 기원, 기본 운용 방식과 전투 성향을 설명한다.
- **무도**: 해당 무공이 추구하는 길과 원리를 한 문단으로 표현한다.
- **기술 카드 묘리**: 각 기술 삽화 아래에 해당 초식의 묘리를 최대 두 줄로 표시한다.
- 무공 설명·무도·묘리는 문파 무공과 공용 무공 모두에 적용한다.

### 3.4 게임 시작 무공 선택

게임 시작 시 플레이어는 무공 4개를 골라 초기 빌드를 구성한다.

| 항목 | 확정 기준 |
|---|---:|
| 선택 무공 수 | 4개 |
| 시작 숙련 | 모두 3성 |
| 시작 기술 수 | 4개 |
| 선택 후보 | 본인 문파 무공 + 공용 무공 |
| 타 문파 전용 무공 | 시작 선택 불가 |

```text
문파 선택
→ 본인 문파 무공과 공용 무공에서 4개 선택
→ 선택 무공 전부 3성으로 시작
→ 각 무공의 3성 기술 해금
→ 첫 전투부터 기술 4개 사용 가능
```

이 규칙은 시작부터 문파 정체성과 공용 무공 조합을 모두 체험하게 하고, 무기·거리·자원 역할이 다른 네 기술로 첫 빌드를 구성하게 한다.

### 3.5 무공 데이터 책임

무공 데이터는 `무공서 → 숙련 → 기술 → 표시` 계층으로 분리한다.

- 무공서는 소속·분류·설명·무도·기술 참조를 소유한다.
- 기술은 대상·사거리·비용·효과·조건·키워드를 소유한다.
- 5성·9성은 새 카드를 만들지 않고 기존 기술에 강화 패치를 적용한다.
- 이름·묘리·아이콘·삽화는 표시 데이터이며 전투 계산에 관여하지 않는다.
- 상세 필드는 [무공 데이터 구조](무공_데이터_구조.md)를 따른다.

## 4. 10문파와 확정 무공서

### 4.1 무당파

- 태극검(검법): 삼환투월 / 연자초수 / 절초·진무태극검
- 태극권(권법): 람작미 / 운수 / 절초·사량발천근
- 태극신공(太極神功, 심법): 태극양의 / 음양회귀 / 진의·태극귀진
- 제운종(보법): 제운답 / 종운등천 / 진의·운외천행

### 4.2 화산파

- 매화검법(검법): 백운출수 / 매화난영 / 절초·태악삼청봉
- 자하신공(심법): 자하초현 / 자기동래 / 진의·자하만천

### 4.3 소림사

- 나한권(권법): 나한출수 / 복호나한 / 절초·금강나한인
- 달마봉법(곤봉법): 반야봉 / 횡소천군 / 절초·달마복마진
- 금강불괴신공(호신공): 금종호체 / 금종조 / 진의·금강불괴

### 4.4 당문

- 당문암기비전(암기술): 추혼비도 / 폭우이화침 / 절초·만천화우
- 오독심경(독공): 오독침맥 / 만독식심 / 진의·만독귀원
- 금침의경(의술): 금침도혈 / 회양구침 / 진의·생사환맥
- 귀영보(보법): 잠영환위 / 귀영천변 / 진의·무흔무영

### 4.5 개방

- 항룡십팔장(장법): 항룡유회 / 비룡재천 / 절초·시승육룡
- 타구봉법(곤봉법): 악견난로 / 오구탈장 / 절초·천하무구
- 취팔선보(보법): 취선망월 / 팔선과해 / 진의·대취무흔

### 4.6 마교

- 혈염도법(도법): 혈영참 / 수라난무 / 절초·혈해멸도
- 역혈마공(마공): 역맥폭기 / 혈기연소 / 진의·마혈귀원

일반 마교 교도의 무공에는 `천마` 명칭을 사용하지 않는다.

### 4.7 양가장

- 양가창법(창법): 이화분영창 / 회마반창 / 절초·칠성연환창
- 양문철벽패술(방패술): 금탕고수 / 철벽추진 / 진의·양문불퇴

### 4.8 백우궁

- 백우궁술(궁술): 백보천양 / 연주구성 / 절초·천우낙성
- 유운보법(보법): 비연답 / 답설무흔 / 진의·풍운무정

### 4.9 사파

- 오호단문도(도법): 맹호하산 / 오호단문 / 절초·군호파악
- 추혼암기술(암기술): 유엽비도 / 추혼십이표 / 절초·천라추혼
- 철골외공(외공): 동피철골 / 횡련강신 / 진의·십삼태보

사파 전용 계통에는 검법·장법·보법을 배치하지 않는다.

### 4.10 녹림

- 녹림창법(창법): 독룡출동 / 횡소천군 / 절초·회마칠성창
- 산림궁술(궁술): 관목잠시 / 연주산우 / 절초·만산비우
- 산림잠행보(보법): 초상비 / 답엽무흔 / 진의·천산무영

### 4.11 공용 무공

공용 무공서는 총 21권이다.

- 핵심무공서 13권: 검법·도법·권법·장법·창법·곤봉법.
- 보조무공서 8권: 심법·호신공·보법·외공.
- 궁술·암기술·독공·마공·방패술·의술은 문파 정체성 보존을 위해 공용에서 제외한다.
- 상세 목록과 기술명은 [공용 무공서 21권](공용_무공서_21권_및_기술명.md)을 따른다.

## 5. 카드와 UIUX

### 5.1 무공 카드 앞면

- 좌상단: 문파 문양 이미지. 공용은 `무(無)` 상징 이미지.
- 최상단 중앙: 사거리 슬롯. 숫자는 아트에 굽지 않고 UI 레이어에서 표시.
- 상단 중앙: 기술 이름. 편집 가능한 텍스트 레이어.
- 우상단: 검법·권법·심법 같은 분류를 텍스트가 아닌 아이콘 이미지로 표시.
- 중앙: 기술 삽화.
- 하단: 비용·효과 없이 무공의 묘리 한 소절.
- 묘리는 최대 두 줄, 큰 글자와 높은 대비를 사용한다.
- 10성 핵심무공은 우측 세로 `절초`, 보조무공은 `진의` 표찰을 사용한다.

### 5.2 상세 팝업 — 후속 단계

상세 팝업은 다음 순서로 정보를 표시한다.

1. 이름
2. 대상
3. 효과 — 피해, 회복량, 방어도, 이동량
4. 특수효과 — 합, 막기, 밀치기, 이동 방식, 반격, 상태 등
5. 비용
6. 필요한 경우 설명과 키워드 해설

### 5.3 합성 파이프라인

최종 카드는 생성 이미지 한 장을 그대로 사용하지 않고 편집 가능한 합성 구조로 제작한다.

- 이미지: 무공서 표지, 기술 삽화, 문양, 분류 아이콘, 카드 프레임, 절초·진의 표찰.
- 데이터·텍스트: 이름, 설명, 무도, 묘리, 성급, 사거리, 비용, 효과, 키워드.
- 생성 이미지 속 문자와 수치는 아트 방향 참고용이며 기준 데이터가 아니다.

## 6. 문파 캐릭터와 종합 시트

각 문파는 남녀 대표 제자 두 명을 가진다.

- 두 인물은 겹치지 않는 독립 세로 패널에 배치한다.
- 같은 문파 복식 체계를 공유하되 자세·시선·실루엣은 다르게 한다.
- 한 명은 핵심무공, 다른 한 명은 보조무공 또는 다른 핵심무공의 성격을 보여준다.
- 별도 무기·복식 상세 패널은 두지 않고 캐릭터 디자인 안에서 전달한다.

기본 구조:

```text
상단: 문파명·문양·정체성 문장
좌측: 남자 제자 패널 + 여자 제자 패널
중앙: 무공서
우측: 무공서별 3성 기술 / 7성 기술 / 10성 절초·진의 카드
```

## 7. 밸런스 원칙

- 문파 간 차이를 단순 피해량으로만 만들지 않는다.
- 거리, 합, 자원, 준비 시간, 조건, 위험과 대응 가능성으로 역할을 만든다.
- 시작 기술 4개가 공격·방어·이동·자원 중 최소 두 역할 이상을 수행할 수 있게 선택 정보를 제공한다.
- 3성 기술은 무공의 기본 운용법을 가르친다.
- 7성 기술은 조건부 연계와 상위 운용을 제공한다.
- 절초·진의는 강력하지만 무공 10성 해금과 전투 중 조건을 요구한다.
- 플레이어가 패배 이유를 거리·자원·합·행동 순서 중 하나 이상으로 설명할 수 있어야 한다.

## 8. 현재 상태

### 확정

- 10칸·10타이밍·10전 구조.
- 두 수 선잠금·동시 공개·순차 해상.
- 핵심 8종·보조 8종과 10문파 배치.
- 28개 문파 전용 무공서와 21개 공용 무공서.
- 3·5·7·9·10성 성장 구조.
- 시작 무공 4개·전부 3성·시작 기술 4개 규칙.
- 무공서의 `무공 설명 + 무도 + 3성·7성·10성 카드` 구조.
- 카드 앞면 정보 구조, 절초·진의 표찰과 편집형 합성 파이프라인.

### 진행 중

- 무공 데이터 필드와 계산 규칙.
- 각 기술의 효과·비용·사거리·밸런스.
- 절초 기세의 정확한 획득·상한·소모 수치.
- 상세 팝업의 실제 레이아웃.
- 10문파·공용 무공의 설명·무도·묘리 작성.
- 최종 실장용 캐릭터·무공서·기술 카드 합성 자산.

### 보류

- 문파별 기믹 수행도를 성과 등급에 직접 반영하는 규칙.
- 복잡한 랜덤 손패·덱빌딩.
- 별도 장비 강화·경제 시스템.
- 10문파 밖 신규 세력.
