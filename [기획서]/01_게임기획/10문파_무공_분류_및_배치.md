# 10문파 무공 분류 및 배치

## 1. 공식 무공 분류

### 핵심무공 8종

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법 / 궁술 / 암기술`

- 핵심무공은 직접 공격, 합 판정, 피해, 밀치기, 장전과 사거리 운용을 담당한다.
- `부법`은 공식 성장 계통에서 제외한다. 도끼는 개별 무기·연출 태그로만 사용할 수 있다.
- `점혈술`은 핵심무공에서 제외하고 의술의 세부 계통으로 편입한다.

### 보조무공 8종

`심법 / 호신공 / 독공 / 마공 / 보법 / 방패술 / 외공 / 의술`

- 의술은 회복·해독·지혈뿐 아니라 점혈·해혈·경맥 봉쇄를 포함한다.
- 신법·경공은 보법의 세부 태그다.
- 보조무공은 문파 기믹, 자원 전환, 명상, 방어, 이동, 상태 생성·해제를 담당한다.

## 2. 10문파 배치

| 문파 | 핵심무공 | 보조무공 | 대표 무공서 권장명 |
|---|---|---|---|
| 무당파 | 검법, 권법 | 심법, 보법 | 태극검법 / 태극권 / 순양무극공 / 제운종 |
| 화산파 | 검법 | 심법 | 매화검법 / 자하신공 |
| 소림사 | 권법, 곤봉법 | 호신공 | 나한권 / 달마봉법 / 금강불괴신공 |
| 당문 | 암기술 | 독공, 의술, 보법 | 당문암기비전 / 오독심경 / 금침의경 / 귀영보 |
| 개방 | 장법, 곤봉법 | 보법 | 항룡십팔장 / 타구봉법 / 취팔선보 |
| 마교 | 도법 | 마공 | 혈염도법 / 역혈마공 |
| 양가장 | 창법 | 방패술 | 양가창법 / 양문철벽패술 |
| 백우궁 | 궁술 | 보법 | 백우궁술 / 유운보법 |
| 사파 | 도법, 암기술 | 외공 | 흑풍도법 / 추혼암기술 / 철골외공 |
| 녹림 | 창법, 궁술 | 보법 | 녹림창법 / 산림궁술 / 산림잠행보 |

## 3. 문파 역할

- **당문**: 암기로 피해와 독을 전달하고, 독공으로 누적 상태를 운용하며, 의술로 해독·회복·점혈·해혈을 수행한다. 보법은 암기 사거리와 재장전 공백을 보완한다.
- **사파**: 변칙 도법과 암기를 혼용하고 외공으로 난전 지속력과 위험한 육체 강화를 얻는다. 검법·장법·보법은 사파 전용 계통에서 제외한다.
- **녹림**: 산채와 산림 전투에 맞춘 창·궁 혼성 전투를 사용한다. 창으로 길목을 막고 궁으로 먼 거리를 압박하며 산림잠행보로 매복·재배치한다.
- **마교**: 일반 교도 무공에는 `천마` 명칭을 사용하지 않는다. 천마 명칭은 천마 개인 또는 특수 계승 콘텐츠로 분리한다.

## 5. 구현 주의

- 문파가 제공하는 보조무공 계통 수와 한 회차 동시 보유 한도는 다르다. 당문은 독공·의술·보법을 제공하지만 실제 보유는 기존 최대 2개 제한을 따른다.
- 카드 아트의 사거리·비용 숫자는 비워 두고 데이터와 UI 레이어에서 편집 가능하게 표시한다.
- 문파 이미지 제작은 이 문서의 배치가 확정된 뒤 진행한다.
