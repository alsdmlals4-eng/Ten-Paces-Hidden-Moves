# 10문파 무공 분류 및 배치

> 이 문서는 활성 게임 기획서의 문파·무공 데이터를 표로 확인하는 상세 부록이다. 이름과 숙련 구조의 최종 상세 원본은 `docs/06_STARTING_FACTION_MASTERY_DATA.md`다.

## 1. 공식 무공 분류

### 핵심무공 8종

`검법 / 도법 / 권법 / 장법 / 창법 / 곤봉법 / 궁술 / 암기술`

### 보조무공 8종

`심법 / 호신공 / 독공 / 마공 / 보법 / 방패술 / 외공 / 의술`

- 부법은 공식 성장 계통에서 제외한다.
- 점혈은 의술의 세부 기술·상태 태그다.
- 신법과 경공은 보법의 세부 태그다.

## 2. 숙련도 구조

| 성급 | 결과 |
|---:|---|
| 3성 | 새로운 기술 1 해금 |
| 5성 | 기술 1 강화 — 새 카드 없음 |
| 7성 | 새로운 기술 2 해금 |
| 9성 | 기술 2 강화 — 새 카드 없음 |
| 10성 | 핵심무공은 절초, 보조무공은 진의 해금 |

## 3. 10문파 배치

| 문파 | 핵심무공서 | 보조무공서 |
|---|---|---|
| 무당파 | 태극검(검법), 태극권(권법) | 순양무극공(심법), 제운종(보법) |
| 화산파 | 매화검법(검법) | 자하신공(심법) |
| 소림사 | 나한권(권법), 달마봉법(곤봉법) | 금강불괴신공(호신공) |
| 당문 | 당문암기비전(암기술) | 오독심경(독공), 금침의경(의술), 귀영보(보법) |
| 개방 | 항룡십팔장(장법), 타구봉법(곤봉법) | 취팔선보(보법) |
| 마교 | 혈염도법(도법) | 역혈마공(마공) |
| 양가장 | 양가창법(창법) | 양문철벽패술(방패술) |
| 백우궁 | 백우궁술(궁술) | 유운보법(보법) |
| 사파 | 오호단문도(도법), 추혼암기술(암기술) | 철골외공(외공) |
| 녹림 | 녹림창법(창법), 산림궁술(궁술) | 산림잠행보(보법) |

당문은 보조무공 세 계통을 제공하지만 한 회차에는 최대 2개만 보유한다.

## 4. 분류 배치 검증

### 핵심무공

| 분류 | 문파 |
|---|---|
| 검법 | 무당, 화산 |
| 도법 | 마교, 사파 |
| 권법 | 무당, 소림 |
| 장법 | 개방 |
| 창법 | 양가장, 녹림 |
| 곤봉법 | 소림, 개방 |
| 궁술 | 백우궁, 녹림 |
| 암기술 | 당문, 사파 |

### 보조무공

| 분류 | 문파 |
|---|---|
| 심법 | 무당, 화산 |
| 호신공 | 소림 |
| 독공 | 당문 |
| 마공 | 마교 |
| 보법 | 무당, 당문, 개방, 백우궁, 녹림 |
| 방패술 | 양가장 |
| 외공 | 사파 |
| 의술 | 당문 |

공식 16종이 모두 최소 한 문파에 배치되어 있다.

## 5. 명칭 경계

- 일반 마교 교도 무공에는 `천마` 명칭을 사용하지 않는다.
- 사파 전용 계통에는 검법·장법·보법을 배치하지 않는다.
- 카드 좌상단은 문파 문양 이미지, 우상단은 무공 종류 아이콘 이미지다.
- 이미지 속 오탈자와 임시 분류가 이 표를 덮어쓰지 않는다.
