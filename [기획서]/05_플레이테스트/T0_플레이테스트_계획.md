# T0 전투 플레이테스트 계획

## 환경

- Godot 4.7.1 stable.
- Windows 목표, 개발 검증은 호환되는 4.7.1 에디터·headless 실행.
- 권장 창: 1280×800 이상.

## 준비 체크

- [ ] Godot에서 `project.godot` 열기.
- [ ] 자동 테스트 실행.
- [ ] 메인 씬 실행.
- [ ] 3회 전투 진행.
- [ ] 첫 회는 설명 없이 관찰.
- [ ] 둘째 회는 툴팁·로그 허용.
- [ ] 셋째 회는 절초 사용을 목표로 진행.
- [ ] 결과를 `플레이테스트_기록_템플릿.md`에 작성.

## 검증 질문

1. AI가 먼저 잠갔다고 믿는가?
2. 두 수를 미리 계획하는 규칙을 이해하는가?
3. 합과 단독 공격을 구분하는가?
4. 기세 증가와 소모의 이유를 설명하는가?
5. 절초 없이도 기본 전투 선택이 재미있는가?

## 즉시 중단 기준

- 입력 잠김·진행 불가·크래시.
- AI가 후행 선택한다고 느끼는 표시.
- 합과 일반 공격을 구분할 수 없음.
- 기세 증가·소모 이유가 로그에서도 불명확.
- 절초가 없으면 선택이 무의미하다는 반응.
