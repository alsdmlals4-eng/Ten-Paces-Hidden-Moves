## 유지 규칙

- 게임 기획·전투·UI/UX·아트·플레이테스트 문서를 종류별로 유지한다.
- 관련 이미지는 문서 본문 바로 아래에 상대경로로 삽입한다.
- 기획 변경과 구현 변경이 있으면 같은 PR에서 관련 문서·이미지·테스트·다음 작업을 갱신한다.
- `python scripts/check_planning_sync.py`가 실패하면 작업을 완료로 보고하지 않는다.
- 원본 사용자 첨부 이미지는 추출 가능한 파일로 제공되면 `assets/예시이미지/원본/`에 보존한다. 현재 두 이미지는 레이아웃과 시선 흐름을 보존한 재구성 SVG로 함께 제공한다.
