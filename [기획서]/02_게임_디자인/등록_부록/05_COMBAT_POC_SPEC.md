# 12세력 대회 세로 슬라이스 명세

> 목적: 10칸 전장·10타이밍·10전 대회에서 공용 행동·합·핵심무공·심법·수련·행운·상대 읽기·제약이 전략적 차이를 만드는지 검증한다. 데모는 5전 예선 결승까지 제공한다.

## 1. 범위

### 데모 포함

- Godot 4.7.1 + GDScript, Windows 데스크톱 데모.
- 10칸 전장, 1대1 AI 결투, 2수 비공개 동시 선택.
- 한 라운드 행동 타이밍 10개, 2타이밍 묶음 5회.
- 1~2전 내부전, 3~4전 예선, 5전 예선 결승.
- 시작 문파 핵심무공·심법 3성.
- 승리 기본 수련 5, C/B/A/S 등급 보너스, 비공개 행운 결과.
- 조건부 4전 종료 절초 개방과 5전 예선 결승 사용.
- 상대 A/B 등급·행운·수련 스타일 생성과 무림식 정보 표시.
- 삼수양보 등 무협식 제약과 승리 추가 보상.

### 전체판 포함

- 전장 10칸·행동 타이밍 10개·대회 10전의 `10-10-10` 구조.
- 8전 8강, 9전 준결승, 10전 결승.
- 동일 타이밍 쌍방 공격의 합 판정과 차이 피해.
- 시작 6세력과 추가 6세력.
- 문파 전용 핵심무공·심법과 각 핵심무공 종류의 공용 무공 2종.
- 핵심무공·심법 1~10성, 절초·진의.
- 심법 보유 최대 2개.
- 보상 선택, 정탐, 재도전, 저장·불러오기.
- 암기벨트, 화살통, 방패의 공용 보조·무기 규칙.

### 제외

- 세계관·대화·인물 관계.
- 랜덤 손패·덱빌딩.
- 별도 장비 강화·경제 시스템.
- 12세력 밖 신규 문파.
- Steamworks 상점·업적·클라우드 연동.
- 문파별 기믹 수행도를 성과 등급에 반영하는 개별 평가 규칙.

## 2. 공통 전투 상태

| 항목 | 값 |
|---|---|
| 전장 | 1~10번 칸, 같은 칸 중첩·교차·자리 교환 가능 |
| 초기 배치 | A=4번 칸, B=6번 칸 |
| 초기 거리 | 2 |
| 라운드 | 행동 타이밍 10개 |
| 선택 단위 | 행동 두 개 비공개 잠금 → 동시 공개 → 순차 해상 |
| 행동력 | 6, 10타이밍 종료 시 전부 회복 |
| 기력 | 12, 라운드 종료 시 소량 회복 |
| 내공 | 6, 라운드 종료 시 기력보다 적게 회복 |
| 합 | 양측 공격이 유효하면 최종 합 비교, 낮은 쪽이 차이 피해 |
| 결투 승리 | 상대 체력 0 또는 정해진 라운드 종료 시 체력 우위 |

AI는 각 2수 묶음 시작 시 공개 정보로 두 행동을 잠근다. 플레이어 행동을 확인한 뒤 수정할 수 없다.

| 3전 | 예선 | 첫 공용·외문 무공 또는 보조장비, 상대 정보 읽기 | 포함 |

- 같은 상대 반복은 풀을 소진하기 전 금지한다.
- 추가 6세력은 AI 적이며 대표 무공을 승리 보상으로 습득할 수 있다.
- 결승 전 수련 단계는 최대 9회다.

## 4. 무공·심법·수련

- 시작 문파 핵심무공과 심법은 각각 3성이다.
- 핵심무공은 여러 개 익힐 수 있다.
- 심법은 보유 자체가 최대 2개다.
- 핵심무공 10성은 최대 1개, 심법 10성은 최대 1개다.
- 핵심무공은 3·6·9성에 기술, 10성에 절초를 얻는다.
- 심법은 3·6·9성에 패시브·기믹·명상 효과를 조합하고 10성에 진의를 얻는다.
- 성급 `n`에 도달하는 비용은 `n`포인트다.
- 핵심무공 3성→10성은 49포인트, 심법 3성→8성은 30포인트다.

## 5. 성과·행운

### 정산식

```text
최종 수련 포인트 = 승리 기본 5 + 등급 보너스 + 행운 앞면 수
```

| 등급 | 등급 보너스 |
|---|---:|
| C | +1 |
| B | +3 |
| A | +5 |
| S | +7 |

A는 의도한 평균 성과다. 성과는 남은 체력, 자원 운용, 무효 행동, 합·거리 판단 같은 공용 지표로 계산한다.

### 비공개 행운 결과표

행운 다이스 3개는 독립 판정하지 않는다. 등급별 앞면 수 결과표에서 한 번 추첨한다.

| 등급 | 0개 | 1개 | 2개 | 3개 | 평균 행운 |
|---|---:|---:|---:|---:|---:|
| C | 0% | 0% | 5% | 95% | 2.95 |
| B | 5% | 5% | 5% | 85% | 2.70 |
| A | 20% | 5% | 20% | 55% | 2.10 |
| S | 60% | 10% | 15% | 15% | 0.85 |

- 개별 다이스 면은 공개하지 않는다.
- 최종 표시는 `B`, `B+`, `B++`, `B+++`처럼 앞면 수를 `+`로 붙인다.
- 낮은 등급일수록 행운 보정이 크지만 최종 기대 수련은 상위 등급이 높다.

### 데모 절초 개방 목표

첫 4전 수련을 핵심무공에 모두 투자한다.

| 첫 4전 성과 | 5전 전 절초 개방 목표 |
|---|---:|
| A 4회 | 약 50% |
| A 3회 + S 1회 | 약 60% |
| A 3회 + B 1회 | 약 30% |

절초는 데모 필수 기능이 아니라 높은 성과와 행운으로 먼저 체험하는 하이라이트다. 절초가 없어도 예선 결승을 시작하고 완료할 수 있어야 한다.

## 6. 상대 성장

### 생성 규칙

- 상대 기준 등급은 A계열 60%, B계열 40%다.
- 상대도 해당 등급의 행운 결과표를 사용해 `+`를 추첨한다.
- 경기당 수련은 플레이어와 같은 `기본 5 + 등급 보너스 + 행운`으로 계산한다.
- 등장 경기 이전까지 누적한 것으로 계산한 포인트를 실제 무공과 심법에 투자한다.
- 상대 등급, 행운, 투자 결과, 수련 스타일은 회차 생성 시 확정한다.
- 재도전과 저장·불러오기에서 재추첨하지 않는다.

### 수련 스타일

| 스타일 | 투자 방향 |
|---|---|
| 균형형 | 핵심무공과 심법을 고르게 투자 |
| 핵심무공 올인형 | 6·9성 기술과 10성 절초를 우선 |
| 심법 연계형 | 문파 기믹, 명상과 자원 순환을 우선 |
| 다무공형 | 주력·심법·공용/외문 무공으로 분산 |

내부 종합 강도는 많이 약한 적, 평균보다 약한 적, 중간, 강적, 매우 강한 적의 5단계로 관리할 수 있다.

## 7. 상대 정보 표현

개발용 태그를 직접 노출하지 않는다.

```text
금지 예: [강적] [A++] [핵심무공 올인형]
```

플레이어에게는 다음으로 암시한다.

- 이명·수식언: 종합 강도와 대표 전법.
- 풍문·최근 전적: 행운과 성장세.
- 무공 평가: 수련 스타일과 성급 경계.
- 정탐: 위 표현을 더 구체적으로 제공하되 내부 수치표를 그대로 노출하지 않음.

이명과 문구는 실제 상대 데이터에 연결한다. 같은 의미에는 같은 어휘 계열을 사용해 반복 플레이로 학습 가능하게 한다.

## 8. 전투 제약

- 전투 시작 전에 상대 이명·풍문·정탐을 확인하고 제약을 선택한다.
- 예: 삼수양보, 단전봉맥, 기혈역행, 일장지양, 병기자박, 호신철회.
- 최대 선택 수와 위험도는 밸런스 데이터로 둔다.
- 특정 문파의 주력 계통을 완전히 금지하는 제약은 만들지 않는다.
- 승리했을 때만 추가 수련 포인트, 보상 선택지, 새로고침, 고급 보상 확률을 정산한다.

## 9. 저장·재도전

- 재도전은 회차당 1회다.
- 게임 전체 최초 재도전은 무료, 이후에는 명망을 사용한다.
- 재도전 시 전투 시작 전 상태로 복구하고 같은 상대와 다시 싸운다.
- 상대 성장, 이명·풍문, 수련 스타일은 유지한다.
- AI의 각 2수 묶음은 공개 정보로 새로 잠근다.
- 저장 데이터는 포맷 버전, 선택 문파, 경기·단계, 상대 생성 결과, 무공·심법·성급·부분 포인트, 장비, 명망, 재도전, 제약과 데모 종료 상태를 보관한다.

## 10. 완료 기준

- 시작 6세력 모두로 새 회차를 시작할 수 있다.
- 데모는 1~2전 내부전, 3~4전 예선, 5전 예선 결승으로 진행된다.
- 전체판은 10전째 결승까지 진행된다.
- 2수 비공개 잠금과 AI 공정성이 유지된다.
- 합 수치와 차이 피해가 로그·UI·테스트에서 일치한다.
- 핵심무공·심법 3·6·9·10성 해금이 실제 전투에 반영된다.
- 심법 보유 최대 2개와 계열별 10성 제한이 지켜진다.
- 등급 보너스와 행운 결과표가 실제 정산과 일치한다.
- 데모 절초 개방률이 목표 조합별 범위에 가깝다.
- 상대 성장과 수련 스타일이 회차 생성·저장·재도전에서 유지된다.
- 상대 이명·풍문이 내부 데이터와 모순되지 않는다.
- 제약 승리 보상이 기본 성장에 필수가 아니면서 추가 선택을 제공한다.
- 저장·불러오기와 Windows 빌드가 완료된다.
