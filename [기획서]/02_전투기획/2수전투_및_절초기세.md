# 2수 전투 및 절초 기세

## 2수 해상

```text
AI 두 수 선잠금
→ 플레이어 두 수 선택·잠금
→ 양측 두 수 동시 공개
→ 1수 해상
→ 전투 종료가 아니면 2수 해상
→ 묶음 완료 보상
```

1수에서 한쪽 체력이 0이 되면 2수는 취소하며 비용·기세를 소모하지 않는다.

## 절초 기세

- 최대·발동 요구: 6.
- 전투 시작: 0.
- 전투 종료: 초기화.
- 양측 모두 정확한 수치를 공개.
- 한 타이밍 최대 +2, 한 묶음 최대 +3.

| 원인 | 획득 |
|---|---:|
| 묶음 정상 완료 | +1 |
| 합 참여 | +1 |
| 합 승리 | +1 |
| 막기 성공 | +1 |
| 회피 성공 | +1 |
| 실제 피격 | +1 |

## 절초 소모

- 핵심무공 해금 및 기세 6일 때만 예약.
- 행동력·기력·내공 부족, 사거리 불성립, 실행 전 전투 종료에는 소모하지 않음.
- 공격이 유효하게 성립한 순간 전량 소모.
- 이후 막기·합 패배로 피해가 0이어도 반환하지 않음.
