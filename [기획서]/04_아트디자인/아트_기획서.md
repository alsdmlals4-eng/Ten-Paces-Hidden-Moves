# 십보강호 아트 기획서

## 0. 한눈에 보기

시각 방향은 **선명한 반실사 무협 인물 + 수묵·한지 질감의 배경 + 먹색·금색 게임 UI**다. 인물·무기·기술 동작은 선명하게, 배경과 장식은 분위기를 만들되 정보 식별을 방해하지 않게 한다.

최신 이미지 자료의 유일한 책임 원본은 [이미지 자료 인덱스](이미지_인덱스.md)다.

## 1. 아트 팀의 책임

- 캐릭터, 배경, 무공서, 카드, 문양, 분류 아이콘, UI 장식, 애니메이션과 VFX를 제작한다.
- 규칙을 빠르게 읽게 만드는 시각 언어를 설계한다.
- 같은 문파와 무공 분류는 모든 화면에서 같은 문양·색·실루엣을 사용한다.
- 생성 이미지 속 잘못된 한글·수치·분류를 최종 자산으로 확정하지 않는다.
- 이미지의 경로·승인 상태·적용 범위를 이미지 인덱스에서 관리한다.
- 새 작업자가 이 문서만 보고 같은 품질로 제작할 수 있도록 색·구도·금지 요소를 기록한다.

## 2. 공통 스타일

### 2.1 그림체

- 캐릭터: 반실사 동양 무협 게임 일러스트.
- 얼굴·손·무기·복식 윤곽: 선명하고 정확하게.
- 배경: 수묵화와 한지 질감, 낮은 대비.
- 공통 UI: 먹색·짙은 갈색·금색.
- 문파색: 복식 안감, 카드 테두리, 기술 궤적에 제한적으로 사용.
- 강한 광원: 절초·진의와 핵심 순간에만 사용.

### 2.2 품질 기준

- 축소 화면에서도 무기·자세·문파가 구분된다.
- 분류 아이콘은 텍스트 없이 실루엣만으로 읽힌다.
- 남녀 캐릭터는 얼굴만 바꾼 복제가 아니라 체형·자세·실루엣이 다르다.
- 기술 삽화는 이름을 읽지 않아도 공격 방식이 추정된다.
- 문파별 기술 궤적이 다르다. 모든 문파에 음양 원·금빛 폭발을 반복하지 않는다.

## 3. 문파 종합 시트

### 3.1 레이아웃

```text
상단: 문파 문양 / 문파명 / 정체성 문장
좌측: 남자 제자 독립 세로 패널 + 여자 제자 독립 세로 패널
중앙: 무공서 세로 열
우측: 무공서별 3성 기술 / 7성 기술 / 10성 절초·진의 카드
```

- 남녀 캐릭터는 겹치지 않는다.
- 좌측 약 40~45%, 중앙 15~20%, 우측 약 40%를 권장한다.
- 캐릭터는 전신 또는 발목까지 보여 복식과 자세가 읽혀야 한다.
- 무공서 한 권과 대응 카드 3장을 같은 가로 행에 정렬한다.
- 별도 무기·복식 확대 패널은 두지 않는다.

### 3.2 캐릭터 역할

- 한 명은 주력 핵심무공을 보여준다.
- 다른 한 명은 보조무공 또는 두 번째 핵심무공을 보여준다.
- 같은 문파 복식 체계를 공유하지만 무공 역할에 따라 소매·보호구·장비가 달라진다.
- 배경은 같은 거점의 다른 지점처럼 연결되며 인물보다 약하게 처리한다.

## 4. 무공 카드 앞면

### 4.1 정보 구조

- 좌상단: 문파 문양 이미지. 공용은 `무(無)` 상징 이미지.
- 최상단 중앙: 사거리 슬롯. 숫자는 UI 레이어.
- 상단 중앙: 기술 이름. 편집 가능한 텍스트.
- 우상단: 검법·권법·심법 등 무공 종류의 아이콘 이미지.
- 중앙: 기술 삽화.
- 하단: 비용·효과 없이 무공의 묘리 한 소절.

### 4.2 묘리 문구

- 최대 두 줄.
- 한 줄 14~18자 권장.
- 기존 예시보다 약 1.4~1.6배 큰 글자.
- 가운데 정렬, 굵고 선명한 붓글씨.
- 삽화와 분리된 바탕과 충분한 여백.
- 문장이 길면 글자를 줄이지 않고 문장을 압축한다.

### 4.3 절초와 진의

**절초**

- 우측 세로 `절초` 표찰.
- 금장과 문파 대표색의 강한 테두리.
- 무기와 동세가 크게 보이는 필살기 연출.
- 일반 카드보다 화면 돌파감과 공격성을 강화한다.

**진의**

- 우측 세로 `진의` 표찰.
- 경맥·자연·호흡·깨달음의 상징.
- 절초보다 고요하지만 신비롭고 완성된 인상.
- 단순 공격 장면이 아니라 무공 원리가 완성되는 순간을 표현한다.

## 5. 문양과 아이콘

### 5.1 문파 문양 10종

| 문파 | 중심 상징 | 대표색 |
|---|---|---|
| 무당 | 태극·구름·산봉 | 남청·백색 |
| 화산 | 매화 다섯 잎·검광 | 백색·매화홍 |
| 소림 | 범종 또는 연꽃·불광 | 황토·금색 |
| 당문 | 비도·침·독방울 | 암청·자주 |
| 개방 | 대나무 봉·호리병 | 녹갈·황토 |
| 마교 | 균열난 원·역류 혈염 | 흑색·심홍 |
| 양가장 | 창·방패·군기 | 군청·은색 |
| 백우궁 | 활·화살·깃털 | 설백·빙청 |
| 사파 | 외날도·비도·갈라진 원 | 암자주·먹색 |
| 녹림 | 산봉·창·화살·목책 | 짙은 녹색·갈색 |

문양 안에 문파 이름을 넣지 않는다.

### 5.2 무공 종류 아이콘 16종

| 종류 | 모티브 |
|---|---|
| 검법 | 곧은 양날검 |
| 도법 | 넓은 외날도와 초승달 참격 |
| 권법 | 정면 주먹 |
| 장법 | 펼친 손과 파동 원 |
| 창법 | 창날과 직선 궤적 |
| 곤봉법 | 봉과 회전 원호 |
| 궁술 | 당겨진 활과 화살 |
| 암기술 | 방사형 침·비도 3개 |
| 심법 | 좌선 인물과 단전 원 |
| 호신공 | 종 또는 보호막 |
| 독공 | 독방울과 뱀 |
| 마공 | 역회전 균열 문양 |
| 보법 | 발자국과 구름 잔상 |
| 방패술 | 정면 원형 방패 |
| 외공 | 암석처럼 갈라진 상체·팔 |
| 의술 | 금침과 약초 잎 |

아이콘 내부에 한글·한자를 넣지 않는다.

## 6. 문파별 아트 바이블

### 무당파
남청·백색 도포, 직검, 원형 검세와 권장, 운해·도관. 키워드: 균형·흐름·절제. 불꽃·중갑 금지.

### 화산파
백색·매화홍, 가벼운 검사, 절벽·매화. 키워드: 사선 검광·꽃잎·상승. 둔중한 전사형 금지.

### 소림사
황토·적갈 승복, 단단한 체형, 권·봉, 산문·석등. 키워드: 낮은 중심·불광·암석. 암살자형 금지.

### 당문
암청·자주, 몸에 붙는 복식, 비도·침통·약낭, 대나무·약방·기관. 키워드: 정밀·은폐·독과 의술. 중갑 금지.

### 개방
녹갈·황토, 덧댄 천옷, 대나무 봉·호리병, 장터·황야. 키워드: 호쾌함·먼지·변칙. 귀족풍 금지.

### 마교
흑색·심홍, 비대칭 겹옷·부분 갑주, 폐허·재. 키워드: 혈염·역류·과부하. 일반 교도에게 천마 문양 금지.

### 양가장
군청·은색 군문 갑주, 창·방패, 군영·성벽. 키워드: 직선·진형·불퇴. 떠돌이 협객풍 금지.

### 백우궁
설백·빙청, 장궁·팔 보호대·흐르는 천, 고지대 활터. 키워드: 정밀·풍향·백색 궤적. 산적형 금지.

### 사파
암자주·먹색, 비대칭 실전 장비·흉터·도·암기, 폐허 객잔. 키워드: 사각·난전·생존. 정제된 정파 제복 금지.

### 녹림
짙은 녹색·가죽 갈색, 산악 생존복·창·활·밧줄, 산채·목책. 키워드: 매복·길목·낙엽·흙먼지. 귀족 궁수와 엘프풍 금지.

## 7. 전투 화면

- 중앙 비무 장면이 감정 중심이다.
- 양측 캐릭터는 전장 양끝에서 서로 마주본다.
- 전장 10칸이 배경 장식보다 우선해 읽힌다.
- 상단은 양측 상태와 절초 기세, 하단은 카드와 두 수 슬롯.
- 해 질 무렵 연무대 같은 따뜻한 환경광을 참고한다.
- 합은 충돌점, 일반 공격은 일방향 타격으로 구분한다.

## 8. 제작 파이프라인

1. 게임 기획서의 명칭·분류·성급 구조 확인.
2. 아트 기획서와 이미지 인덱스의 최신 방향 확인.
3. 텍스트 없는 캐릭터·배경·카드 삽화 제작.
4. 문파 문양·분류 아이콘·표찰을 별도 자산으로 제작.
5. 이름·사거리·묘리·절초·진의를 편집 가능한 레이어로 합성.
6. 사용자 승인 뒤 원본 저장소 반입.
7. 이미지 인덱스에 경로·상태·적용 범위 기록.
8. 실제 UI 축소 크기에서 가독성 검수.

## 9. 현재 상태

### 확정

- 반실사 인물 + 수묵 배경 + 먹색·금색 UI.
- 남녀 분리 패널·무공서·카드 구조.
- 카드 앞면 구조와 큰 묘리 문구.
- 분류를 텍스트가 아닌 아이콘으로 표시.
- 절초·진의 세로 표찰.
- 10문파 색·실루엣·배경·기술 궤적 방향.

### 진행 중

- 문파 문양 10종 최종 원본.
- 무공 종류 아이콘 16종 최종 원본.
- 무당파 종합 시트 재제작.
- 28개 무공서와 84개 카드 삽화.
- 전투 화면 반실사 자산 적용.

### 검증 필요

- 대화에서 승인된 최신 시안의 저장소 원본 반입.
- 실제 카드 크기의 묘리 가독성.
- UI 합성 시 문양·분류·기술 데이터 매핑.

## 10. 다음 작업

1. 문파 문양 10종과 무공 종류 아이콘 16종을 개별 자산으로 분리.
2. 무당파 종합 시트를 정확한 분류 매핑으로 재구성.
3. 무당파 12개 카드의 텍스트 없는 삽화 제작.
4. UI 합성에서 묘리 글자 크기와 축소 가독성 검증.
5. 승인 원본을 이미지 인덱스에 등록.

## 12. 변경 시 갱신 대상

캐릭터·카드·아이콘·배경·UI 장식·애니메이션·VFX 또는 이미지 승인 상태가 바뀌면 이 문서와 이미지 인덱스를 같은 작업에서 갱신한다.
