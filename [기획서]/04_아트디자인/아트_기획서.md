# 십보강호 아트 기획서

## 0. 한눈에 보기

십보강호의 시각 방향은 **선명한 반실사 무협 인물 + 수묵·한지 질감의 배경 + 먹색·금색 게임 UI**다. 캐릭터와 기술 동작은 명확하게 읽혀야 하며, 배경과 장식은 분위기를 만들되 전투 정보와 카드 식별을 방해하지 않는다.

최신 이미지 자료의 유일한 인덱스는 [이미지 자료 인덱스](이미지_인덱스.md)다. 다른 문서에서는 이미지를 복제하지 않고 해당 인덱스를 링크한다.

## 1. 아트 팀의 책임

- 문파 캐릭터, 배경, 무공서, 기술 카드, 문양, 분류 아이콘, UI 장식, 애니메이션과 VFX를 제작한다.
- 게임 규칙을 정확히 읽게 만드는 시각 언어를 설계한다.
- 같은 문파와 무공 분류는 모든 화면에서 같은 문양과 색을 사용한다.
- 생성 이미지 속 잘못된 한글이나 수치를 최종 자산으로 확정하지 않는다.
- 이미지 원본, 승인 상태, 적용 범위를 이미지 인덱스에서 관리한다.
- 신규 작업자가 이 문서만 보고 동일한 품질과 방향으로 자산을 만들 수 있도록 실루엣·색·구도·금지 요소를 구체적으로 기록한다.

## 2. 공통 아트 스타일

### 2.1 그림체

- 인물: 반실사 동양 무협 게임 일러스트.
- 얼굴·손·무기·복식 윤곽은 선명하게.
- 배경: 수묵화와 한지 질감, 낮은 대비.
- 광원: 자연광 중심. 절초·진의에만 강한 문파색 광원을 사용.
- 선: 먹선과 섬세한 의복선. 지나친 애니메이션 셀 채색은 피한다.
- 색: 공통 먹색·갈색·금색 위에 문파 대표색을 제한적으로 사용.

### 2.2 품질 기준

- 축소 화면에서도 캐릭터의 무기·자세·문파가 구분된다.
- 카드 아이콘은 텍스트 없이 실루엣만으로 분류된다.
- 남녀 캐릭터는 얼굴만 바꾼 복제품이 아니라 자세·신장·체형·실루엣이 다르다.
- 기술 삽화는 무공 이름을 읽지 않아도 공격 방식이 추정된다.
- 문파별 기술 궤적이 서로 다르다. 모든 기술에 음양 원이나 금빛 폭발을 반복하지 않는다.

## 3. 문파 종합 시트 규격

### 3.1 레이아웃

```text
상단: 문파 문양 / 문파명 / 정체성 문장
좌측: 남자 제자 독립 패널 / 여자 제자 독립 패널
중앙: 무공서 세로 열
우측: 3성 기술 / 7성 기술 / 10성 절초·진의 카드 열
```

- 남녀는 서로 겹치지 않고 각각 독립된 세로 칸에 선다.
- 좌측은 전체 폭의 약 40~45%, 중앙은 15~20%, 우측은 40% 안팎을 사용한다.
- 캐릭터 패널은 전신 또는 발목까지 보여 문파 복식과 전투 자세가 읽혀야 한다.
- 무공서 한 권과 대응 기술 카드 세 장을 같은 가로 행으로 정렬한다.
- 별도의 무기·복식 확대 패널은 두지 않는다.

### 3.2 캐릭터 역할

- 한 캐릭터는 주력 핵심무공을 보여준다.
- 다른 캐릭터는 보조무공 또는 두 번째 핵심무공을 보여준다.
- 동일 문파 복식 언어를 공유하지만, 전투 방식에 따라 소매·보호구·장비 배치를 다르게 한다.
- 캐릭터 뒤 배경은 각자의 역할을 보조하되 두 패널이 같은 장소의 다른 지점처럼 연결되어야 한다.

## 4. 무공 카드 앞면 규격

### 4.1 정보 구조

- 좌상단: 문파 문양 이미지. 공용 무공은 `무(無)`를 상징하는 독립 문양.
- 최상단 중앙: 사거리 슬롯. 숫자는 별도 UI 레이어.
- 상단 중앙: 기술 이름. 별도 텍스트 레이어.
- 우상단: 무공 종류 아이콘 이미지.
- 중앙: 기술 삽화.
- 하단: 무공의 묘리 한 소절.

카드 하단에는 비용, 효과, 수치, 키워드를 넣지 않는다. 이 정보는 상세 팝업에서 처리한다.

### 4.2 묘리 문구

- 최대 두 줄.
- 한 줄당 약 14~18자 권장.
- 기존 시안보다 1.4~1.6배 큰 글자.
- 중앙 정렬.
- 획이 굵고 알아보기 쉬운 붓글씨.
- 삽화와 분리된 한지색 또는 어두운 반투명 바탕.
- 문구가 길면 글자를 줄이지 말고 문장을 압축한다.

### 4.3 절초와 진의

**절초**

- 우측 세로 표찰.
- 금장과 문파 대표색의 강한 테두리.
- 인물과 무기의 동세를 크게 보여준다.
- 일반 카드보다 공격적이고 필살기다운 광원과 화면 돌파감을 사용한다.

**진의**

- 우측 세로 표찰.
- 원형 기운, 경맥, 자연 현상, 깨달음의 상징을 강조한다.
- 절초보다 고요하지만 신비롭고 완성된 인상.
- 단순 공격 장면보다 무공 원리가 완성되는 순간을 표현한다.

## 5. 무공 종류 아이콘 16종

아이콘은 카드 우상단에서 텍스트 대신 사용한다. 공통 원형 메달 안에 굵은 단색 실루엣을 넣는다.

| 종류 | 중심 모티브 |
|---|---|
| 검법 | 곧은 양날검과 얇은 검광 |
| 도법 | 넓은 외날도와 초승달 참격 |
| 권법 | 정면을 향한 주먹 |
| 장법 | 펼친 손바닥과 파동 원 |
| 창법 | 창날과 직선 궤적 |
| 곤봉법 | 봉과 회전 원호 |
| 궁술 | 당겨진 활과 화살 |
| 암기술 | 방사형 침·비도 3개 |
| 심법 | 좌선 인물과 단전 원 |
| 호신공 | 종 또는 인체를 감싸는 보호막 |
| 독공 | 독방울과 뱀 |
| 마공 | 역회전 균열 문양 |
| 보법 | 발자국과 구름 잔상 |
| 방패술 | 정면 원형 방패 |
| 외공 | 암석처럼 갈라진 상체·팔 |
| 의술 | 금침과 약초 잎 |

아이콘 내부에 한글·한자를 넣지 않는다.

## 6. 문파별 아트 바이블

### 6.1 무당파

- 대표색: 남청·백색·먹색, 은은한 금색.
- 실루엣: 넓은 도포와 흐르는 옷자락.
- 문양: 태극, 구름, 산봉.
- 남성: 태극검의 정적이고 낮은 검세.
- 여성: 태극권·순양무극공·제운종의 원형 손동작과 흐르는 이동.
- 배경: 운해, 도관, 석계단, 산봉우리.
- 기술 궤적: 원, 흐름, 음양 전환, 구름.
- 금지: 갑주 과다, 불꽃 폭발, 광폭한 표정.

### 6.2 화산파

- 대표색: 백색·회색·매화홍.
- 실루엣: 가볍고 날렵한 검사, 무당보다 짧고 날카로운 소매.
- 문양: 매화 다섯 잎, 봉우리, 얇은 검광.
- 배경: 절벽, 매화나무, 차가운 바람.
- 기술 궤적: 사선 검광, 꽃잎, 상승·낙하 동작.
- 금지: 둔중한 전사형, 과도하게 장식적인 꽃무늬.

### 6.3 소림사

- 대표색: 황토·갈색·적갈, 금빛 불광.
- 실루엣: 단단한 체형, 단순한 승복, 낮은 중심.
- 문양: 범종 또는 연꽃과 불광.
- 배경: 산문, 석등, 법당.
- 기술 궤적: 무거운 정권, 봉 회전, 종형 보호막, 깨지는 돌.
- 금지: 가벼운 암살자형, 귀족 장식.

### 6.4 당문

- 대표색: 암청·자주·먹색, 독의 녹색 포인트.
- 실루엣: 몸에 붙는 복식, 넓은 소매 속 수납 구조.
- 문양: 비도·침, 방사선, 독방울.
- 배경: 대나무숲, 약방, 기관 회랑.
- 기술 궤적: 가는 은빛 투사선, 독무, 금침과 경맥.
- 금지: 중갑, 정면 돌격형.

### 6.5 개방

- 대표색: 녹갈·황토·회색.
- 실루엣: 여러 천을 덧댄 자유로운 복식.
- 문양: 대나무 봉과 호리병.
- 배경: 장터 외곽, 황야, 대나무, 먼지.
- 기술 궤적: 장력 파동, 봉 회전, 흙먼지, 취중 잔상.
- 금지: 귀족적이고 지나치게 깨끗한 정파복.

### 6.6 마교

- 대표색: 흑색·심홍·암적.
- 실루엣: 비대칭 겹옷, 부분 갑주, 그을린 옷자락.
- 문양: 균열난 원과 역류하는 혈염.
- 배경: 폐허, 재, 검붉은 하늘.
- 기술 궤적: 피처럼 무거운 도광, 역류 경맥, 과부하 균열.
- 금지: 일반 교도에게 천마 명칭·문양, 서양 악마풍.

### 6.7 양가장

- 대표색: 군청·은색·흑회색, 붉은 군기.
- 실루엣: 규율 잡힌 군문 갑주, 창·방패.
- 문양: 방패 앞 창 또는 교차 창.
- 배경: 군영, 성벽, 깃발, 훈련장.
- 기술 궤적: 직선 관통, 방패 중첩, 창진.
- 금지: 자유분방한 떠돌이 협객풍.

### 6.8 백우궁

- 대표색: 설백·빙청·은색.
- 실루엣: 긴 활, 비대칭 소매, 팔 보호대, 흐르는 천.
- 문양: 활·화살·깃털.
- 배경: 고지대 누각, 활터, 설기와 바람.
- 기술 궤적: 흰 깃털 잔상, 얇고 긴 사선, 시간차 연사.
- 금지: 산적형 궁수, 무거운 중갑.

### 6.9 사파

- 대표색: 암자주·먹색·갈흑.
- 실루엣: 통일되지 않은 실전 장비, 비대칭 소매와 흉터.
- 문양: 외날도와 비도, 갈라진 원.
- 배경: 폐허 객잔, 뒷골목, 어두운 거리.
- 기술 궤적: 대각선 도광, 사각 암기, 탁한 연기, 육체 충돌.
- 금지: 정제된 정파 제복과 완전한 좌우대칭.

### 6.10 녹림

- 대표색: 짙은 녹색·가죽 갈색·회색.
- 실루엣: 산악 생존복, 가죽 보호구, 창·활·밧줄.
- 문양: 산봉우리, 창날, 화살, 목책.
- 배경: 산채, 목책, 망루, 숲길, 협곡.
- 기술 궤적: 길목을 막는 수평 창, 나무 사이 화살, 낙엽과 흙먼지.
- 금지: 엘프형 숲문양, 귀족 궁수, 군문식 완전 정렬.

## 7. 전투 화면 아트

- 중앙의 비무 장면이 감정 중심이다.
- 양측 캐릭터는 전장 양끝에서 서로 마주본다.
- 전장 10칸이 배경 장식보다 우선해 읽혀야 한다.
- 상단은 양측 상태와 절초 기세, 하단은 카드와 두 수 슬롯.
- 해 질 무렵의 연무대처럼 따뜻한 환경광을 기본 참고로 한다.
- 합은 무기 충돌점, 일반 공격은 명확한 일방향 타격으로 구분한다.
- 거리 실패·자원 부족·행동 취소는 별도 폴백 연출을 갖는다.

## 8. 이미지 제작·반입 파이프라인

1. 게임 기획서의 명칭·분류·성급 구조를 확인한다.
2. 아트 기획서와 이미지 인덱스의 최신 방향을 확인한다.
3. 텍스트 없는 캐릭터·배경·카드 삽화를 제작한다.
4. 문파 문양·분류 아이콘·표찰을 별도 자산으로 제작한다.
5. 이름·사거리·묘리·절초·진의는 편집 가능한 레이어로 합성한다.
6. 사용자 승인 뒤 원본을 저장소에 반입한다.
7. 이미지 인덱스에 경로·상태·적용 범위를 기록한다.
8. 게임 화면에서 실제 축소 크기와 가독성을 검수한다.

## 9. 현재 상태

### 확정

- 반실사 인물 + 수묵 배경 + 먹색·금색 UI.
- 문파 종합 시트의 남녀 분리 패널·무공서·카드 구조.
- 카드 앞면 구조와 큰 묘리 문구.
- 무공 종류를 텍스트가 아닌 아이콘으로 표시.
- 절초·진의 세로 표찰.
- 10문파 외형·색·기술 궤적 방향.

### 진행 중

- 문파 문양 10종 최종 원본.
- 무공 종류 아이콘 16종 최종 원본.
- 무당파 종합 시트 재제작.
- 28개 무공서와 84개 카드 삽화.
- 전투 화면 반실사 아트 적용.

### 원본 반입 필요

대화에서 승인된 최신 카드·문파 시트·아이콘 시안은 아직 저장소 원본으로 반입되지 않았다. 세부 목록은 이미지 인덱스를 따른다.

## 10. 다음 작업
