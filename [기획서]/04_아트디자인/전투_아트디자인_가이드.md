# 전투 아트 디자인 가이드

## 예시 이미지 A — 무협 장면과 카드의 결합

![전투 장면과 카드 중심 참고](../assets/예시이미지/전투_장면_카드_참고.svg)

참고할 요소:

- 중앙의 비무 장면이 전투의 감정 중심이 된다.
- 카드와 자원은 장면을 가리지 않는 하단 정보층에 둔다.
- 양측 상태는 상단 좌우 대칭으로 배치한다.
- 궁극기·절초는 상단 기세와 카드 상태가 같은 의미를 전달한다.

복제하지 않을 요소:

- 임시 수치·임시 문구.
- 과도한 카드 수와 장식.
- 전투 규칙에 없는 타임라인·보상 표시.

## 예시 이미지 B — 정보 중심 전투 와이어프레임

![전투 UI 와이어프레임 참고](../assets/예시이미지/전투_UI_와이어프레임_참고.svg)

참고할 요소:

- 10칸 전장과 양측 위치가 한눈에 보인다.
- 1수·2수 슬롯이 카드 선택과 분리되어 있다.
- 상대 선잠금과 현재 단계가 화면 중심에서 읽힌다.
- 제한된 청색·적색 포인트로 양측을 구분한다.

## T0 아트 범위

- 실제 캐릭터·배경 원화 대신 도형·텍스트 폴백.
- 기세 6칸, 위치 셀, 카드 상태, 로그 가독성 검증.
- 고급 컷인·VFX·무기 충돌 연출은 T1 이후.
