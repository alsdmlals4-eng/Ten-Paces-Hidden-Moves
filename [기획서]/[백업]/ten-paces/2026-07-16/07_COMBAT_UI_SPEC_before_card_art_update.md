# 12세력 대회 전투 UI 명세

> 목적: 플레이어가 시작 문파 선택부터 9전 대진까지 이해하고, 모든 결투에서 공용 행동과 문파 규칙의 차이를 명확히 읽게 한다.

## 1. 회차·대진 화면

| 영역 | 표시 내용 |
|---|---|
| 시작 문파 | 화산·개방·당문·소림·무당·마교의 무기, 대표 규칙, 추천 거리 |
| 대진 진행 | 내부전 / 추천패 / 예선 / 본선 / 준결승 / 결승, 현재 경기와 9전 진행도 |
| 다음 상대 | 세력, 무기·사거리, 공개 대표 규칙, 선호 거리, 정탐 정보 |
| 보상 선택 | 무공서, 수련도, 보조장비, 회복, 정탐의 효과와 적용 대상 |
| 대성 패널 | 보유 문파 무공의 1~5성 진행, 다음 해금 효과, 외문 규칙 여부 |

추가 6세력은 대진·보상·도감에서 같은 형식으로 표시한다. 녹림채는 ‘사파 / 도적·산적’ 분류와 약탈도 기력 탈취를 함께 표시한다. 플레이어가 모르는 정보는 ‘정탐 전’으로 분명히 가린다.

## 2. 전투 화면

| 영역 | 표시 내용 |
|---|---|
| 적 상태 | 체력·기력·내공, 무기 장전, 방패, 독·마화, 대표 유파 규칙 |
| 10칸 전장 | 1~10번 칸, A=4 / B=6의 초기 배치, 같은 칸 중첩과 현재 거리 |
| 플레이어 상태 | 체력·기력·내공, 무기·보조장비, 상태, 유파 규칙 |
| 행동열 | 6슬롯. 각 슬롯의 공용 행동, 비용, 예상 사거리, 보조장비 부착 |
| 결과 로그 | 이동 → 방어·회피 → 공격·밀치기 → 피해·상태 → 회복 순서 |

## 3. 행동열 배치 규칙

- 공용 행동은 접근, 후퇴, 공격, 막기, 회피, 밀치기, 무기 교체, 재장전이다.
- 행동 슬롯과 행동력은 분리해 표기한다.
- 행동력·기력·내공 부족, 사거리 불가, 재장전 필요를 배치 시점에 표시한다.
- 이동 후 사거리 판정임을 공격·밀치기 미리보기에서 보여 준다.
- 암기벨트·화살통은 기존 행동 위에 부착된 무행동 사용 횟수로 표시한다.
- 방패는 막기 증가와 회피 감소를 동시에 표시한다.
- 양측 행동열은 모두 잠긴 뒤 같은 화면에서 공개한다.

## 4. 결과 설명 기준

- 결과 로그는 항상 ‘공용 행동 → 무기 규칙 → 유파/대성 수정 → 최종 결과’ 순으로 설명한다.
- 예: ‘막기 40% + 금강호신 15% = 55% 경감’
- 예: ‘활 공격 후 재장전 필요 → 화살통 1회 사용으로 즉시 장전 유지’
- 예: ‘마화 사용: 체력 5 소모, 공격 강화, 마화 1중첩. 라운드 종료 시 중첩 피해’
- AI가 선택한 행동은 공개 후 현재 턴의 공개 정보와 행동열만으로 설명한다.

## 5. UI 검증

- 어떤 문파를 선택해도 다음에 바뀌는 공용 행동과 이유를 읽을 수 있다.
- 본선 진입 전후 보상·대성·상대 정보가 같은 화면 흐름에서 연결된다.
- 소림·당문뿐 아니라 추가 6세력의 장전·방패·독·마화·기력 탈취 차이가 숨겨지지 않는다.
